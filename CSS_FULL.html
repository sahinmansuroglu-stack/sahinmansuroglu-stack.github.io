Giriş - CSS ile Sayfaları Güzelleştirin

CSS (Cascading Style Sheets), HTML ile oluşturduğumuz web sayfası iskeletini görsel olarak biçimlendirmemizi sağlayan bir stil dilidir. Renkler, yazı tipleri, yerleşim düzeni ve daha birçok görsel özelliği CSS ile kontrol ederiz.

Bu doküman, HTML bilginizi bir üst seviyeye taşıyarak web sayfalarınıza nasıl profesyonel bir görünüm kazandıracağınızı öğretecek. Hazırsanız, web tasarımının renkli dünyasına dalalım!

CSS Nasıl Eklenir?

HTML belgenize CSS eklemenin üç yolu vardır:

1. Harici Stil Sayfası (External CSS)

En yaygın ve tavsiye edilen yöntemdir. CSS kodlarınızı ayrı bir .css dosyasına yazarsınız ve <link> etiketiyle HTML sayfanıza bağlarsınız. Bu yöntem, kod tekrarını önler ve yönetimi kolaylaştırır.

<!-- index.html dosyasının <head> bölümünde -->
<link rel="stylesheet" href="stilim.css">
/* stilim.css dosyası */
body {
  background-color: #f0f0f0;
}
h1 {
  color: navy;
}

2. Dahili Stil Sayfası (Internal CSS)

CSS kodlarınızı HTML belgesinin <head> bölümü içine, <style> etiketleri arasına yazarsınız. Bu yöntem sadece tek bir sayfa için özel stiller tanımlamak istediğinizde kullanışlıdır.

<!-- index.html dosyasının <head> bölümünde -->
<style>
  body {
    background-color: lightblue;
  }
  h1 {
    color: white;
    text-align: center;
  }
</style>

3. Satır İçi Stiller (Inline CSS)

Stilleri doğrudan ilgili HTML etiketinin içine style özelliği ile eklersiniz. Bu yöntem, sadece tek bir elemana özel, küçük bir stil değişikliği yapmak için kullanılır. Genellikle tavsiye edilmez çünkü yönetimi zordur.

<h1 style="color: blue; text-align: right;">Bu Mavi Bir Başlık</h1>
<p style="font-size: 20px;">Bu paragrafın yazı boyutu büyük.</p>

Çözümlü Örnekler 💡

  1. Satır içi stil ile <h2> rengini değiştirme:

    <h2 style="color: teal;">Bu bir alt başlık.</h2>

  2. Dahili stil ile <body> yazı boyutunu ayarlama:

    <head>
    <style>
        body { font-size: 18px; }
    </style>
</head>

  3. theme.css adında bir harici stil sayfası bağlama:

    <head>
    <link rel="stylesheet" href="theme.css">
</head>

  4. Satır içi stil ile birden fazla özellik uygulama:

    <div style="background-color: #333; color: white; padding: 10px;">Koyu Tema Kutu</div>

  5. Dahili stil ile liste işaretlerini kaldırma:

    <head>
    <style>
        ul { list-style-type: none; }
    </style>
</head>

Sıra Sende! (Bölüm 1) 🧠

Aşağıdaki HTML yapısını kullanarak istenen stilleri uygulayın:

<!DOCTYPE html>
<html>
<head>
    <title>Alıştırma</title>
</head>
<body>
    <h1>Başlık</h1>
    <p>Bu bir paragraftır.</p>
    <span>Bu bir span etiketidir.</span>
</body>
</html>
  • <h1> başlığının rengini satır içi (inline) CSS kullanarak kırmızı yapın.
  • Dahili (internal) CSS kullanarak tüm <p> etiketlerinin yazı rengini yeşil yapın.
  • Harici (external) bir CSS dosyası oluşturup <body>'nin arka plan rengini #333 (koyu gri) olarak ayarlayın ve bunu HTML dosyanıza bağlayın.
  • <span> etiketine satır içi stil ekleyerek yazı tipini kalın (font-weight: bold;) yapın.
  • Dahili CSS kullanarak tüm <h1> etiketlerini ortalayın (text-align: center;).
  • Harici CSS dosyanızda, sayfanın varsayılan metin rengini white olarak ayarlayın.

Sıra Sende! (Bölüm 2) 🧠

Şimdi de bu HTML yapısını stilize edin:

<div class="kart">
  <h3>Kart Başlığı</h3>
  <p>Bu bir kart içeriğidir.</p>
</div>
  • Dahili stil kullanarak .kart sınıfına sahip `div`'e 1px solid gray değerinde bir kenarlık (border) ekleyin.
  • <h3> başlığının rengini satır içi stil ile skyblue yapın.
  • Tüm bu yeni stilleri (kenarlık ve başlık rengi) harici CSS dosyanıza taşıyın.
  • Satır içi stil kullanarak paragrafın yazı boyutunu 14px yapın.
  • Dahili stil ile .kart'a 20px `padding` verin.
  • Harici CSS dosyanızda tüm div'lerin varsayılan dış boşluğunu (margin) `10px` yapın.

Seçiciler (Selectors)

Seçiciler, stil uygulamak istediğiniz HTML elemanlarını hedeflemenizi sağlayan kurallardır.

Örnek HTML Yapısı

<div class="container">
    <h1>Ana Başlık</h1>
    <p>Bu normal bir paragraf.</p>
    <p id="giris">Bu ID'si olan bir paragraf.</p>
    <div class="uyari">Bu bir uyarı mesajıdır.</div>
    <a href="https://google.com" target="_blank">Google</a>
</div>
/* Etiket Seçici: Tüm <p> etiketlerini hedefler */
p { color: gray; }

/* ID Seçici: id="giris" olan tek bir elemanı hedefler */
#giris { border: 1px solid blue; }

/* Sınıf Seçici: class="uyari" olan tüm elemanları hedefler */
.uyari { color: red; font-weight: bold; }

/* Nitelik Seçici: href niteliği "https://google.com" olan tüm <a> etiketlerini hedefler */
a[href="https://google.com"] { text-decoration: none; }

/* Evrensel Seçici: Sayfadaki tüm elemanları hedefler (dikkatli kullanılmalı) */
* { box-sizing: border-box; }

/* Gruplama Seçicisi: Birden fazla seçiciyi aynı anda hedefler */
h1, h2, h3 { font-family: Arial, sans-serif; }

Çözümlü Örnekler 💡

  1. Soy Seçici (Descendant Selector): Bir <div> içindeki tüm <p>'leri seçer.

    div p { font-style: italic; }

  2. Çocuk Seçici (Child Selector): Bir <ul>'nin doğrudan altındaki <li>'leri seçer.

    ul > li { list-style-type: square; }

  3. Bitişik Kardeş Seçici (Adjacent Sibling Selector): Bir <h2>'den hemen sonra gelen <p>'yi seçer.

    h2 + p { margin-top: 0; }

  4. Genel Kardeş Seçici (General Sibling Selector): Bir <h2>'den sonra gelen tüm <p> kardeşlerini seçer.

    h2 ~ p { color: purple; }

  5. Birden Fazla Sınıf Seçici: Hem .kart hem de .one-cikan sınıfına sahip elemanları seçer.

    .kart.one-cikan { border: 2px solid gold; }

Sıra Sende! (Bölüm 1) ✍️

Aşağıdaki HTML yapısını kullanarak istenen stilleri uygulayın:

<div class="container">
    <h2>Alt Başlık 1</h2>
    <p>Paragraf 1</p>
    <img id="logo" src="logo.png" alt="logo">
    <h2>Alt Başlık 2</h2>
    <a href="sayfa.html" target="_blank">Yeni Sayfa</a>
    <button class="buton">Tıkla</button>
</div>
  • Tüm <h2> başlıklarının altını çizin (text-decoration: underline;).
  • id="logo" olan bir elemanın genişliğini 100 piksel yapın (width: 100px;).
  • class="buton" olan tüm elemanların arka plan rengini turuncu yapın (background-color: orange;).
  • target="_blank" niteliğine sahip tüm linklere (<a>) kesikli bir kenarlık verin (border: 1px dashed white;).
  • class="container" olan div'in doğrudan altındaki tüm <p> etiketlerinin rengini lightblue yapın.
  • Tüm <h1>, <h2>, ve <h3> etiketlerinin alt dış boşluğunu 10px yapın (margin-bottom: 10px;).

Sıra Sende! (Bölüm 2) ✍️

Şimdi de bu HTML yapısını stilize edin:

<nav>
  <ul>
    <li><a href="#anasayfa">Ana Sayfa</a></li>
    <li><a href="#hakkimizda">Hakkımızda</a></li>
  </ul>
</nav>
<section>
    <p>Bu ilk paragraf.</p>
    <h2>Bir Başlık</h2>
    <p>Bu ikinci paragraf.</p>
    <p>Bu üçüncü paragraf.</p>
</section>
  • nav elemanının içindeki tüm linklerin (<a>) altındaki çizgiyi kaldırın. (Soy seçici)
  • <ul> elemanının doğrudan altındaki <li> elemanlarına alttan 1px'lik kesikli bir kenarlık ekleyin. (Çocuk seçici)
  • <h2> başlığından hemen sonra gelen <p> elemanının yazı tipini kalın yapın. (Bitişik kardeş seçici)
  • <h2> başlığından sonra gelen tüm <p> kardeşlerinin rengini silver yapın. (Genel kardeş seçici)
  • href niteliği #anasayfa olan linkin rengini tomato yapın. (Nitelik seçici)
  • nav ve section etiketlerine aynı anda 10px'lik bir padding verin. (Gruplama seçici)

Renkler

CSS'te renkleri belirtmenin birkaç yolu vardır.

Örnek HTML Yapısı

<h1>Başlık 1</h1>
<h2>Başlık 2</h2>
<div class="overlay">
    <p class="vurgu">Bu bir paragraf.</p>
</div>
/* İsimlendirilmiş Renkler */
h1 { color: tomato; }

/* HEX Kodları */
h2 { color: #FF6347; } /* tomato ile aynı renk */

/* RGBA (Red, Green, Blue, Alpha - şeffaflık) */
.overlay { background-color: rgba(0, 0, 0, 0.5); } /* %50 şeffaf siyah */

/* HSL (Hue, Saturation, Lightness) */
.vurgu { color: hsl(240, 100%, 50%); } /* parlak mavi */

Çözümlü Örnekler 💡

  1. İsimlendirilmiş renk ile kenarlık rengi:

    div { border: 2px solid dodgerblue; }

  2. Kısa HEX kodu ile metin rengi:

    p { color: #f0c; } /* #ff00cc ile aynı */

  3. HSL ile arka plan rengi:

    .section { background-color: hsl(120, 60%, 70%); } /* açık yeşil */

  4. HSLA ile yarı şeffaf metin rengi:

    .sub-text { color: hsla(0, 0%, 100%, 0.75); } /* %75 şeffaf beyaz */

  5. `currentColor` anahtar kelimesi: Kenarlık rengini metin rengiyle aynı yapar.

    button { color: green; border: 1px solid currentColor; }

Sıra Sende! (Bölüm 1) 🎨

Aşağıdaki HTML yapısını kullanarak istenen stilleri uygulayın:

<p>Bu paragrafın rengi değişecek.</p>
<h2>Bu başlığın rengi değişecek.</h2>
<div id="kutu1">Arka plan rengi değişecek.</div>
<div id="kutu2">Şeffaf arka plan rengi olacak.</div>
<button>Buton</button>
  • Paragrafın rengini HEX kodu kullanarak #8A2BE2 (blueviolet) yapın.
  • id="kutu1" olan <div> elemanının arka plan rengini RGB kullanarak rgb(60, 179, 113) (mediumseagreen) yapın.
  • id="kutu2" olan <div> elemanının arka plan rengini RGBA kullanarak %70 şeffaf bir kırmızı yapın.
  • <h2> başlığının rengini isimlendirilmiş renk olarak gold yapın.
  • Bir <div> elemanına HSL kullanarak hsl(120, 70%, 80%) değerinde bir arka plan rengi verin.
  • <button> elemanının kenarlık rengini, %25 şeffaf siyah yapmak için RGBA kullanın.

Sıra Sende! (Bölüm 2) 🎨

Şimdi de bu HTML yapısını stilize edin:

<div class="kutu-a" style="height:50px; width:50px; margin:5px;"></div>
<div class="kutu-b" style="height:50px; width:50px; margin:5px;"></div>
<div class="kutu-c" style="height:50px; width:50px; margin:5px;"></div>
<div class="kutu-d" style="height:50px; width:50px; margin:5px;"></div>
<p class="metin">Bu bir metindir.</p>
  • .kutu-a'nın arka planını HEX kodu #4682B4 (steelblue) yapın.
  • .kutu-b'nin arka planını RGB rgb(218, 112, 214) (orchid) yapın.
  • .kutu-c'nin arka planını HSL hsl(25, 80%, 50%) (bir tür turuncu) yapın.
  • .kutu-d'nin arka planını RGBA ile %40 şeffaf yeşil yapın.
  • .metin paragrafının rengini isimlendirilmiş renk olarak wheat yapın.
  • .kutu-a'ya kısa HEX kodu #0f0 kullanarak 2px'lik bir kenarlık verin.

Arka Plan (Background)

Elemanların arka planını renklendirebilir veya görsel ekleyebilirsiniz.

Örnek HTML Yapısı

<div class="kutu">Bu kutunun arka planı stillendirilecek.</div>
.kutu {
  /* Arka plan rengi */
  background-color: #f5f5f5;

  /* Arka plan görseli */
  background-image: url('desen.png');

  /* Görselin tekrarlanma durumu */
  background-repeat: no-repeat; /* repeat, repeat-x, repeat-y */

  /* Görselin konumu */
  background-position: center top; /* x y ekseni */

  /* Görselin boyutu */
  background-size: cover; /* contain, 100px 50px */
  
  /* Tüm arka plan özelliklerini tek seferde yazma (kısa yol) */
  background: #f5f5f5 url('desen.png') no-repeat center top / cover;
}

Çözümlü Örnekler 💡

  1. Doğrusal Gradyan (Linear Gradient) Arka Plan:

    div { background-image: linear-gradient(to right, red, blue); }

  2. Sabitlenmiş Arka Plan Görseli: Sayfa kaysa bile görsel sabit kalır.

    body {
  background-image: url('manzara.jpg');
  background-attachment: fixed;
}

  3. Birden Fazla Arka Plan Görseli:

    div {
  background-image: url('logo.png'), url('desen.gif');
  background-repeat: no-repeat, repeat;
  background-position: center, top left;
}

  4. `background-size: contain`: Görselin tamamı görünecek şekilde kutuya sığdırılır.

    div {
  background-image: url('resim.jpg');
  background-repeat: no-repeat;
  background-size: contain;
}

  5. Yedek Renk (Fallback Color): Görsel yüklenemezse arka plan rengi görünür.

    div { background: lightblue url('bulunamayan-resim.jpg'); }

Sıra Sende! (Bölüm 1) 🏞️

Aşağıdaki HTML yapısını kullanarak istenen stilleri uygulayın:

<div id="resimli-kutu" style="height: 300px; border: 1px solid white;"></div>
  • id="resimli-kutu" olan <div> elemanının arka planına 'arkaplan.jpg' adında bir görsel ekleyin.
  • Görselin yatayda (x ekseninde) tekrar etmesini sağlayın (background-repeat: repeat-x;).
  • Görselin konumunu sağ alt köşe olarak ayarlayın (background-position: right bottom;).
  • Görselin kutuya tam sığacak şekilde, en-boy oranını koruyarak boyutlanmasını sağlayın (background-size: contain;).
  • Maviden mora doğru giden dikey bir doğrusal gradyan arka planı oluşturun.
  • Arka plan görselinin sayfa ile birlikte kaymamasını, sabit kalmasını sağlayın (background-attachment: fixed;).

Sıra Sende! (Bölüm 2) 🏞️

Şimdi de bu HTML yapısını stilize edin:

<section class="hero" style="height: 400px; color: white;">
  <h1>Hoş Geldiniz</h1>
</section>
  • .hero bölümüne background kısa yolu ile hem #222 arka plan rengini hem de 'hero.jpg' görselini ekleyin.
  • Görselin tekrar etmemesini (no-repeat) ve yatayda/dikeyde ortalanmasını (center center) sağlayın.
  • Görselin tüm section alanını kaplaması için background-size: cover; özelliğini ekleyin.
  • Kırmızı, sarı ve mavi renklerinden oluşan 45 derecelik bir doğrusal gradyan arka planı oluşturun.
  • .hero bölümüne ikinci bir arka plan görseli olarak sağ üstte konumlandırılmış, tekrar etmeyen bir 'logo.png' ekleyin.
  • Tüm arka plan özelliklerini tek bir background kısa yolu ile yazın.

Kutu Modeli (Box Model)

Web'deki her eleman, bir kutu olarak düşünülebilir. Kutu modeli, bu kutuların içeriğini, dolgusunu, kenarlığını ve dış boşluğunu tanımlar.

İçerik (Content)
Padding
Margin & Border
.kutu {
  width: 300px;
  height: 200px;

  /* İçerik ile kenarlık arasındaki boşluk */
  padding: 20px; /* Dört taraf da 20px */
  
  /* Kenarlık */
  border: 5px solid red;
  
  /* Elemanın dışındaki boşluk */
  margin: 30px;

  /* Bu özellik, padding ve border'ı genişliğe dahil eder, hayat kurtarır! */
  box-sizing: border-box;
}

Çözümlü Örnekler 💡

  1. Her taraf için farklı `padding`:

    /* üst, sağ, alt, sol */
div { padding: 10px 20px 30px 40px; }

  2. Yatayda ortalamak için `margin`:

    .ortala {
  width: 80%;
  margin-left: auto;
  margin-right: auto;
}

  3. Noktalı kenarlık stili (`dotted`):

    .kutu { border: 2px dotted green; }

  4. Sadece üst ve alt kenarlığı ayarlama:

    .kutu {
  border-top: 1px solid black;
  border-bottom: 1px solid black;
}

  5. Kenar yuvarlaklığı (`border-radius`):

    .yuvarlak-kose { border-radius: 15px; }

Sıra Sende! (Bölüm 1) 📦

Aşağıdaki HTML yapısını kullanarak istenen stilleri uygulayın:

<div class="alistirma-kutusu">Bu bir kutu.</div>
  • class="alistirma-kutusu" olan div'in genişliğini 200px, yüksekliğini 100px yapın.
  • Bu div'e 10px'lik bir padding verin.
  • Div'e 2px kalınlığında, düz (solid) ve siyah bir border ekleyin.
  • Div'in üst dış boşluğunu (margin-top) 15px yapın.
  • Tek bir padding özelliği ile üst/alt 10px, sağ/sol 20px iç boşluk verin.
  • Kutuyu yatayda ortalamak için margin özelliğini kullanın.

Sıra Sende! (Bölüm 2) 📦

Şimdi de bu HTML yapısını stilize edin:

<button class="buton">Gönder</button>
  • Butona 15px dikey ve 30px yatay padding verin.
  • Butonun sadece sol tarafına 5px kalınlığında, solid ve tomato renginde bir border ekleyin.
  • Butonun alt dış boşluğunu (margin-bottom) 20px yapın.
  • Butonun köşelerini 8px yuvarlaklaştırın (border-radius).
  • Butonun toplam genişliğinin padding ve border'dan etkilenmemesi için box-sizing: border-box; özelliğini ekleyin.
  • Butona max-width: 200px; özelliği ekleyin.

Metin Biçimlendirme

Metinlerin görünümünü kontrol etmek için birçok CSS özelliği vardır.

Örnek HTML Yapısı

<div class="metin-blogu">
    <h1>Makale Başlığı</h1>
    <p>Bu, makalenin içeriğidir.</p>
    <a href="#">Devamını Oku</a>
</div>
.metin-blogu {
  font-family: 'Helvetica', Arial, sans-serif;
  font-size: 16px;
  line-height: 1.6;
}
.metin-blogu h1 {
  font-size: 32px;
  font-weight: bold;
  letter-spacing: 1px;
}
.metin-blogu a {
  text-decoration: underline;
  color: blue;
}

Çözümlü Örnekler 💡

  1. Metni büyük harfe çevirme (`text-transform`):

    h1 { text-transform: uppercase; }

  2. Metin gölgesi ekleme (`text-shadow`):

    .baslik { text-shadow: 2px 2px 5px gray; }

  3. Kelime aralığını artırma (`word-spacing`):

    p { word-spacing: 10px; }

  4. Paragraf girintisi (`text-indent`):

    p { text-indent: 50px; }

  5. İtalik yazı stili (`font-style`):

    .vurgu { font-style: italic; }

Sıra Sende! (Bölüm 1) 📝

Aşağıdaki HTML yapısını kullanarak istenen stilleri uygulayın:

<h1>Hoş Geldiniz</h1>
<p>Bu bir deneme paragrafıdır.</p>
<span>Bu metin BÜYÜK harf olmalı.</span>
<a href="#">Daha Fazla Bilgi</a>
  • Sayfadaki tüm paragrafların (<p>) yazı tipini Verdana yapın.
  • <h1> başlığının yazı boyutunu 36px ve metin hizalamasını ortalı (center) yapın.
  • Linkin (<a>) altındaki çizgiyi kaldırın (text-decoration: none;).
  • Paragrafın harf aralığını (letter-spacing) 2px olarak ayarlayın.
  • <span> etiketindeki metni text-transform: uppercase; kullanarak büyük harfe çevirin.
  • <h1> başlığına hafif bir metin gölgesi ekleyin (text-shadow: 1px 1px 3px black;).

Sıra Sende! (Bölüm 2) 📝

Şimdi de bu alıntı metnini stilize edin:

<blockquote>
  Bu bir alıntı metnidir. Okunabilirliği artırmak için satır yüksekliğini ayarlamak önemlidir.
</blockquote>
  • <blockquote> elemanının yazı tipini (font-family) Georgia yapın.
  • Metnin yazı stilini italik (font-style: italic;) olarak ayarlayın.
  • Satır yüksekliğini (line-height) 1.8 yaparak satırlar arasındaki boşluğu artırın.
  • Yazı kalınlığını (font-weight) 300 (light) yapın.
  • Kelime aralığını (word-spacing) 4px olarak ayarlayın.
  • Metni her iki yana yaslayın (text-align: justify;).

Display Özelliği

display özelliği, bir elemanın nasıl davranacağını ve görüneceğini belirler. En temel değerleri block ve inline'dır.

Örnek HTML Yapısı

<div>Bu bir div (block).</div>
<span>Bu bir span (inline).</span>
<a href="#" class="buton">Buton (inline-block)</a>
<p class="gizli">Bu paragraf görünmeyecek.</p>
/* block: Tüm satırı kaplar, genişlik ve yükseklik alabilir. */
div { display: block; background-color: #274046; margin-bottom: 5px; }

/* inline: Sadece içeriği kadar yer kaplar, genişlik/yükseklik almaz. */
span { display: inline; background-color: #533342; }

/* inline-block: inline gibi davranır ama genişlik/yükseklik alabilir. */
.buton {
  display: inline-block;
  padding: 10px 20px;
  background-color: skyblue;
}

/* none: Elemanı tamamen gizler ve sayfa düzeninden kaldırır. */
.gizli { display: none; }

Çözümlü Örnekler 💡

  1. Yatay menü oluşturma: <li> elemanları normalde block'tur. inline-block ile yan yana dizilebilirler.

    nav ul li { display: inline-block; margin-right: 15px; }

  2. `visibility: hidden` vs `display: none`: `visibility: hidden` elemanı gizler ama kapladığı alan boş kalır. `display: none` ise elemanı tamamen kaldırır.

    .yer-kapla { visibility: hidden; }
.yok-ol { display: none; }

  3. Bir <div>'i liste elemanı gibi gösterme:

    div.liste-elemani { display: list-item; margin-left: 20px; }

  4. `inline` elemana `padding` etkisi: Dikey padding, diğer satırların üzerine taşabilir.

    span { display: inline; padding: 20px; background: orange; }

  5. Tablo düzeni oluşturma: `display` ile elemanlara tablo özellikleri verilebilir.

    .tablo { display: table; }
.satir { display: table-row; }
.hucre { display: table-cell; border: 1px solid white; }

Sıra Sende! (Bölüm 1) 🧱

Aşağıdaki HTML yapısını kullanarak istenen stilleri uygulayın:

<span class="blok-yap">Beni blok yap.</span>
<ul>
  <li>Ana Sayfa</li><li>Hakkımızda</li><li>İletişim</li>
</ul>
<div class="gosterilmesin">Beni gizle.</div>
<div class="yer-tutsun">Beni gizle ama yerimi koru.</div>
  • class="blok-yap" olan <span> elemanına display: block; vererek bir alt satıra geçmesini sağlayın.
  • <ul> içindeki <li> elemanlarını display: inline-block; ile yan yana dizerek bir menü oluşturun.
  • class="gosterilmesin" olan elemanı display: none; ile tamamen gizleyin.
  • <a> etiketine display: inline-block; vererek, ona padding ve background-color uygulayın.
  • class="yer-tutsun" olan elemanı visibility: hidden; ile gizleyin. display: none ile arasındaki farkı gözlemleyin.
  • Üç <span> elemanını bir <div> içine koyun ve span'ları display: block; ile alt alta sıralayın.

Şimdi de bu HTML yapısını stilize edin:

  <div class="satir">
    <div class="hucre">Hücre 1.1</div>

  • Bir paragraf içindeki <strong> etiketini display: inline-block; yaparak ve transform uygulayarak üzerine gelindiğinde hafifçe büyütün.

Flexbox

Flexbox, kutuları bir satır veya sütun içinde hizalamak, sıralamak ve dağıtmak için kullanılan güçlü bir yerleşim modelidir. Modern ve esnek tasarımlar için vazgeçilmezdir.

Örnek HTML Yapısı

<div class="kapsayici">
  <div class="kutu kutu1">1</div>
  <div class="kutu kutu2">2</div>
  <div class="kutu">3</div>
</div>
/* Kapsayıcı Eleman (Flex Container) */
.kapsayici {
  display: flex;
  justify-content: center; /* Yatay eksende ortala */
  align-items: center;    /* Dikey eksende ortala */
  flex-wrap: wrap;        /* Elemanlar sığmazsa alt satıra indir */
  height: 200px;
  background-color: #1e293b;
}

/* İç Elemanlar (Flex Items) */
.kutu { background-color: #0ea5e9; margin: 5px; padding: 20px; color: white; }
.kutu1 { flex-grow: 1; } /* Boş alanı doldurmak için 1 oranında büyür */
.kutu2 { flex-grow: 2; } /* Boş alanı doldurmak için 2 oranında büyür */

Çözümlü Örnekler 💡

  1. Elemanları ters sırada dizme (`flex-direction`):

    .kapsayici { display: flex; flex-direction: row-reverse; }

  2. Tek bir elemanı farklı hizalama (`align-self`):

    .kapsayici { display: flex; align-items: center; }
.farkli-eleman { align-self: flex-end; } /* Kapsayıcının sonuna hizalanır */

  3. Elemanların sırasını değiştirme (`order`):

    .eleman1 { order: 2; }
.eleman2 { order: 1; }

  4. Esneklik (`flex` shorthand): `flex-grow`, `flex-shrink`, `flex-basis` özelliklerini bir arada tanımlar.

    .kutu { flex: 1 1 100px; } /* Büyüyebilir, küçülebilir, başlangıç boyutu 100px */

  5. Dikeyde ortalanmış ve yatayda aralıklı menü:

    nav {
  display: flex;
  justify-content: space-around;
  align-items: center;
}

Sıra Sende! (Bölüm 1) 📐

Aşağıdaki HTML yapısını kullanarak istenen stilleri uygulayın:

<div class="flex-kapsayici">
  <div class="item-a">A</div>
  <div class="item-b">B</div>
  <div class="item-c">C</div>
</div>
  • .flex-kapsayici'ya display: flex; verin.
  • Bu elemanları yatayda (justify-content) aralarında eşit boşluk olacak şekilde (space-between) hizalayın.
  • Elemanları dikeyde (align-items) ortalayın (center).
  • Kapsayıcının yönünü (flex-direction) dikey (column) olarak değiştirin.
  • Ortadaki elemanın (.item-b) diğerlerine göre 2 kat daha fazla büyümesini sağlayın (flex-grow: 2;).
  • Elemanların görsel sırasını order özelliği ile C, A, B olacak şekilde değiştirin.

Şimdi de bu kart yapısını stilize edin:

  • .baslik ve .alt-bilgi'nin küçülmemesini sağlayın (flex-shrink: 0;).
  • Tüm elemanları yatayda ortalamak için align-items: center; kullanın.
  • .alt-bilgi'nin tek başına dikeyde en sona yaslanmasını sağlayın (align-self: flex-end;).

Grid

CSS Grid, hem satır hem de sütunları aynı anda kontrol edebildiğiniz iki boyutlu bir yerleşim sistemidir. Karmaşık sayfa düzenleri için Flexbox'tan daha güçlü olabilir.

Örnek HTML Yapısı

<div class="grid-kapsayici">
  <div class="kutu header">Header</div>
  <div class="kutu sidebar">Sidebar</div>
  <div class="kutu main">Main Content</div>
  <div class="kutu footer">Footer</div>
</div>
/* Kapsayıcı Eleman (Grid Container) */
.grid-kapsayici {
  display: grid;
  grid-template-columns: 1fr 3fr; /* 1'e 3 oranında 2 sütun */
  gap: 10px;
}
.kutu { background-color: #1e293b; padding: 20px; border-radius: 5px; }

/* İç Elemanlar (Grid Items) */
.header { grid-column: 1 / 3; } /* 1. sütunda başla, 3.den önce bitir (2 sütun kapla) */
.footer { grid-column: 1 / 3; }

Çözümlü Örnekler 💡

  1. `grid-template-areas` ile sayfa düzeni:

    .kapsayici {
  display: grid;
  grid-template-areas:
    "baslik baslik"
    "menu ana-icerik"
    "alt-bilgi alt-bilgi";
}
.header { grid-area: baslik; }
.sidebar { grid-area: menu; }
/* ... ve diğerleri */

  2. `span` ile satır birleştirme: Bir elemanın birden fazla satır kaplamasını sağlar.

    .uzun-eleman { grid-row: span 2; }

  3. `minmax()` ile esnek sütunlar: Sütun genişliğinin minimum ve maksimum değerlerini belirler.

    .grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); }

  4. Grid elemanlarını hizalama:

    .kapsayici {
  display: grid;
  justify-items: center; /* Yatayda ortala */
  align-items: center;   /* Dikeyde ortala */
}

  5. İsimlendirilmiş grid çizgileri:

    .grid {
  display: grid;
  grid-template-columns: [sol-basla] 1fr [sol-bitis sag-basla] 1fr [sag-bitis];
}
.eleman { grid-column: sol-basla / sag-bitis; }

Sıra Sende! (Bölüm 1) ▦

Aşağıdaki HTML yapısını kullanarak istenen stilleri uygulayın:

  <div class="item1">1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
  <div class="item5">5</div>
  <div>6</div>
</div>
  • .grid-alistirma'ya display: grid; verin.
  • Bu grid'i 3 eşit sütuna bölün (grid-template-columns: repeat(3, 1fr);).
  • Grid elemanları arasına 15px'lik bir boşluk (gap) ekleyin.
  • İlk elemanın (.item1) 2 sütun kaplamasını sağlayın (grid-column: span 2;).
  • Grid'i 2 satırdan oluşacak şekilde tanımlayın (grid-template-rows: 100px 100px;).
  • Beşinci elemanın (.item5) ikinci satırda başlamasını sağlayın (grid-row-start: 2;).

Position Özelliği

position özelliği, bir elemanın sayfa üzerindeki konumlandırma yöntemini belirler. top, right, bottom, left özellikleri ile kullanılır.

Örnek HTML Yapısı

<div class="kapsayici">
  <div class="kutu relative">Relative</div>
  <div class="kutu absolute">Absolute</div>
</div>
<div class="kutu fixed">Fixed</div>
<header class="sticky">Sticky Header</header>

.kapsayici { position: relative; height: 150px; border: 2px dashed gray; }

/* relative: Normal konumuna göre 20px aşağı kaydırılır. */
.relative { position: relative; top: 20px; }
  
/* absolute: Kapsayıcı'nın sağ üst köşesine konumlanır. */
.absolute { position: absolute; top: 0; right: 0; }
  
/* fixed: Tarayıcı penceresinin sağ alt köşesine sabitlenir. */
.fixed { position: fixed; bottom: 10px; right: 10px; }

/* sticky: Sayfa kaydırıldığında üstte yapışık kalır. */
.sticky { position: sticky; top: 0; background-color: #334155; padding: 10px;}

Çözümlü Örnekler 💡

  1. `absolute` ile ortalama: Bir elemanı hem dikey hem yatayda ortalamak için kullanılır.

    .kapsayici { position: relative; }
.ortalanmis {
  position: absolute;
  top: 50%; left: 50%;
  transform: translate(-50%, -50%);
}

  2. `z-index` ile katman sırası: `position` değeri olan elemanların üst üste gelme sırasını belirler. Yüksek değer en üstte olur.

    .arkada { position: absolute; z-index: 1; }
.onde { position: absolute; z-index: 2; }

  3. Overlay (Kaplama) efekti: Sayfanın üzerine yarı şeffaf bir katman eklemek.

    .overlay {
  position: fixed; top: 0; left: 0;
  width: 100%; height: 100%;
  background: rgba(0,0,0,0.7);
  z-index: 10;
}

  4. `static` (varsayılan) vs `relative`: `static` elemanlar `top/left` gibi özelliklerden etkilenmez. `relative` ise normal akışta kalır ama kaydırılabilir.

    .static { position: static; top: 50px; } /* çalışmaz */
.relative { position: relative; top: 50px; } /* çalışır */

  5. `sticky` kenar çubuğu:

    .sidebar { position: sticky; top: 20px; }

Sıra Sende! (Bölüm 1) 📌

Aşağıdaki HTML yapısını kullanarak istenen stilleri uygulayın:

<div id="goreceli">Ben relative'im.</div>
<div id="sabit">Ben fixed'im.</div>
<div id="kapsayici" style="position: relative; height: 100px; border: 1px solid;">
  <div id="mutlak">Ben absolute'um.</div>
  <div id="ustte">Üstteyim</div>
  <div id="altta">Alttayım</div>
</div>
<h2 id="yapiskan">Yapışkan Başlık</h2>
  • #goreceli elemanını position: relative; yaparak normal konumundan 10px aşağı kaydırın.
  • #sabit elemanını sayfanın sağ üst köşesine sabitleyin (fixed).
  • #kapsayici'ya position: relative; ve içindeki #mutlak elemanına position: absolute; vererek, iç elemanı kapsayıcının sol alt köşesine yerleştirin.
  • #ustte ve #altta elemanlarını üst üste bindirin ve z-index kullanarak #ustte elemanının görünmesini sağlayın.
  • Uzun bir sayfa oluşturduğunuzu varsayarak, #yapiskan başlığın sayfa kaydırıldığında en üste yapışıp kalmasını sağlayın (position: sticky; top: 0;).
  • Bir input alanının içine (sağ üst köşesine) ikon gibi bir <div> yerleştirin. (İpucu: input'u bir kapsayıcıya alın).

Şimdi de bu bildirim kutusunu stilize edin:

  <img src="avatar.png" class="avatar">
  <span class="bildirim-sayaci">3</span>

  • Bir <footer> elemanının, position: sticky; ve bottom: 0; ile sayfanın en altına yapışmasını sağlayın.

Sözde Sınıflar (Pseudo-classes)

Sözde sınıflar, bir elemanın özel bir durumunu (örneğin fareyle üzerine gelindiğinde) seçmek için kullanılır. : ile başlarlar.

Örnek HTML Yapısı

<a href="#">Bir Link</a>
<button>Tıkla Bana</button>
<input type="text" placeholder="Odaklan...">
<ul>
  <li>Öğe 1</li>
  <li>Öğe 2</li>
  <li>Öğe 3</li>
</ul>
/* Fareyle üzerine gelinen link */
a:hover { color: red; }

/* Tıklanma anındaki buton */
button:active { transform: scale(0.98); }

/* Odaklanılmış input alanı */
input:focus { border-color: skyblue; outline: none; }

/* Listedeki ilk eleman */
li:first-child { font-weight: bold; }

/* Listedeki tek sayılı elemanlar */
li:nth-child(odd) { background-color: #1e293b; }

Çözümlü Örnekler 💡

  1. Ziyaret edilmiş ve edilmemiş linkler:

    a:link { color: blue; } /* Ziyaret edilmemiş */
a:visited { color: purple; } /* Ziyaret edilmiş */

  2. Pasif (`disabled`) butonları biçimlendirme:

    button:disabled { background-color: gray; cursor: not-allowed; }

  3. Her üç elemanda bir (`nth-child`):

    li:nth-child(3n) { background-color: darkred; }

  4. Boş olmayan (`:not(:empty)`) paragraflar:

    p:not(:empty) { border: 1px solid gray; }

  5. `::before` ve `::after` Sözde Elemanları: Bir elemanın içeriğinden önce veya sonra içerik ekler.

    a[target="_blank"]::after { content: ' 🔗'; }

</ul>

<a href="dosya.pdf">Rapor (PDF)</a>

  • .focus-input alanına tıklandığında (:focus) kenarlığının (border) 2px olmasını sağlayın.
  • .liste'deki son elemanın (:last-child) yazı rengini turuncu yapın.
  • Paragrafların ilk harfini ::first-letter sözde elemanı ile büyütün ve renklendirin.

Geçişler (Transitions)

transition özelliği, bir CSS özelliğinin değerinin zaman içinde yumuşak bir şekilde değişmesini sağlar. Bu, animasyonlu efektler oluşturmanın basit bir yoludur.

Örnek HTML Yapısı

<button class="buton">Yumuşak Geçiş</button>
.buton {
  background-color: blue;
  padding: 15px 30px;
  color: white;
  border: none;
  
  /* Kısa yol: Hangi özellik, ne kadar sürede, nasıl bir hızla değişecek */
  transition: background-color 0.4s ease-in-out, transform 0.3s ease;
}

.buton:hover {
  background-color: red;
  transform: translateY(-5px); /* Üzerine gelince 5px yukarı kaydır */
}

Çözümlü Örnekler 💡

  1. Tüm özelliklere geçiş uygulama:

    div { transition: all 0.5s ease; }

  2. Geçiş gecikmesi (`transition-delay`): Animasyonun başlamadan önce bekleme süresi.

    div:hover { width: 300px; }
div { transition: width 1s; transition-delay: 0.5s; }

  3. Hız eğrisi (`transition-timing-function`): Geçişin hızlanma ve yavaşlama şekli.

    div { transition: transform 1s linear; } /* Sabit hız */

  4. Kutu gölgesine geçiş (`box-shadow`):

    .kutu {
  box-shadow: 2px 2px 5px black;
  transition: box-shadow 0.3s;
}
.kutu:hover { box-shadow: 5px 5px 15px black; }

  5. Renk ve Opaklık geçişi:

    .kutu {
  background: blue; opacity: 1;
  transition: background 0.5s, opacity 0.5s;
}
.kutu:hover { background: red; opacity: 0.5; }

Sıra Sende! (Bölüm 1) 🪄

Aşağıdaki HTML yapısını kullanarak istenen stilleri uygulayın:

<div class="gecisli-kutu"></div>
  • .gecisli-kutu elemanına bir başlangıç genişliği ve yüksekliği verin.
  • Bu kutunun üzerine fareyle gelindiğinde (:hover) genişliğinin (width) artmasını sağlayın.
  • Bu genişlik değişimine 0.5 saniyelik bir transition ekleyerek yumuşak bir animasyon oluşturun.
  • Kutunun opaklığının (opacity) da 1'den 0'a düşmesini sağlayın ve buna 1 saniyelik bir geçiş verin.
  • Genişlik geçişinin başlaması için 0.2 saniyelik bir gecikme (transition-delay) ekleyin.
  • Kutunun üzerine gelindiğinde 45 derece dönmesini (transform: rotate(45deg);) sağlayın ve transform özelliğine yumuşak bir geçiş ekleyin.
