C# Temel Kavramlar Soru & Cevap

1- C# dilinin .NET Framework ile ilişkisi nedir? Kısaca Açıklayınız.

Cevap: C#, .NET Framework üzerinde çalışan bir programlama dilidir. C# ile yazılan kodlar, .NET Framework tarafından sağlanan sınıf kütüphaneleri ve çalışma zamanı ortamı (CLR) ile çalışır.

2- Olaylar (Events) kavramı nedir? Kısaca açıklayınız.

Cevap: Olaylar, kullanıcı etkileşimleri (örneğin, bir butona tıklama) veya sistem bildirimleri gibi belirli durumları izler ve bu durumlar gerçekleştiğinde bir işlem (metot) yapılmasını sağlar.

3- try-catch-finally bloğu ne işe yarar?

Cevap: try bloğu hata oluşabilecek kodları içerir, catch bloğu try bloğunda bir hata oluşursa yakalar ve işler, finally bloğu ise hata olsa da olmasa da her durumda çalışacak kodları içerir (kaynak temizleme vb.).

4- Aşağıdaki işlemin sonucunu hesaplayınız:

(5 + 3 * 2) / (8 - 6) + 7 % 4

Adımlar: (5 + 6) / (2) + (7 % 4) -> (11 / 2) + 3 -> 5 + 3 (Tamsayı bölmesi) -> 8

Sonuç: 8 (Not: İşlem önceliğine göre önce parantez içleri ve çarpma/bölme/mod yapılır. 11/2 işlemi C#'ta tamsayı bölmesi olduğu için sonuç 5'tir, 5.5 değil.)

5- Aşağıdaki işlemin sonucunu hesaplayınız:

!(true && (false || true)) || (false && true)

Adımlar: !(true && true) || (false) -> !(true) || false -> false || false

Sonuç: false

6- Sınıf (Class) nedir? Kısaca açıklayınız.

Cevap: Sınıflar, nesnelerin özelliklerini (değişkenler) ve davranışlarını (metotlar) tanımlayan şablonlardır. Form uygulamalarındaki 'Form' veya 'Button' birer sınıftır.

7- Bir Form uygulamasında butona tıklandığında kod çalışmasını nasıl sağlarsınız?

Cevap: Buton nesnesine çift tıklanır. Bu, otomatik olarak butonun Click (Tıklama) olayına (event) bağlı bir metot oluşturur. Çalışması istenen kodlar bu metodun süslü parantezleri { } içine yazılır.

8- namespace (İsim Uzayı) anahtar kelimesinin amacı nedir?

Cevap: Sınıfları ve diğer türleri mantıksal olarak gruplandırmak için kullanılır. Büyük projelerde aynı isme sahip sınıfların çakışmasını engeller (Örn: System.Windows.Forms).

9- Değer Türü (Value Type) ile Referans Türü (Reference Type) arasındaki temel fark nedir?

Cevap: Değer Türleri (int, bool, double) veriyi doğrudan bellekte (Stack) saklar. Referans Türleri (string, object, Sınıflar) ise verinin bellekteki (Heap) adresini saklar.

10- Değişken Kapsamı (Variable Scope) nedir?

Cevap: Bir değişkenin programın hangi bölümlerinden erişilebilir olduğunu belirleyen kuraldır. Örneğin, bir metodun içinde { } tanımlanan bir değişken, o metodun dışından kullanılamaz.

11- if-else karar yapısının temel amacı nedir?

Cevap: Belirli bir mantıksal koşulun (true) veya (false) olmasına bağlı olarak farklı kod bloklarının çalıştırılmasını sağlar. Programın akışını yönlendirir.

12- switch-case ifadesi hangi durumlarda if-else yapısına tercih edilir?

Cevap: Tek bir değişkenin alabileceği çok sayıda sabit değere (örn: 1, 2, 3 veya 'A', 'B', 'C') göre farklı işlemler yapılması gerektiğinde tercih edilir. Kodu daha okunabilir hale getirir.

13- for döngüsünün yazım formatını (sözdizimini) açıklayınız.

Cevap: for(başlangıç; koşul; artış/azalış) { ... } şeklinde kullanılır.
1. Başlangıçta bir sayaç tanımlanır (örn: int i = 0).
2. Koşul sağlandığı sürece döngü çalışır (örn: i < 10).
3. Her döngü sonunda sayaç güncellenir (örn: i++).

14- break; komutu döngülerde ne amaçla kullanılır?

Cevap: İçinde bulunduğu döngü (for, while) veya switch bloğunu anında sonlandırmak için kullanılır. Döngünün normal koşulu beklenmeden dışarı çıkılır.

15- Hata ayıklamada (debugging) 'Breakpoint' (Kesme Noktası) kullanmanın amacı nedir?

Cevap: Programın çalışmasını kodun belirli bir satırında geçici olarak durdurmak için kullanılır. Bu noktada, programcı değişkenlerin o anki değerlerini inceleyebilir ve programın akışını adım adım izleyerek hatayı tespit edebilir.

16- Form uygulamasında lblBaslik isimli bir Label nesnesinin metnini "Merhaba" olarak nasıl değiştirirsiniz?

Cevap: Nesnenin Text özelliği kullanılır. Kod: lblBaslik.Text = "Merhaba";

17- Kod dosyasının başına yazılan using System; ifadesinin amacı nedir?

Cevap: System isim uzayındaki (namespace) sınıfları (Console, Math vb.) kullanırken başına System. yazma zorunluluğunu kaldırır. Kodu kısaltır.

18- int ve string veri tipleri arasındaki temel fark nedir?

Cevap: int (tamsayı), 10, -5, 0 gibi sayısal verileri saklar ve matematiksel işlemlerde kullanılır. string (metin), "Merhaba", "123" gibi metinsel ifadeleri saklar.

19- Tür Dönüşümü (Type Casting) nedir? string tipindeki "50" değerini int tipine nasıl dönüştürürsünüz?

Cevap: Bir veri tipindeki değişkenin başka bir veri tipine dönüştürülmesidir.
Kod: string metin = "50";
int sayi = Convert.ToInt32(metin); (veya int.Parse(metin))

20- (10 == 10) ifadesinin sonucu nedir? == operatörü ne tür bir operatördür?

Cevap: Sonuç true (doğru)'dur. == operatörü bir "Karşılaştırma Operatörüdür" ve iki değerin birbirine eşit olup olmadığını kontrol eder.

21- while döngüsü nedir ve for döngüsünden temel farkı nedir?

Cevap: while döngüsü, parantez içindeki koşul true olduğu sürece çalışan bir döngüdür. for döngüsü genellikle tekrar sayısı belli olduğunda (örn: 10 kez dön) kullanılırken, while döngüsü tekrar sayısı belli olmadığında (örn: kullanıcı 'q' tuşuna basana kadar dön) tercih edilir.

22- Aşağıdaki mantıksal işlemin sonucunu hesaplayınız:
(true || false) && (5 > 3)

Adımlar: (true) && (true)

Sonuç: true (|| VEYA demektir, biri true ise sonuç true'dur. && VE demektir, ikisi de true ise sonuç true'dur.)

23- continue; komutu döngülerde ne amaçla kullanılır?

Cevap: Döngünün o anki adımını (iterasyon) atlamak için kullanılır. continue görüldüğünde, döngü bloğunun geri kalanı çalıştırılmaz ve döngü bir sonraki adımdan (veya koşul kontrolünden) devam eder. break gibi döngüyü kırmaz.

24- 'Yazım Hatası' (Syntax Error) nedir?

Cevap: Programlama dilinin kurallarına uymayan kod yazımıdır (Örn: noktalı virgülü unutmak, parantezi kapatmamak). Bu hatalar programın derlenmesini (çalıştırılmasını) engeller.

25- 'Mantık Hatası' (Logic Error) nedir?

Cevap: Programın dil kurallarına göre doğru yazıldığı (syntax hatası olmadığı) ancak programcının hatası yüzünden beklenmedik veya yanlış sonuçlar ürettiği durumlardır (Örn: + yerine - kullanmak, yanlış koşul yazmak). Bulunması en zor hata türüdür.

26- Form uygulamasında TextBox (Metin Kutusu) nesnesinin amacı nedir?

Cevap: Kullanıcıdan metin veya sayı gibi klavye girdileri almak için kullanılır. (Örn: kullaniciAdi = txtKullaniciAdi.Text;)

27- bool (Boolean) veri tipi nedir ve hangi değerleri alabilir?

Cevap: Mantıksal bir veri tipidir. Yalnızca true (doğru) veya false (yanlış) değerlerini alabilir. Genellikle if gibi karar yapılarının koşullarında kullanılır.

28- Artırma (++) ve Azaltma (--) operatörleri ne işe yarar? int a = 5; a++; kodundan sonra a'nın değeri ne olur?

Cevap: Bir tamsayı değişkenin değerini 1 artırmak (++) veya 1 azaltmak (--) için kullanılır. Kod çalıştıktan sonra a'nın değeri 6 olur.

29- Atama Operatörü (=) ile Eşitlik Operatörü (==) arasındaki fark nedir?

Cevap: Atama operatörü (=), sağdaki değeri soldaki değişkene atar (Örn: int a = 10;). Eşitlik operatörü (==), iki değerin birbirine eşit olup olmadığını kontrol eder ve sonuç olarak true veya false döndürür (Örn: if (a == 10)).

30- Mantıksal DEĞİL (!) operatörü ne işe yarar? !(5 > 3) işleminin sonucu nedir?

Cevap: Bir mantıksal ifadenin sonucunu tersine çevirir (true ise false, false ise true yapar). (5 > 3) ifadesi true olduğundan, !(true) işleminin sonucu false olur.

31- İç içe if (Nested if) yapısı nedir?

Cevap: Bir if veya else bloğunun içine başka bir if bloğu yazılmasıdır. Bir koşul sağlandıktan *sonra* başka bir koşulu daha kontrol etmek için kullanılır.

32- do-while döngüsünün while döngüsünden farkı nedir?

Cevap: do-while döngüsünde koşul, döngü bloğu *bir kez çalıştıktan sonra* kontrol edilir. Bu nedenle, koşul en başta false olsa bile döngü bloğu her zaman en az bir kez çalışır. while döngüsü ise koşul en başta false ise hiç çalışmayabilir.

33- 'Sonsuz Döngü' (Infinite Loop) nedir ve for döngüsü ile nasıl oluşur?

Cevap: Döngünün durma koşulunun hiçbir zaman sağlanamaması durumudur. Programın kilitlenmesine neden olur. Örnek: for(int i = 0; i >= 0; i++) { ... } (i her zaman 0'dan büyük veya eşit kalacaktır).

34- Hata ayıklamada 'Adım Adım Çalıştırma' (Step Over / F10) tekniği nedir?

Cevap: Programı breakpoint'te durdurduktan sonra, bir sonraki kod satırına geçmek için kullanılır. Eğer satır bir metot çağrısıysa, metodun içine girmeden (Step Into) metodun tamamını çalıştırır ve bir sonraki satıra geçer.

35- Form uygulamasında txtGirdi isimli TextBox'tan alınan metni lblSonuc isimli Label'a nasıl yazdırırsınız?

Cevap: lblSonuc.Text = txtGirdi.Text;

a++; 
Console.WriteLine(a);

Cevap: 6 (a++ operatörü 'a'nın değerini 1 artırır.)

37- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int x = 10; 
x -= 3; 
Console.WriteLine(x);

Cevap: 7 (x -= 3 ifadesi x = x - 3 demektir.)

38- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

Console.WriteLine(10 / 4);

Cevap: 2 (İki tamsayı (int) bölündüğünde, C# ondalık kısmı atar ve sonucu tamsayı (int) olarak verir.)

39- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

bool x = true; 
Console.WriteLine(!x);

Cevap: false (! (DEĞİL) operatörü true değerini false yapar.)

40- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int yas = 15; 
if (yas >= 18) 
{ 
    Console.WriteLine("Girebilir"); 
} 
else 
{ 
    Console.WriteLine("Giremez"); 
}

Cevap: Giremez (Koşul (15 >= 18) false olduğu için else bloğu çalışır.)

41- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

for (int i = 0; i < 2; i++) 
{ 
    Console.Write("Test "); 
}

Cevap: Test Test (Döngü i=0 ve i=1 için (2 kez) çalışır. Console.Write alt satıra inmez.)

42- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

Console.WriteLine(5 == 5);

Cevap: true (== (Eşitlik) operatörü, iki değerin eşit olup olmadığını kontrol eder. 5, 5'e eşittir.)

43- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int a = 10; 
int b = 3; 
Console.WriteLine(a % b);

Cevap: 1 (% (Mod) operatörü, bölme işleminden kalanı verir. 10'un 3'e bölümünden kalan 1'dir.)

44- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int a = 5; 
int b = ++a; 
Console.WriteLine(b);

Cevap: 6 (++a (ön ek artırma) operatörü, önce 'a'nın değerini 6 yapar, *sonra* bu yeni değeri 'b'ye atar.)

45- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

bool x = true; 
bool y = false; 
Console.WriteLine(x && (y || true));

Cevap: true (Adımlar: (y || true) -> true olur. (x && true) -> (true && true) -> true olur.)

46- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int not = 75; 
if (not >= 80) 
{ 
    Console.WriteLine("A"); 
} 
else if (not >= 70) 
{ 
    Console.WriteLine("B"); 
} 
else 
{ 
    Console.WriteLine("C"); 
}

Cevap: B (İlk koşul (75 >= 80) false'tur. İkinci koşul (75 >= 70) true'dur, 'B' yazar ve blok sonlanır.)

47- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int gun = 9; 
switch(gun) 
{ 
    case 1: 
        Console.WriteLine("Pzt"); 
        break; 
    case 7: 
        Console.WriteLine("Pzr"); 
        break; 
    default: 
        Console.WriteLine("Hata"); 
        break; 
}

Cevap: Hata (Değer (9), case 1 veya case 7 ile eşleşmediği için default (varsayılan) bloğu çalışır.)

48- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int sayac = 0; 
while (sayac < 3) 
{ 
    sayac++; 
} 
Console.WriteLine(sayac);

Cevap: 3 (Döngü 0, 1, 2 için çalışır, 'sayac'ı 1, 2, 3 yapar. sayac 3 olunca (3 < 3) koşulu false olur, döngü durur ve 3 yazar.)

49- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

for (int i = 0; i < 5; i++) 
{ 
    if (i == 2) break; 
    Console.Write(i); 
}

Cevap: 01 (i=0 yazar, i=1 yazar, i=2 olduğunda break komutu döngüyü tamamen sonlandırır.)

50- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int a = 5; 
int b = a++; 
Console.WriteLine(a + "," + b);

Cevap: 6,5 (a++ (son ek artırma) operatörü, 'a'nın *o anki* değerini (5) 'b'ye atar. *Sonra* 'a'nın değerini 6 yapar.)

51- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

Console.WriteLine(10 / 4.0);

Cevap: 2.5 (Sayıların tamsayı (int) olması yerine, 4.0 bir double (ondalıklı) olduğu için C# tamsayı bölmesi *yapmaz*, ondalıklı bölme yapar.)

52- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int x = 10; 
if (x > 5 && x < 10) 
{ 
    Console.WriteLine("İçeride");

else 
{ 
    Console.WriteLine("Dışarıda");

Cevap: Dışarıda (Koşulun true olması için (x > 5) VE (x < 10) olması gerekir. (10 < 10) koşulu false olduğu için else bloğu çalışır.)

53- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar? (C# 8.0+)

int seviye = 3; 
string s = seviye switch 
{ 
    > 5 => "Yüksek", 
    < 5 => "Düşük", 
    _ => "Orta"

Cevap: Düşük (Modern switch ifadesi kullanılır. seviye (3), < 5 koşuluyla eşleşir.)

54- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int toplam = 0; 
for (int i = 0; i < 5; i++) 
{ 
    if (i % 2 == 0) continue; 
    toplam += i; 
} 
Console.WriteLine(toplam);

Cevap: 4 (continue, i çift olduğunda (0, 2, 4) o adımı atlar. toplam'a sadece tek sayılar (1, 3) eklenir. Toplam = 1 + 3 = 4.)

55- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int i = 5; 
do 
{ 
    Console.Write(i); 
    i++; 
} while (i < 5);

Cevap: 5 (do-while döngüsü, koşul yanlış (false) olsa bile en az bir kez çalışır. 5'i yazar, i=6 olur, koşul (6 < 5) kontrol edilir ve döngü durur.)

56- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

string ad = "Ali";
if (ad != "Veli")
{
    Console.WriteLine("Hoşgeldin");
}

Cevap: Hoşgeldin (!= (Eşit Değil) operatörüdür. "Ali", "Veli"ye eşit olmadığı için koşul true olur ve if bloğu çalışır.)

57- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

for (int i = 3; i > 0; i--)
{
    Console.Write(i + " ");
}

Cevap: 3 2 1 (Döngü i=3, i=2 ve i=1 için çalışır. i-- operatörü sayacı azaltır. i=0 olduğunda (0 > 0) koşulu false olur ve döngü durur.)

61- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

if (sayi > 5)
{
    Console.Write("A");
    if (sayi == 10)
    {

Cevap: AB (İlk if (10 > 5) true olur, 'A' yazar. İçteki if (10 == 10) da true olur, 'B' yazar. else bloğu atlanır.)

62- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int kod = 1;
switch (kod)
{
    case 1:
        Console.Write("Ekle ");
        // break unutuldu!
    case 2:
        Console.Write("Düzenle ");
        break;
    case 3:
        Console.Write("Sil");
        break;
}

Cevap: Ekle Düzenle (kod = 1 olduğu için case 1 çalışır, "Ekle " yazar. case 1'de break olmadığı için kod bir sonraki case'e (fall-through) düşer, case 2'yi de çalıştırır, "Düzenle " yazar ve break görerek durur.)

63- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int a = 5;
a = a++;
Console.WriteLine(a);

Cevap: 5 (a++ (son ek artırma) operatörü, 'a'nın o anki değerini (5) alır ve atama işlemi için kullanır. Atama bittikten *sonra* 'a'nın değeri geçici olarak 6 olur, ancak a = 5 ataması zaten gerçekleştiği için 'a'nın değeri 5 olarak kalır. Bu, C#'ta kafa karıştırıcı bir kullanımdır.)

64- Aşağıdaki kod bloğu çalıştırıldığında konsola ne yazar?

int sayac = 0;
while (true) // Sonsuz döngü
{
    sayac++;
    if (sayac == 3)
    {
        break; // Döngüyü kır
    }
}
Console.WriteLine(sayac);

Cevap: 3 (while(true) bir sonsuz döngü başlatır. s=1 olur, s=2 olur, s=3 olur. if (3 == 3) koşulu true olunca break çalışır ve döngü sonlanır. sayac'ın son değeri 3'tür.)

65- Aşağıdaki mantıksal işlemin sonucu nedir?

(true && false) || !(false || false)

Adımlar: (false) || !(false) -> false || true

Sonuç: true (|| (VEYA) operatöründe, ifadelerden birinin true olması sonucu true yapar.)
