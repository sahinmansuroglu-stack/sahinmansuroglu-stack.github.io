Giriş - Neler Öğreneceksiniz?

Merhaba, Kotlin öğrenme yolculuğunuza hoş geldiniz! 🌟 Bu doküman, sıfırdan başlayarak Kotlin'in temel ve ileri seviye özelliklerini keşfetmeniz için özenle hazırlandı. Her konu, anlaşılır açıklamalar ve en az 5 adet açıklayıcı örnekle desteklendi.

En önemlisi, artık her bölümün sonunda kendinizi test etmeniz için "Sıra Sende!" adında bir alıştırma kısmı var. Haydi başlayalım ve Kotlin'in gücünü keşfedelim!

Hello World - İlk Kotlin Programımız

Her programlama dilinde olduğu gibi, Kotlin'e "Hello World" yazarak başlıyoruz. Bu basit program, bir Kotlin uygulamasının ana giriş noktası olan `main` fonksiyonunu ve konsola çıktı veren `println` fonksiyonunu tanıtır.

// Örnek 1: Temel Hello World programı
fun main() {
    println("Hello World!")
}
// Örnek 2: Türkçe mesaj ile
fun main() {
    println("Merhaba Dünya!")
}
// Örnek 3: Farklı bir fonksiyondan çağırma
fun main() {
    selamla()
}

fun selamla() {
    println("Program başladı.")
    println("Hoş geldiniz!")
}
// Örnek 4: Argüman alan main fonksiyonu
fun main(args: Array<String>) {
    println("Program başladı. Argüman sayısı: ${args.size}")
    if (args.isNotEmpty()) {
        println("İlk argüman: ${args[0]}")
    }
}
// Örnek 5: Farklı satırlarda çıktı
fun main() {
    print("Merhaba ")
    print("Dünya")
    println("!") // imleci bir alt satıra atar
}

Sıra Sende! 🧠

  • Konsola "Sınav başladı!" yazdıran bir program yazın.
  • `main` fonksiyonu içinden çağrılan `sinavMesaji` adlı bir fonksiyon oluşturun ve bu fonksiyon "Başarılar dileriz!" mesajını yazdırsın.
  • `main` fonksiyonuna argüman olarak "Ali" ismini vererek, konsola "Merhaba Ali!" yazdıran bir program yazın.
  • Konsola sırasıyla "Kotlin", "Öğreniyorum" ve "!" kelimelerini farklı `print` fonksiyonlarıyla, ama aynı satırda yazdırın.
  • Yukarıdaki örnekleri kullanarak, "Adım [Ad], Soyadım [Soyad]" şeklinde bir çıktı veren program yazın.

    val ad = "Ahmet"

}

Kotlin Değişkenleri ve Veri Tipleri

Kotlin'de iki anahtar kelime ile değişken tanımlarız: `val` (değiştirilemez) ve `var` (değiştirilebilir).

// Örnek 1: Değiştirilemeyen (immutable) değişken - val
val isim = "Ahmet"
// isim = "Mehmet" // Hata verir: val değişkeni yeniden atanamaz
println(isim)
// Örnek 2: Değiştirilebilen (mutable) değişken - var
var yas = 30
println("Yaşım: $yas")
yas = 31 // Değerini değiştirebiliriz
println("Yeni yaşım: $yas")
// Örnek 3: Tip belirtimi
val pi: Double = 3.14159
var mesaj: String = "Kotlin Öğreniyorum"
println("$mesaj, Pi sayısı: $pi")
// Örnek 4: Boş bir değişken tanımlama (null değil)
var sayi: Int = 10
sayi = 20
println(sayi)
// Örnek 5: Farklı tiplerde değişkenler
val isActive: Boolean = true
val harf: Char = 'K'
println("Aktif mi: $isActive, Baş harf: $harf")

Daha Fazla Kotlin Veri Tipi

Kotlin, temel veri tiplerini nesne (object) olarak ele alır. Bu, her bir tipin kendine ait metotları ve özellikleri olduğu anlamına gelir.

// Örnek 1: Long veri tipi
val buyukSayi: Long = 10000000000L // L harfi, Long olduğunu belirtir
println("Büyük Sayı: $buyukSayi")
// Örnek 2: Float veri tipi
val sicaklik: Float = 25.5f // f harfi, Float olduğunu belirtir
println("Sıcaklık: $sicaklik")
// Örnek 3: Byte ve Short veri tipleri
val kucukSayi1: Byte = 127 // Byte, -128'den 127'ye kadar değer alır
val kucukSayi2: Short = 32767 // Short, -32,768'den 32,767'ye kadar değer alır
println("Byte değeri: $kucukSayi1, Short değeri: $kucukSayi2")
// Örnek 4: Char ve String
val ilkHarf: Char = 'A'
val metin: String = "Kotlin'e Hoş Geldiniz!"
println("İlk harf: $ilkHarf, Metin: $metin")
// Örnek 5: Boolean tipi
val isRaining: Boolean = false
val isSunny: Boolean = true
println("Yağmur yağıyor mu? $isRaining")
println("Hava güneşli mi? $isSunny")

Sıra Sende! ✍️

  • Değeri 1200 olan bir `val` değişkeni oluşturun.
  • Adı `stok` olan ve değeri 50 olan bir `var` değişkeni oluşturun, sonra bu değeri 45 olarak değiştirip her iki değeri de yazdırın.
  • Bir Boolean (Boolean) değişkeni tanımlayın ve ismini `isLoggedIn` koyun, değerini `false` olarak atayın.
  • Bir karakter (Char) değişkeni oluşturun ve değerini 'E' olarak atayın.
  • Double tipinde, `val` anahtar kelimesini kullanarak `katsayi` adında bir değişken tanımlayın ve değerini 2.71828 olarak belirleyin.

Çözümler ✅

// 1. val değişkeni
val deger = 1200
println(deger)

// 2. var değişkeni
var stok = 50
println("Stok: $stok")
stok = 45
println("Yeni stok: $stok")

// 3. Boolean değişkeni
val isLoggedIn: Boolean = false
println("Giriş yapıldı mı: $isLoggedIn")

// 4. Char değişkeni
val harf: Char = 'E'
println("Karakter: $harf")

// 5. Double tipinde val değişkeni
val katsayi: Double = 2.71828
println("Katsayı: $katsayi")

String Interpolation

Değişkenleri veya ifadeleri bir metin dizesi içine kolayca gömmemizi sağlar. `$degisken` veya `${ifade}` şeklinde kullanılır.

// Örnek 1: Temel değişken gömme
val isim = "Ali"
val yas = 25
println("Merhaba, ben $isim ve $yas yaşındayım.")
// Örnek 2: Matematiksel işlem gömme
val fiyat = 150
val kdv = 0.18
val toplamFiyat = fiyat + (fiyat * kdv)
println("Ürünün toplam fiyatı: ${fiyat * (1 + kdv)}")
println("Yuvarlanmış toplam fiyat: ${"%.2f".format(toplamFiyat)}")
// Örnek 3: Metot çağrısı gömme
val mesaj = "Kotlin harika!"
println("Mesajın uzunluğu: ${mesaj.length}")
// Örnek 4: Koşullu ifade gömme
val saat = 14
val selam = if (saat < 12) "Günaydın" else "İyi günler"
println("Şu an ${saat} ve ${selam.lowercase()}!")
// Örnek 5: Çoklu değişken kullanımı
val ad = "Zeynep"
val soyad = "Yılmaz"
val tamAd = "$ad $soyad"
println("Tam Adım: $tamAd")

Sıra Sende! 💡

  • `urunAdi` ve `urunFiyati` değişkenlerini kullanarak, "Ayakkabı fiyatı 250.0 TL'dir." çıktısını elde edin.
  • `genislik` ve `uzunluk` değişkenleriyle bir dikdörtgenin alanını hesaplayıp, "Dikdörtgenin alanı [alan] birim karedir." şeklinde yazdırın.
  • `metin` adında bir string değişkeni oluşturun ve ters çevirip, "Ters metin: [ters_metin]" şeklinde yazdırın.
  • `siparisSayisi` adlı bir değişken 10'dan büyükse "Hızlı Teslimat" değilse "Normal Teslimat" yazdıran bir ifadeyi bir string içine gömün.
  • `kitapAdi` ve `yazarAdi` değişkenlerini tek bir stringde birleştirin ve "Kitap: [KitapAdı], Yazar: [YazarAdı]" çıktısını verin.

Çözümler ✅

// 1. Ürün fiyatı çıktısı
val urunAdi = "Ayakkabı"
val urunFiyati = 250.0
println("$urunAdi fiyatı ${urunFiyati} TL'dir.")

// 2. Dikdörtgenin alanı
val genislik = 5
val uzunluk = 8
println("Dikdörtgenin alanı ${genislik * uzunluk} birim karedir.")

// 3. Ters metin
val metin = "Merhaba"
println("Ters metin: ${metin.reversed()}")

// 4. Koşullu ifade
val siparisSayisi = 12
val durum = if (siparisSayisi > 10) "Hızlı Teslimat" else "Normal Teslimat"
println("Sipariş durumu: $durum")

// 5. Kitap bilgileri
val kitapAdi = "Uçurtma Avcısı"
val yazarAdi = "Khaled Hosseini"
println("Kitap: $kitapAdi, Yazar: $yazarAdi")

Kotlin Operatörleri

Operatörler, değerler üzerinde işlem yapmak için kullanılan sembollerdir.

// Örnek 1: Aritmetik operatörler
val a = 10
val b = 5
println("Toplama: ${a + b}")
println("Çıkarma: ${a - b}")
println("Çarpma: ${a * b}")
println("Bölme: ${a / b.toDouble()}")
println("Mod: ${a % b}")
// Örnek 2: Atama operatörleri
var x = 10
x += 5 // x = x + 5
println("x yeni değeri: $x")
x *= 2 // x = x * 2
println("x'in son değeri: $x")
// Örnek 3: Karşılaştırma operatörleri
val c = 10
val d = 20
println("c == d? ${c == d}")
println("c != d? ${c != d}")
println("d > c? ${d > c}")
// Örnek 4: Mantıksal operatörler
val isTrue = true
val isFalse = false
println("isTrue && isFalse? ${isTrue && isFalse}")
println("isTrue || isFalse? ${isTrue || isFalse}")
println("!isTrue? ${!isTrue}")
// Örnek 5: Birden fazla operatör kullanımı
val e = 5
val f = 10
val g = 15
val sonuc = (e < f) && (g > f)
println("Sonuç: $sonuc")

Sıra Sende! ⚙️

  • `sayi1` ve `sayi2` değişkenlerini tanımlayın (değerleri siz belirleyin) ve `sayi1`'in `sayi2`'den küçük veya eşit olup olmadığını kontrol edin.
  • `krediSkoru` değişkeni 700'den büyükse VE `gelir` değişkeni 5000'den büyükse true döndüren bir mantıksal ifade yazın.
  • `sayi` değişkenini 5'e bölün ve kalanı bir başka değişkene atayarak yazdırın.
  • `toplam` adında bir `var` değişkeni oluşturun, başlangıç değeri 0 olsun. Bu değişkene sırasıyla 10, 5 ve 20 ekleyerek son değerini yazdırın.
  • İki değişkenin birbirine eşit OLMADIĞINI kontrol eden bir karşılaştırma ifadesi yazın.

Çözümler ✅

// 1. Karşılaştırma
val sayi1 = 15
val sayi2 = 20
println("Sayı1, Sayı2'den küçük veya eşit mi? ${sayi1 <= sayi2}")

// 2. Mantıksal ifade
val krediSkoru = 750
val gelir = 6000
val uygun = krediSkoru > 700 && gelir > 5000
println("Kredi uygun mu? $uygun")

// 3. Mod operatörü
val sayi = 17
val kalan = sayi % 5
println("5'e bölümünden kalan: $kalan")

// 4. Atama operatörleri
var toplam = 0
toplam += 10
toplam += 5
toplam += 20
println("Toplamın son değeri: $toplam")

// 5. Eşit olmama durumu
val x = 10
val y = 15
println("x, y'ye eşit değil mi? ${x != y}")

Koşul İfadeleri (If-Else)

`if`, `else if` ve `else` ifadeleri, belirli koşullara bağlı olarak kodun farklı bölümlerini çalıştırmak için kullanılır.

// Örnek 1: Basit if-else
val yas = 17
if (yas >= 18) {
    println("Ehliyet alabilirsiniz.")
} else {
    println("Ehliyet için yaşınız yetersiz.")
}
// Örnek 2: if-else if-else zinciri
val not = 85
if (not >= 90) {
    println("Notunuz A")
} else if (not >= 80) {
    println("Notunuz B")
} else if (not >= 70) {
    println("Notunuz C")
} else {
    println("Notunuz D")
}
// Örnek 3: String karşılaştırması
val kullaniciAdi = "admin"
if (kullaniciAdi == "admin") {
    println("Yönetici girişi başarılı.")
} else {
    println("Kullanıcı adı yanlış.")
}
// Örnek 4: Birden fazla koşul
val saat = 22
val isWeekend = true
if (saat > 20 && isWeekend) {
    println("Film izleme zamanı.")
} else {
    println("Ders çalışma zamanı.")
}
// Örnek 5: if'in bir blok olarak kullanımı
val puan = 65
if (puan > 50) println("Geçtiniz!") else println("Kaldınız.")

Sıra Sende! 🎯

  • `sayi` adında bir değişken tanımlayın. Eğer bu sayı 0'dan büyükse "Pozitif", 0'dan küçükse "Negatif" ve 0'a eşitse "Sıfır" yazdıran bir kod yazın.
  • `kullaniciSifre` değişkeni "12345"e eşitse "Giriş başarılı.", aksi takdirde "Hatalı şifre." yazdırın.
  • `sicaklik` değişkeni 15'in altındaysa "Hava soğuk", 25'in üzerindeyse "Hava sıcak" ve aralıktaysa "Hava ılık" yazdırın.
  • `isLoggedin` değişkeni `true` ise VE `isAdmin` değişkeni `true` ise "Yönetici paneli" yazdırın, aksi takdirde "Normal kullanıcı paneli" yazdırın.
  • Bir öğrencinin notu 50'den büyük veya eşitse "Geçti", değilse "Kaldı" yazdıran basit bir if-else ifadesi yazın.

    println("Giriş başarılı.")
} else {
    println("Hatalı şifre.")
}

    println("Hava soğuk")
} else if (sicaklik > 25) {
    println("Hava sıcak")
} else {
    println("Hava ılık")
}

    println("Yönetici paneli")

If-Else İfadesi

Kotlin'de `if-else` bir değer döndüren ifade olarak kullanılabilir. Bu, kodu daha kısa ve okunaklı hale getirir.

// Örnek 1: if-else ifadesi
val yas = 20
val durum = if (yas >= 18) "Reşit" else "Reşit değil"
println("Durum: $durum")
// Örnek 2: Farklı blok boyutları
val sicaklik = 15
val havaDurumu = if (sicaklik > 25) {
    "Sıcak"
} else if (sicaklik > 15) {
    "Ilık"
} else {
    "Soğuk"
}
println("Hava durumu: $havaDurumu")
// Örnek 3: "Find the weather condition" - Hava durumu tespiti
val sicaklik2 = 28
val yagmurYagiyor = true
val hava = if (sicaklik2 > 25 && !yagmurYagiyor) {
    "Hava sıcak ve açık"
} else if (yagmurYagiyor) {
    "Yağmurlu ve serin"
} else {
    "Bulutlu ve ılık"
}
println("Durum: $hava")
// Örnek 4: Bir fonksiyonun dönüş değerinde if-else ifadesi
fun enBuyukSayi(a: Int, b: Int): Int {
    return if (a > b) a else b
}
println("En büyük sayı: ${enBuyukSayi(10, 5)}")
// Örnek 5: Değer ataması
val isim = "Canan"
val tebrikMesaji = "Merhaba ${if (isim.length > 5) "Sayın ${isim}" else isim}"
println(tebrikMesaji)

Sıra Sende! 🧠

  • `sinavNotu` değişkeni 50'den büyükse "Geçtiniz!" değilse "Kaldınız!" değerini `sonuc` adlı değişkene atayın.
  • `siparisSayisi` adlı bir değişkeni kontrol ederek, 100'den büyükse 0.20, 50'den büyükse 0.10, aksi takdirde 0.0 değerini `indirimOrani` değişkenine atayın.
  • `saat` değişkenini kontrol ederek, "Sabah", "Öğle" veya "Akşam" değerlerinden birini döndüren bir if-else ifadesi yazın ve sonucu bir değişkene atayın.
  • Bir fonksiyon yazın. Bu fonksiyon, parametre olarak aldığı sayının tek mi çift mi olduğunu belirten bir string değer döndürsün.
  • `kullaniciYetki` değişkeni "admin" ise `isAdmin` değişkenine `true` değerini atayan, aksi halde `false` değerini atayan bir if-else ifadesi yazın.

Çözümler ✅

// 1. Sınav notu
val sinavNotu = 60
val sonuc = if (sinavNotu > 50) "Geçtiniz!" else "Kaldınız!"
println(sonuc)

// 2. İndirim oranı
val siparisSayisi = 75
val indirimOrani = if (siparisSayisi > 100) 0.20 else if (siparisSayisi > 50) 0.10 else 0.0
println("İndirim oranı: $indirimOrani")

// 3. Günün saati
val saat = 19
val gunBolumu = if (saat < 12) "Sabah" else if (saat < 18) "Öğle" else "Akşam"
println("Günün şu anki bölümü: $gunBolumu")

// 4. Tek mi çift mi fonksiyonu
fun tekMiCiftMi(sayi: Int): String {
    return if (sayi % 2 == 0) "Çift" else "Tek"
}
println("5 bir ${tekMiCiftMi(5)} sayıdır.")

// 5. Yetki kontrolü
val kullaniciYetki = "admin"
val isAdmin = if (kullaniciYetki == "admin") true else false
println("Kullanıcı admin mi? $isAdmin")

Range (Aralıklar)

Kotlin'de `..` operatörü ile kapalı aralıklar (`[x, y]`), `until` ile ise açık aralıklar (`[x, y)`) oluşturulabilir. Bu, döngülerde veya koşul ifadelerinde çok kullanışlıdır.

// Örnek 1: Sayı aralığı ile for döngüsü
for (i in 1..5) { // 1'den 5'e (5 dahil)
    println(i)
}
// Örnek 2: Harf aralığı ile for döngüsü
for (char in 'a'..'c') {
    println(char)
}
// Örnek 3: Tersten sayma
for (i in 5 downTo 1) { // 5'ten 1'e (1 dahil)
    println(i)
}
// Örnek 4: Belirli adımlarla
for (i in 1..10 step 2) { // 1, 3, 5, 7, 9
    println(i)
}
// Örnek 5: until kullanımı (son değer dahil değil)
for (i in 1 until 5) { // 1, 2, 3, 4
    println(i)
}

When İfadesi

`when` ifadesi, bir değerin olası tüm durumlarını kontrol etmek için kullanılır ve çoklu `if-else-if` zincirine göre daha okunaklı ve güçlü bir alternatiftir.

// Örnek 1: Basit when ifadesi
val not = 3
when (not) {
    1 -> println("Çok Kötü")
    2 -> println("Kötü")
    3 -> println("Orta")
    4 -> println("İyi")
    5 -> println("Çok İyi")
    else -> println("Geçersiz not")
}
// Örnek 2: Virgül ile birden fazla koşul
val char = 'a'
when (char) {
    'a', 'e', 'i', 'o', 'u' -> println("Sesli harf")
    else -> println("Sessiz harf")
}
// Örnek 3: Bir ifade olarak when
val sayi = 10
val sonuc = when {
    sayi % 2 == 0 -> "Çift Sayı"
    else -> "Tek Sayı"
}
println(sonuc)
// Örnek 4: Aralık (`range`) kontrolü
val yas = 15
when (yas) {
    in 0..10 -> println("Çocuk")
    in 11..18 -> println("Genç")
    else -> println("Yetişkin")
}
// Örnek 5: Farklı tipte koşullar
val obj: Any = "Merhaba" // Any, her türlü tipi temsil eder
when (obj) {
    1 -> println("Bir rakam")
    "Merhaba" -> println("Bir string")
    is Long -> println("Long tipi bir sayı")
    !is String -> println("String değil")
    else -> println("Bilinmeyen tip")
}

Döngüler (For, While, Do-While)

Döngüler, bir kod bloğunu belirli sayıda veya bir koşul doğru olduğu sürece tekrarlamak için kullanılır.

// Örnek 1: For döngüsü (listede gezinme)
val meyveler = listOf("elma", "armut", "kiraz")
for (meyve in meyveler) {
    println(meyve)
}
// Örnek 2: While döngüsü
var sayac = 0
while (sayac < 5) {
    println("Sayac: $sayac")
    sayac++
}
// Örnek 3: Do-while döngüsü
var i = 0
do {
    println("Do-while ile i: $i")
    i++
} while (i < 3)
// Örnek 4: İndeks ile for döngüsü
val liste = listOf("a", "b", "c")
for (index in liste.indices) {
    println("İndeks $index, Değer ${liste[index]}")
}
// Örnek 5: Break ve Continue
for (num in 1..10) {
    if (num == 5) {
        continue // 5'i atla
    }
    if (num == 8) {
        break // 8'de döngüyü sonlandır
    }
    println(num)
}

Null Güvenliği

Kotlin, `NullPointerException` hatalarını önlemek için tasarlanmıştır. Değişkenler varsayılan olarak null olamaz. Null olabilmesi için tipin sonuna `?` eklenmelidir.

// Örnek 1: Null-geçebilir değişken
var isim: String? = "Ahmet"
isim = null
println(isim)
// Örnek 2: Güvenli çağrı operatörü
var metin: String? = "Kotlin"
val uzunluk = metin?.length
println("Uzunluk: $uzunluk") // 6
metin = null
val yeniUzunluk = metin?.length
println("Yeni uzunluk: $yeniUzunluk") // null
// Örnek 3: Elvis operatörü
var soyad: String? = null
val adSoyad = soyad ?: "Soyadı Yok"
println("Soyad: $adSoyad")
// Örnek 4: !! operatörü
var sehir: String? = "Ankara"
val sehirUzunluk = sehir!!.length // sehir'in null olmadığına eminiz
println("Şehir uzunluğu: $sehirUzunluk")
sehir = null
// val hata = sehir!!.length // Hata fırlatır!
// Örnek 5: Let fonksiyonu ile null kontrolü
var yas: Int? = 25
yas?.let {
    println("Yaş değeri $it")
}
yas = null
yas?.let {
    println("Bu satır çalışmaz")
}

Sıra Sende! 🛡️

  • `telefonNumarasi` adında, `String?` tipinde bir değişken oluşturun. Bir `if` ifadesi kullanarak null olup olmadığını kontrol edin.
  • `kullaniciMesaji` adında `String?` tipinde bir değişken tanımlayın, null değeri verin ve uzunluğunu güvenli çağrı (`?.`) kullanarak yazdırın.
  • `kullaniciAdi` adında `String?` tipinde bir değişken oluşturun. `null` ise "Misafir" değerini atayan bir Elvis operatörü (`?:`) ifadesi yazın.
  • `oyuncuPuani` adında `Int?` bir değişken tanımlayın. Eğer `null` değilse, değerini 10 artırın.
  • `islemSonucu` adlı `Double?` bir değişken tanımlayın. `!!` operatörü ile uzunluğunu yazdırın ve bu işlemin neden riskli olduğunu açıklayan bir yorum ekleyin.

Çözümler ✅

    println("Telefon numarası atanmamış.")
}

// 2. Güvenli çağrı

println("Kullanıcı adı: $ad")

// 4. Değer artırma

oyuncuPuani = oyuncuPuani?.plus(10)
println("Yeni puan: $oyuncuPuani")

// 5. !! operatörü

try-catch ile İstisna Yönetimi

`try-catch` blokları, programınızın çökmesine neden olabilecek hataları (istisnaları) yakalamak ve yönetmek için kullanılır.

// Örnek 1: Sayı formatı hatası
try {
    val sayi = "abc".toInt()
} catch (e: NumberFormatException) {
    println("Hata: Geçersiz sayı formatı.")
} finally {
    println("İşlem tamamlandı.")
}
// Örnek 2: Bölme işlemi hatası (0'a bölme)
fun bolme(a: Int, b: Int): Double {
    return try {
        (a / b).toDouble()
    } catch (e: ArithmeticException) {
        println("Hata: Sıfıra bölme hatası.")
        0.0
    }
}
println("Sonuç: ${bolme(10, 2)}")
println("Sonuç: ${bolme(10, 0)}")
// Örnek 3: Birden fazla catch bloğu
try {
    val arr = arrayOf(1, 2)
    println(arr[5])
} catch (e: ArrayIndexOutOfBoundsException) {
    println("Hata: Dizi sınırları dışında bir indekse erişim.")
} catch (e: Exception) {
    println("Genel bir hata oluştu.")
}
// Örnek 4: Dosya okuma hatası simülasyonu
fun dosyaOku() {
    try {
        throw Exception("Dosya bulunamadı hatası!")
    } catch (e: Exception) {
        println("Hata yakalandı: ${e.message}")
    }
}
dosyaOku()
// Örnek 5: try-catch bir ifade olarak
val sayiStr = "123"
val sayi = try {
    sayiStr.toInt()
} catch (e: NumberFormatException) {
    0 // Hata durumunda 0 döndür
}
println("Sonuç: $sayi")

Sıra Sende! ⚠️

  • Kullanıcıdan bir metin girmesini isteyin. Eğer girilen metin bir sayıya çevrilemiyorsa, "Geçersiz giriş. Lütfen bir sayı girin." hatası verin.
  • `bolme` fonksiyonunda ikinci parametre 0 ise `ArithmeticException` fırlatın ve `try-catch` ile bu hatayı yakalayın.
  • `metinler` adlı bir string listesi oluşturun ve 5. indeksteki elemana erişmeye çalışın. `IndexOutOfBoundsException` hatasını yakalayın.
  • İç içe `try-catch` blokları kullanarak bir dosya okuma işlemi simüle edin. Dış blokta "Dosya okuma işlemi başladı.", iç blokta "Dosya okunamadı" gibi mesajlar yazdırın.

`data class`, veri tutmak için tasarlanmıştır. Derleyici otomatik olarak `equals()`, `hashCode()`, `toString()`, `copy()` ve `componentN()` gibi faydalı metotları oluşturur.

println(user2) // User(name=Ali, age=26)
// Örnek 2: copy() metodu

// Örnek 3: Destructuring (ayrıştırma)

// Örnek 4: Data class listesi

Kalıtım (Inheritance)

Kalıtım, bir sınıftan başka bir sınıf oluşturma mekanizmasıdır. Alt sınıf (child class), üst sınıftan (parent class) özellik ve metotları miras alır. Varsayılan olarak, Kotlin sınıfları `final`'dır, yani miras alınamaz. Miras alınabilmesi için `open` anahtar kelimesiyle belirtilmelidir.

// Örnek 1: Temel kalıtım
open class Vasıta(val tekerlekSayisi: Int) {
    fun hareketEt() {
        println("Vasıta hareket ediyor.")
    }
}
class Bisiklet(tekerlekSayisi: Int) : Vasıta(tekerlekSayisi)
val bisiklet = Bisiklet(2)
println("Bisikletin tekerlek sayısı: ${bisiklet.tekerlekSayisi}")
bisiklet.hareketEt()
// Örnek 2: Fonksiyon ve özellik miras alma
open class Hayvan {
    open fun sesCikar() { // override edilebilmesi için open olmalı
        println("Hayvan bir ses çıkarıyor.")
    }
}
class Kopek : Hayvan() {
    override fun sesCikar() { // metodu geçersiz kıldık
        println("Hav hav!")
    }
}
val kopek = Kopek()
kopek.sesCikar()
// Örnek 3: İki seviyeli kalıtım
open class Sekil
open class Kare : Sekil()
class Dikdortgen : Kare()
val dikdortgen = Dikdortgen()
// Örnek 4: Kurucu parametrelerini geçirme
open class Kisi(val ad: String, val yas: Int)
class Calisan(ad: String, yas: Int, val pozisyon: String) : Kisi(ad, yas)
val calisan = Calisan("Ayşe", 30, "Mühendis")
println("${calisan.ad} adlı çalışanın pozisyonu: ${calisan.pozisyon}")
// Örnek 5: Farklı seviyelerden erişim
open class A {
    fun mesajA() = println("A sınıfı")
}
open class B : A() {
    fun mesajB() = println("B sınıfı")
}
class C : B() {
    fun mesajC() {
        mesajA()
        mesajB()
        println("C sınıfı")
    }
}
val c = C()
c.mesajC()

open class Parent {

Soyut (Abstract) Sınıf ve Metot

Soyut sınıflar, nesnesi oluşturulamayan sınıflardır ve `abstract` anahtar kelimesiyle tanımlanır. Genellikle ortak davranışlar için bir temel sınıf olarak hizmet eder.

// Örnek 1: Soyut sınıf ve metot
abstract class Varlık(val ad: String) {
    abstract fun selamla()
}
class Insan(ad: String) : Varlık(ad) {
    override fun selamla() {
        println("Merhaba, ben bir insanım. Adım $ad.")
    }
}
val insan = Insan("Ahmet")
insan.selamla()

abstract class Calisan(val ad: String) {
    abstract fun isYap()
    fun molaVer() {
        println("$ad mola veriyor.")
    }
}
class Programci(ad: String) : Calisan(ad) {
    override fun isYap() {
        println("$ad kod yazıyor.")
    }
}
val programci = Programci("Mehmet")
programci.isYap()
programci.molaVer()
// Örnek 3: Soyut özellik

Arayüz (Interface)

`Interface`, bir sınıfın sahip olması gereken davranışları tanımlar. Bir sınıf birden fazla `interface`'i uygulayabilir.

// Örnek 1: Temel interface
interface HareketEdebilir {
    fun ilerle()
}
class Araba : HareketEdebilir {
    override fun ilerle() {
        println("Araba hareket ediyor.")
    }
}
val araba = Araba()
araba.ilerle()
// Örnek 2: Birden fazla interface
interface Hizlanabilir {
    fun hizlan(hiz: Int)
}
interface Durabilir {
    fun dur()
}
class Bisiklet : Hizlanabilir, Durabilir {
    override fun hizlan(hiz: Int) {
        println("Bisiklet $hiz km/s hıza ulaştı.")
    }
    override fun dur() {
        println("Bisiklet durdu.")
    }
}
val bisiklet = Bisiklet()
bisiklet.hizlan(20)
bisiklet.dur()
// Örnek 3: Interface'te varsayılan metot
interface SesCikaran {
    fun sesCikar()
    fun uyu() {
        println("Nesne uyuyor.")
    }
}
class Kedi : SesCikaran {
    override fun sesCikar() {
        println("Miyav!")
    }
}
val kedi = Kedi()
kedi.sesCikar()
kedi.uyu()
// Örnek 4: Interface'te özellikler
interface Verimli {
    val verim: Int
}
class Ampul(override val verim: Int) : Verimli
val ampul = Ampul(95)
println("Ampulün verimi: ${ampul.verim}%")

    fun yuru()

kus.yuru()

Object ve Companion Object

object: Singleton (tekil) deseni uygulamak için kullanılır. Bu sınıfın sadece bir örneği oluşturulur.

companion object: Bir sınıfın kendisine ait, nesne örneklerine bağlı olmayan (statik gibi) üyelere erişmek için kullanılır.

// Örnek 1: Singleton object
object DatabaseDriver {
    fun connect() {
        println("Veritabanına bağlandı.")
    }
}

// Örnek 2: Companion object
class Matematik {
    companion object {
        const val PI = 3.14159
        fun daireAlanHesapla(yaricap: Double) = PI * yaricap * yaricap
    }
}
// Örnek 3: Nesne olarak anonim sınıf (Object expression)
val anonimListener = object {
    fun onButtonClick() {
        println("Butona tıklandı!")
    }
}
// Örnek 4: companion object'e isim verme
class Fabrika {
    companion object Uretim {
        fun yeniUrunUret() = println("Yeni bir ürün üretiliyor.")
    }
}
// Örnek 5: object ve interface
interface BilgiGetir {
    fun bilgi()
}
object UygulamaAyarlari : BilgiGetir {
    val tema = "Açık"
    override fun bilgi() {
        println("Uygulama ayarları: Tema '$tema'")
    }
}

Sıra Sende! 🧑‍🤝‍🧑

  • `Logger` adında, `log(mesaj)` fonksiyonuna sahip tekil bir `object` oluşturun.
  • `YardimciFonksiyonlar` adında, `stringBuyukHarfeCevir(metin)` fonksiyonuna sahip bir `companion object` yazın.
  • `StatikVeri` adında bir sınıf oluşturun ve içinde `kullaniciSayisi` adında bir `companion object` özelliği tanımlayın.
  • `Button` sınıfı için, tıklama olayı dinleyicisini bir `object expression` (anonim sınıf) olarak uygulayın.
  • `Fabrika` sınıfına, birincil kurucu almayan ve `uret()` metoduna sahip bir `companion object` ekleyin.
