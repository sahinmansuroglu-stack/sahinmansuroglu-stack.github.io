Soru 6 Döngü İşlemleri ve Kayıt Silme (25 Puan)

Bölüm 1 (20 Puan): foreach döngüsü ile listedeki her bir öğrencinin bilgilerini yazdırınız.

  • Eğer öğrenci İlkokul ise ortaokuldanMezunOlmaYiliniEkranaYaz() çağır.
  • Eğer öğrenci Ortaokul ise gidebileceğiOkullariEkranaYaz() çağır.

Bölüm 2 (5 Puan): Listeden Ad:Ahmet soyadı:EKER olan öğrenciyi sildiriniz. Ardından listeyi tekrar yazdırınız.

Program.cs (Devamı) 
// ... Main metodu içerisindeki ogrenciler listesi devamı ...

            // BÖLÜM 1: Foreach ve Tip Kontrolü (is keyword)
            Console.WriteLine("=== ÖĞRENCİ LİSTESİ ===");
            
            foreach (var ogr in ogrenciler)
            {
                // Temel bilgileri yaz
                ogr.bilgileriEkranaYaz(); 

                // Tip kontrolü ve Type Casting
                if (ogr is IlkOkulOgrencisi ilkOkulOgr)
                {
                    ilkOkulOgr.ortaokuldanMezunOlmaYiliniEkranaYaz();
                }
                else if (ogr is OrtaOkulOgrencisi ortaOkulOgr)
                {
                    ortaOkulOgr.gidebileceğiOkullariEkranaYaz();
                }
            }

            // BÖLÜM 2: Silme İşlemi
            Console.WriteLine("\n=== SİLME İŞLEMİ (Ahmet EKER) ===");
            
            // Listeden belirli koşula uyan elemanı bulma
            // Not: System.Linq kütüphanesi gereklidir.
            var silinecekOgrenci = null as Ogrenci;

            foreach (var item in ogrenciler)
            {
                if(item.Ad == "Ahmet" && item.Soyad == "EKER")
                {
                    silinecekOgrenci = item;
                    break;
                }
            }

            if (silinecekOgrenci != null)
            {
                ogrenciler.Remove(silinecekOgrenci);
                Console.WriteLine("Ahmet EKER listeden silindi.");
            }

            // Listeyi Tekrar Yazdırma
            Console.WriteLine("\n=== GÜNCEL LİSTE ===");
            foreach (var ogr in ogrenciler)
            {
                Console.WriteLine(ogr.ToString());
            }

            Console.ReadLine();
