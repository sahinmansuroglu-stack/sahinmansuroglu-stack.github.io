Soru 2 Interface Tanımlama (5 Puan)

Aşağıda verilenlere göre IOgrenciIslem adında bir interface tanımlayınız.

  • string tipinde matematikDersIcerigi() adında bir metot gövdesi bulunacak.
  • int tipinde mezunOlacagiYasiHesapla() adında bir metot gövdesi tanımlayınız.
IOgrenciIslem.cs 
namespace SinavUygulamasi
{
    // Soru 2: Interface Tanımlaması
    public interface IOgrenciIslem
    {
        // Interface içerisinde metotların sadece imzaları bulunur, gövdeleri olmaz.
        
        string matematikDersIcerigi();
        
        int mezunOlacagiYasiHesapla();
    }
}
