⏳ Neden "Task" Kullanıyoruz?

Modern C# dünyasında veritabanı işlemleri için Task (Asenkron yapı) kullanmak bir tercih değil, zorunluluktur. İşte sebebi:

Arayüz Donmasını Engeller

Veritabanına bir sorgu gönderdiğinizde cevabın gelmesi zaman alabilir. Eğer Task kullanmazsanız, bu süre boyunca formunuz kilitlenir (not responding). Task sayesinde formunuz canlı kalmaya devam eder.

Arka Planda Çalışma

Programınız veritabanından cevap gelmesini beklerken, bilgisayarınızın işlemcisini yormaz. İş bittiğinde await komutu ile kaldığı yerden devam eder.

🎯 Özetle

Task = Yapılacak işin kendisi.
async = Bu metodun içinde beklememiz gereken işler olduğunu söyler.
await = "İş bitene kadar burada dur, ama programın geri kalanını kilitleme" demektir.

🏗️ Neden Modüler Yapı?

Kodlarımızı parçalara (Model, Servis, UI) ayırmak, projemizin gelecekte büyümesini kolaylaştırır. Bir hata olduğunda nerede arayacağımızı biliriz ve aynı kodları tekrar tekrar yazmak zorunda kalmayız.

Daha Temiz Kod

Her sınıfın sadece bir görevi vardır (Single Responsibility).

Kolay Bakım

Veritabanı şifresi değiştiğinde sadece tek bir dosyayı güncellemeniz yeterli olur.

2
Veritabanı Servisi

Tüm SQL sorgularının ve Dapper komutlarının bulunduğu "beyin" kısmıdır. UI (Arayüz) bu servisi çağırarak iş yapar.

VeritabaniServis.cs 
using Dapper;
using MySql.Data.MySqlClient;
using System.Collections.Generic;
using System.Data;
using System.Linq;
using System.Threading.Tasks;

public class VeritabaniServis
{
    private readonly string _connStr = "host=localhost;port=3306;user id=root;password=mtal123;database=okuldb;";

    private IDbConnection BaglantiAl() => new MySqlConnection(_connStr);

    // Verileri Listele
    public async Task<List<Ogrenci>> ListeleAsync()
    {
        using var db = BaglantiAl();
        return (await db.QueryAsync<Ogrenci>("SELECT * FROM ogrenciler")).ToList();
    }

    // Yeni Kayıt Ekle
    public async Task<int> EkleAsync(Ogrenci ogrenci)
    {
        using var db = BaglantiAl();
        string sql = "INSERT INTO ogrenciler (Ad, Soyad, Sinif, Puan) VALUES (@Ad, @Soyad, @Sinif, @Puan)";
        return await db.ExecuteAsync(sql, ogrenci);
    }

    // Kaydı Güncelle
    public async Task<int> GuncelleAsync(Ogrenci ogrenci)
    {
        using var db = BaglantiAl();
        string sql = "UPDATE ogrenciler SET Ad=@Ad, Soyad=@Soyad, Sinif=@Sinif, Puan=@Puan WHERE OgrenciID=@OgrenciID";
        return await db.ExecuteAsync(sql, ogrenci);
    }

    // Kaydı Sil
    public async Task<int> SilAsync(int id)
    {
        using var db = BaglantiAl();
        return await db.ExecuteAsync("DELETE FROM ogrenciler WHERE OgrenciID = @id", new { id });
    }
}

3
Arayüz Entegrasyonu

Button click olaylarında servisi nasıl kullanacağınızı ve DataGrid seçimini nasıl yapacağınızı gösterir.

Form1.cs (Kısmi)

💡 Kritik İpucu

Güncelleme ve Silme işlemleri için her zaman bir _seciliId değişkeni tutun. Bu ID'yi DataGrid'e her tıklandığında güncelleyin.

// Servis örneği (Global tanımlayın)
VeritabaniServis _servis = new VeritabaniServis();
int _seciliId = 0;

// LİSTELEME BUTONU
private async void btnListele_Click(object sender, EventArgs e) 
{
    dgvOgrenciler.DataSource = await _servis.ListeleAsync();
}

// EKLEME BUTONU
private async void btnEkle_Click(object sender, EventArgs e) 
{
    var ogrenci = new Ogrenci {
        Ad = txtAd.Text, Soyad = txtSoyad.Text, 
        Sinif = txtSinif.Text, Puan = int.Parse(txtPuan.Text)
    };
    int sonuc = await _servis.EkleAsync(ogrenci);
    if(sonuc > 0) MessageBox.Show("Başarıyla eklendi!");
    await Listele(); // Listeyi yenile
}

// GÜNCELLEME BUTONU
private async void btnGuncelle_Click(object sender, EventArgs e) 
{
    if(_seciliId == 0) return;
    var ogrenci = new Ogrenci {
        OgrenciID = _seciliId,
        Ad = txtAd.Text, Soyad = txtSoyad.Text, 
        Sinif = txtSinif.Text, Puan = int.Parse(txtPuan.Text)
    };
    int sonuc = await _servis.GuncelleAsync(ogrenci);
    if(sonuc > 0) MessageBox.Show("Güncelleme başarılı!");
    await Listele();
}

// DATAGRID TIKLAMA (Veriyi UI'a Taşıma)
private void dgvOgrenciler_CellClick(object sender, DataGridViewCellEventArgs e) 
{
    if (e.RowIndex >= 0) {
        var satir = dgvOgrenciler.Rows[e.RowIndex];
        _seciliId = (int)satir.Cells["OgrenciID"].Value;
        
        // Kutuları doldur
        txtAd.Text = satir.Cells["Ad"].Value.ToString();
        txtSoyad.Text = satir.Cells["Soyad"].Value.ToString();
        txtSinif.Text = satir.Cells["Sinif"].Value.ToString();
        txtPuan.Text = satir.Cells["Puan"].Value.ToString();
    }
}
