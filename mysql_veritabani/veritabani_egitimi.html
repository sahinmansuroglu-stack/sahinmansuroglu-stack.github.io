Veritabanı Dünyasına Giriş

C# Nesne Tabanlı Programlama Macerasında Yeni Bir Durak!

📦 1. Temel Veritabanı Kavramları

Merhaba gençler! C# ile harika programlar yazıyoruz, nesneler (object) oluşturuyoruz, sınıflar (class) tasarlıyoruz. Peki, programımızı kapattığımızda bu verilere ne oluyor? Maalesef uçup gidiyor! İşte verilerimizin kalıcı olmasını sağlayan sihirli depolara Veritabanı (Database) diyoruz.

Bir veritabanını, okuldaki kocaman bir dosya dolabı veya devasa bir dijital kütüphane gibi düşünebilirsiniz. İçinde çok düzenli bir sistem vardır.

🗄️ Veritabanı (Database)

Tüm verilerimizin düzenli bir şekilde tutulduğu ana depodur. Tıpkı okuldaki ana arşiv odası gibidir. (Örn: OkulDB)

📊 Tablo (Table)

Veritabanı içindeki klasörlerdir. Benzer türdeki verileri gruplar. Örneğin; Ogrenciler tablosu, Ogretmenler tablosu.

📝 Satır / Kayıt (Row/Record)

Tablodaki her bir veridir. Örneğin; "101 numaralı Ahmet Yılmaz'ın tüm bilgileri" bir satırdır.

🏷️ Sütun / Alan (Column)

Kayıtların özellikleridir. C#'taki class property'leri gibidir. Örneğin; Ad, Soyad, Yas birer sütundur.

🔑 Birincil Anahtar (Primary Key - PK)

Bir kaydı diğerlerinden ayıran benzersiz değerdir. Sizin TC Kimlik Numaranız veya Okul Numaranız gibidir. İki öğrencinin adı aynı olabilir ama okul numarası aynı olamaz!

🐬 2. MySQL Sunucusu Nedir?

Veritabanımız kendi başına duramaz; onu yönetecek, verileri hızlıca bulup getirecek, güvenliği sağlayacak bir "yöneticiye" ihtiyaç vardır. İşte MySQL Server, dünyadaki en popüler, ücretsiz ve çok güçlü veritabanı yönetim sistemlerinden (RDBMS) biridir.

Biz C# programımızdan MySQL'e bağlanıp, ona "Bana 10-A sınıfındaki öğrencileri getir!" veya "Yeni bir oyunu sisteme kaydet!" diyeceğiz. MySQL de bizim için arka planda bu karmaşık işlemleri ışık hızında yapacak.

🏗️ 3. Veritabanı ve Tablo Oluşturma

MySQL ile konuşmak için kullandığımız özel dile SQL (Structured Query Language) denir. C#'tan farklıdır ama yazması çok eğlencelidir!

Veritabanı Oluşturma

Önce verilerimizi koyacağımız ana depomuzu oluşturalım:

-- Yeni bir veritabanı yarat
CREATE DATABASE OkulDB;

-- İşlem yapacağımız veritabanını seç (Bu çok önemli!)
USE OkulDB;

Tablo Oluşturma

Şimdi bu veritabanı içine öğrencileri kaydedeceğimiz Ogrenciler adında bir tablo yapalım. Sütunların veri tiplerini (sayı mı, metin mi?) belirtmeliyiz.

CREATE TABLE Ogrenciler (
    OgrenciID INT AUTO_INCREMENT PRIMARY KEY, -- Benzersiz numara, kendi kendine artar
    Ad VARCHAR(50),                           -- Maks. 50 karakterlik metin
    Soyad VARCHAR(50),                        -- Maks. 50 karakterlik metin
    Sinif VARCHAR(10),                        -- Sınıf bilgisi (Örn: 10-A)
    Puan INT                                  -- Tam sayı
);

C# ile Karşılaştırma:
SQL'deki VARCHAR(50) = C#'taki string
SQL'deki INT = C#'taki int
AUTO_INCREMENT ise çok havalıdır! MySQL, her yeni kayıt eklendiğinde ID'yi 1, 2, 3 diye otomatik belirler, bizim uğraşmamıza gerek kalmaz.

⚡ 4. Temel SQL Komutları (CRUD)

Veritabanı dünyasında 4 temel eylem vardır: Create (Oluştur/Ekle), Read (Oku/Listele), Update (Güncelle), Delete (Sil). İngilizce baş harflerinden buna kısaca CRUD denir.

A. Veri Ekleme (INSERT INTO)

Tabloya yeni öğrenciler kaydedelim. OgrenciID vermiyoruz çünkü MySQL onu kendisi verecek.

-- Tek bir öğrenci ekleme
INSERT INTO Ogrenciler (Ad, Soyad, Sinif, Puan) 
VALUES ('Ali', 'Yılmaz', '10-A', 85);

-- Aynı anda birden fazla öğrenci ekleme (Çok daha hızlı!)
INSERT INTO Ogrenciler (Ad, Soyad, Sinif, Puan) VALUES 
('Ayşe', 'Demir', '10-B', 95),
('Mehmet', 'Kaya', '11-C', 70),
('Zeynep', 'Şahin', '10-A', 100);

B. Veri Okuma / Listeleme (SELECT)

En çok kullanacağınız komut! Verileri çekmek ve filtrelemek için kullanırız.

-- 1. Tablodaki TÜM öğrencileri ve tüm bilgileri getirir (* her şey demektir)
SELECT * FROM Ogrenciler;

-- 2. Sadece Ad ve Soyad sütunlarını getirir
SELECT Ad, Soyad FROM Ogrenciler;

-- 3. FİLTRELEME (WHERE): Sadece 10-A sınıfındaki öğrencileri getirir
SELECT * FROM Ogrenciler WHERE Sinif = '10-A';

-- 4. BİRDEN FAZLA ŞART: 10-A sınıfında olan VE puanı 90'dan büyük olanlar
SELECT * FROM Ogrenciler WHERE Sinif = '10-A' AND Puan > 90;

-- 5. SIRALAMA (ORDER BY): Öğrencileri puana göre yüksekten düşüğe (DESC) sıralar
SELECT * FROM Ogrenciler ORDER BY Puan DESC;

C. Veri Güncelleme (UPDATE)

Diyelim ki Mehmet'in sınav sonucuna itiraz edildi ve puanı 80 oldu. Bunu güncellemeliyiz.

UPDATE Ogrenciler 
SET Puan = 80 
WHERE Ad = 'Mehmet' AND Soyad = 'Kaya';

D. Veri Silme (DELETE)

Okuldan mezun olan veya ayrılan bir kaydı silmek için kullanılır.

-- OgrenciID'si 2 olan öğrenciyi siler (En güvenli silme yöntemi ID kullanmaktır)
DELETE FROM Ogrenciler WHERE OgrenciID = 2;

⚠️ DİKKAT: DELETE veya UPDATE komutlarında WHERE şartını yazmayı unutursanız, tablodaki TÜM öğrencileri siler veya güncellersiniz! Sistemleri çökerten en büyük acemi hatası budur. Daima filtre kullanın!

🎮 Sıra Sende! (Görev Zamanı)

Şimdi klavye sende! C# ile yapacağımız bir "Oyun Kütüphanesi" programı için veritabanı altyapısını kurman gerekiyor.

Görev Listesi:

  1. OyunDunyasi adında bir veritabanı oluştur ve seç.
  2. İçine Oyunlar adında bir tablo oluştur. Tabloda şu özellikler (sütunlar) olsun:
    • ID (Tam sayı, Birincil anahtar, Otomatik artan)
    • OyunAdi (Metin - Maks 100 karakter)
    • Tur (Metin - Aksiyon, RPG, Spor vb. Maks 50 karakter)
    • MetacriticPuani (Tam sayı - 100 üzerinden)
  3. Tabloya en sevdiğin 3 farklı oyunu tek bir sorgu ile ekle (INSERT).
  4. Puanı 80'den büyük olan oyunları listele (SELECT).
  5. Eklediğin oyunlardan birinin puanını 99 olarak güncelle (UPDATE).
  6. Son olarak, türü 'Spor' olan oyunları veritabanından sil (DELETE). (Eğer spor oyunu eklemediysen, eklediğin bir türü sil)

✅ Çözümler

Umarım önce kendin denemişsindir! Gelişmek için hata yapmak en güzel yoldur. İşte doğru SQL komutları:

1. Veritabanı Oluşturma ve Seçme:

CREATE DATABASE OyunDunyasi;
USE OyunDunyasi;

2. Tablo Oluşturma:

CREATE TABLE Oyunlar (
    ID INT AUTO_INCREMENT PRIMARY KEY,
    OyunAdi VARCHAR(100),
    Tur VARCHAR(50),
    MetacriticPuani INT
);

3. Oyun Ekleme:

INSERT INTO Oyunlar (OyunAdi, Tur, MetacriticPuani) VALUES 
('Red Dead Redemption 2', 'Aksiyon-Macera', 97),
('Elden Ring', 'RPG', 96),
('EA Sports FC 24', 'Spor', 76);

4. Puanı 80'den Büyük Olanları Listeleme:

SELECT * FROM Oyunlar WHERE MetacriticPuani > 80;

5. Puan Güncelleme:

-- RDR2'nin puanını 99 yapalım
UPDATE Oyunlar 
SET MetacriticPuani = 99 
WHERE OyunAdi = 'Red Dead Redemption 2';

6. Kayıt Silme:

-- Türü Spor olanları silelim
DELETE FROM Oyunlar WHERE Tur = 'Spor';
