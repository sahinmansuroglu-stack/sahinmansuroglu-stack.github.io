Web Tasarım Projesi - Stillendirme Adımları

Aşağıdaki adımları takiperek style.css dosyasını oluşturunuz ve projenizi tamamlayınız.

1. Adım: Temel Ayarlar & Reset

Tüm sayfa için varsayılan boşlukları sıfırlayın ve yazı tipini ayarlayın.

  • body etiketine margin: 0 ve padding: 0 verin.
  • Yazı tipi (font-family) olarak Arial, sans-serif kullanın.
  • Arka plan rengini (background-color) #f4f4f4 yapın.
body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
    background-color: #f4f4f4;
}

2. Adım: Genel Kapsayıcı (Container)

Sayfa içeriğini ortalamak için .container sınıfını düzenleyin.

  • Genişliği (width) 1080px olarak ayarlayın.
  • Sayfada ortalamak için margin: 0 auto kullanın.
  • Arka plan rengini white (beyaz) yapın.
.container {
    width: 1080px;
    margin: 0 auto;
    background-color: white;
}

3. Adım: Header ve Logo Düzeni (Flexbox)

Logo ve menüyü yan yana getirmek için .main-header sınıfını yapılandırın.

  • display: flex kullanarak Flexbox yapısına geçin.
  • justify-content: space-between ile logoyu sola, menüyü sağa yaslayın.
  • align-items: center ile elemanları dikeyde ortalayın.
  • İç boşluk (padding) olarak 10px 20px verin.
  • Altına ince bir çizgi ekleyin: border-bottom: 2px solid #ddd.
.main-header {
    background-color: #f4f4f4;
    padding: 10px 20px;
    display: flex;
    justify-content: space-between;
    align-items: center;
    border-bottom: 2px solid #ddd;
}

4. Adım: Navigasyon Linkleri

Menü linklerini (nav a) daha şık hale getirin.

  • Yazı rengini koyu gri (#333) ve kalınlığını (bold) ayarlayın.
  • Alt çizgiyi (text-decoration) kaldırın.
  • Linkler arasına mesafe koymak için margin: 0 10px verin.
  • Hover Durumu: Üzerine gelince rengi mavi (#007bff) olsun.
  • Active Durumu: .active sınıfına sahip linkin arka planı #333, yazı rengi white olsun.
nav a {
    margin: 0 10px;
    text-decoration: none;
    color: #333;
    font-weight: bold;
}

nav a:hover {
    color: #007bff;
}

nav a.active {
    background-color: #333;
    color: white;
    padding: 5px 10px;
    border-radius: 5px;
}

5. Adım: Ana İçerik Düzeni (CSS Grid)

İçerik ve kenar çubuğunu yan yana getirmek için main etiketini Grid yapısına çevirin.

  • display: grid kullanın.
  • Sütunları Ayarlayın: Sol taraf geniş (3 birim), sağ taraf dar (1 birim) olacak şekilde grid-template-columns: 3fr 1fr tanımlayın.
  • Sütunlar arasına 20px boşluk (gap) bırakın.
  • Tüm içeriğe 20px iç boşluk (padding) verin.
main {
    padding: 20px;
    display: grid;
    grid-template-columns: 3fr 1fr;
    gap: 20px;
}

6. Adım: İçerik ve Sidebar Görünümü

Sol içerik (.content) ve sağ menü (.sidebar) kutularını düzenleyin.

  • .content arka planını beyaz yapın.
  • .sidebar arka planını açık gri (#f9f9f9) ve iç boşluğunu 10px yapın.
.content {
    background-color: white;
}

.sidebar {
    background-color: #f9f9f9;
    padding: 10px;
}

7. Adım: Footer (Alt Bilgi)

Sayfanın en altındaki footer alanını düzenleyin.

  • Arka plan rengini koyu gri (#333), yazı rengini beyaz yapın.
  • Yazıyı ortalayın (text-align: center).
  • İç boşluk (padding) 10px verin.
footer {
    background-color: #333;
    color: white;
    text-align: center;
    padding: 10px;
    margin-top: 10px;
}

8. Adım: Mobil Uyumluluk (Responsive)

Ekran küçüldüğünde (örn: tablet/telefon) tasarımın bozulmamasını sağlayın.

  • Bir Medya Sorgusu (@media) yazın: max-width: 768px için:
  • Main Grid: grid-template-columns değerini 1fr yaparak tek sütuna düşürün.
  • Container: Genişliği (width) 100% yapın.
@media (max-width: 768px) {
    main {
        grid-template-columns: 1fr;
    }

    .container {
        width: 100%;
    }
}
