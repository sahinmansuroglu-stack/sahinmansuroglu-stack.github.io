Mesaj ve Dialog Kutuları

Kullanıcı ile Etkileşim ve Dosya İşlemleri

Anasayfa Kontroller Dialog Kutuları Çözümlü Projeler Proje Soruları

1. MessageBox (Mesaj Kutusu)

Kullanıcıya uyarı, bilgi veya soru sormak için açılan küçük pencerelerdir.

Basit Mesaj

MessageBox.Show("İşlem Başarılı!");

Başlık ve Butonlu Mesaj

MessageBox.Show("Kaydetmek istiyor musunuz?", "Onay", MessageBoxButtons.YesNo);

İkonlu Mesaj

MessageBox.Show("Hata oluştu!", "Hata", MessageBoxButtons.OK, MessageBoxIcon.Error);

Cevabı Kontrol Etme

DialogResult sonuc = MessageBox.Show("Çıkış yapılsın mı?", "Çıkış", MessageBoxButtons.YesNo, MessageBoxIcon.Question);

if (sonuc == DialogResult.Yes)
{
    Application.Exit(); // Uygulamayı kapat
}

2. OpenFileDialog (Dosya Açma Penceresi)

Kullanıcının bilgisayardan bir dosya seçmesini sağlar.

OpenFileDialog dosyaAc = new OpenFileDialog();
dosyaAc.Filter = "Metin Dosyaları|*.txt|Tüm Dosyalar|*.*";
dosyaAc.Title = "Bir Dosya Seçin";

if (dosyaAc.ShowDialog() == DialogResult.OK)
{
    string dosyaYolu = dosyaAc.FileName;
    MessageBox.Show("Seçilen Dosya: " + dosyaYolu);
}

3. SaveFileDialog (Dosya Kaydetme Penceresi)

Kullanıcının bir dosyayı nereye kaydedeceğini seçmesini sağlar.

SaveFileDialog dosyaKaydet = new SaveFileDialog();
dosyaKaydet.Filter = "Metin Dosyaları|*.txt";

if (dosyaKaydet.ShowDialog() == DialogResult.OK)
{
    // Dosya kaydetme işlemleri buraya (File.WriteAllText vb.)
    MessageBox.Show("Kaydedilecek yer: " + dosyaKaydet.FileName);
}

4. ColorDialog (Renk Seçme Penceresi)

Kullanıcının bir renk paletinden renk seçmesini sağlar.

ColorDialog renkSec = new ColorDialog();

if (renkSec.ShowDialog() == DialogResult.OK)
{
    this.BackColor = renkSec.Color; // Formun arka plan rengini değiştir
}

5. FontDialog (Yazı Tipi Seçme Penceresi)

Kullanıcının yazı tipi, boyutu ve stilini seçmesini sağlar.

FontDialog yaziTipi = new FontDialog();

if (yaziTipi.ShowDialog() == DialogResult.OK)
{
    label1.Font = yaziTipi.Font; // Label'ın fontunu değiştir
}
