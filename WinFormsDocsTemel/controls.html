Temel UI Kontrolleri

Windows Forms Uygulamalarının Yapı Taşları

Anasayfa Kontroller Dialog Kutuları Çözümlü Projeler Proje Soruları

1. Label (Etiket)

Kullanıcıya metin bilgisi göstermek için kullanılır. Genellikle statik yazılar için tercih edilir.

// Label'a kod ile yazı atama
label1.Text = "Merhaba Dünya!";
label1.ForeColor = Color.Red; // Yazı rengini değiştirme

2. TextBox (Metin Kutusu)

Kullanıcıdan veri girişi almak için kullanılır.

// TextBox'tan veri alma
string ad = textBox1.Text;

// TextBox'a veri yazma
textBox1.Text = "Varsayılan Değer";

// Şifreli karakter gösterme (PasswordChar)
textBox1.PasswordChar = '*';

3. Button (Düğme)

Kullanıcının bir işlemi tetiklemesi için kullanılır (Kaydet, Giriş Yap vb.).

// Buton tıklama olayı (Click Event)
private void button1_Click(object sender, EventArgs e)
{
    label1.Text = "Butona Tıklandı!";
    button1.Enabled = false; // Butonu pasif yapma
}

4. CheckBox (İşaret Kutusu)

Evet/Hayır şeklinde birden fazla seçim yapılmasına olanak tanır.

if (checkBox1.Checked)
{
    MessageBox.Show("Seçenek işaretlendi.");
}
else
{
    MessageBox.Show("Seçenek işaretlenmedi.");
}

5. RadioButton (Seçenek Düğmesi)

Bir grup seçenek arasından sadece bir tanesinin seçilmesine izin verir.

if (radioButton1.Checked)
{
    // 1. seçenek seçili
}
else if (radioButton2.Checked)
{
    // 2. seçenek seçili
}

6. ListBox (Liste Kutusu)

Alt alta sıralanmış verileri listelemek için kullanılır.

// Eleman ekleme
listBox1.Items.Add("Elma");
listBox1.Items.Add("Armut");

// Seçili elemanı silme
if (listBox1.SelectedIndex != -1)
{
    listBox1.Items.RemoveAt(listBox1.SelectedIndex);
}

// Tüm elemanları temizleme
listBox1.Items.Clear();

7. ComboBox (Açılır Liste)

ListBox'a benzer ama daha az yer kaplar, açılır bir menü sunar.

// Eleman ekleme
comboBox1.Items.Add("İstanbul");
comboBox1.Items.Add("Ankara");

// Seçili elemanı alma
string sehir = comboBox1.SelectedItem.ToString();
