Windows Forms Nedir?

Windows Forms (WinForms), Microsoft .NET Framework üzerinde çalışan, grafiksel kullanıcı arayüzü (GUI) oluşturmak için kullanılan bir teknolojidir.

📌 Önemli: Windows Forms ile masaüstü uygulamaları geliştirebilir, formlar üzerine butonlar, metin kutuları ve diğer kontrolleri sürükle-bırak yöntemiyle yerleştirebilirsiniz.

Windows Forms'un Avantajları

  • Kolay Kullanım: Sürükle-bırak ile hızlı tasarım
  • Zengin Kontrol Seti: 50'den fazla hazır kontrol
  • Olay Tabanlı Programlama: Click, KeyPress gibi olaylarla çalışma
  • Hızlı Geliştirme: Visual Studio ile entegre çalışma

Proje Oluşturma

Adımlar:

  1. Visual Studio'yu açın
  2. "Create a new project" seçin
  3. Arama kutusuna "Windows Forms App" yazın
  4. "Windows Forms App (.NET Framework)" şablonunu seçin
  5. Proje adını girin ve "Create" butonuna tıklayın

Program.cs - Başlangıç Noktası
using System;
using System.Windows.Forms;

namespace WindowsFormsApp1
{
    static class Program
    {
        [STAThread]
        static void Main()
        {
            Application.EnableVisualStyles();
            Application.SetCompatibleTextRenderingDefault(false);
            Application.Run(new Form1()); // Ana form başlatılır
        }
    }
}

Form Özellikleri (Properties)

Properties panelinden formun görünümünü ve davranışını değiştirebilirsiniz:

Özellik Açıklama Örnek Değer
Name Formun kod içindeki adı frmAna

Size Formun boyutu (Genişlik, Yükseklik) 800, 600
StartPosition Açılış konumu CenterScreen
BackColor Arka plan rengi White
FormBorderStyle Kenarlık stili FixedSingle
MaximizeBox Büyütme butonu aktif mi True / False
Icon Formun ikonu myicon.ico
Form Özelliklerini Kod ile Ayarlama
public Form1()
{
    InitializeComponent();
    
    // Form özelliklerini kod ile ayarlama
    this.Text = "Benim Uygulamam";
    this.Size = new Size(800, 600);
    this.StartPosition = FormStartPosition.CenterScreen;
    this.BackColor = Color.LightBlue;
    this.FormBorderStyle = FormBorderStyle.FixedSingle;
    this.MaximizeBox = false;
}

Form Olayları (Events)

Olay Ne Zaman Tetiklenir?
Load Form yüklendiğinde (açılmadan hemen önce)
Shown Form ilk kez gösterildiğinde
FormClosing Form kapatılmak üzereyken
FormClosed Form kapandıktan sonra
Resize Form boyutu değiştiğinde
Form Olayları Örneği
private void Form1_Load(object sender, EventArgs e)
{
    MessageBox.Show("Form yükleniyor...");
}

private void Form1_FormClosing(object sender, FormClosingEventArgs e)
{
    DialogResult sonuc = MessageBox.Show(
        "Çıkmak istediğinizden emin misiniz?",
        "Çıkış",
        MessageBoxButtons.YesNo,
        MessageBoxIcon.Question);
    
    if (sonuc == DialogResult.No)
    {
        e.Cancel = true; // Kapanmayı iptal et
    }
}
