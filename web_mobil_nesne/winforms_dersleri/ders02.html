1. Label (Etiket)

Label, kullanıcıya metin bilgisi göstermek için kullanılır. Genellikle statik yazılar veya açıklamalar için tercih edilir.

Bu bir Label kontrolüdür
Özellik Açıklama
Text Gösterilecek metin
ForeColor Yazı rengi
BackColor Arka plan rengi
Font Yazı tipi ve boyutu
AutoSize Metne göre otomatik boyutlanma
TextAlign Metnin hizalanması
Label Kullanımı
// Label'a yazı atama
label1.Text = "Merhaba Dünya!";

// Yazı rengini değiştirme
label1.ForeColor = Color.Red;

// Yazı tipini değiştirme
label1.Font = new Font("Arial", 14, FontStyle.Bold);

// Arka plan rengi
label1.BackColor = Color.Yellow;

2. TextBox (Metin Kutusu)

TextBox, kullanıcıdan veri girişi almak için kullanılır. En temel ve en çok kullanılan kontrollerden biridir.

Özellik Açıklama
Text Kutudaki metin
MaxLength Maksimum karakter sayısı
PasswordChar Şifre karakteri (* veya ●)
Multiline Çok satırlı giriş
ReadOnly Salt okunur
PlaceholderText İpucu metni (.NET 5+)
TextBox Kullanımı
// TextBox'tan değer alma
string ad = textBox1.Text;

// TextBox'a değer yazma
textBox1.Text = "Varsayılan Değer";

// Şifreli karakter gösterme
txtSifre.PasswordChar = '*';

// Maksimum karakter sınırı
txtTelefon.MaxLength = 11;

// Çok satırlı TextBox
txtAciklama.Multiline = true;
txtAciklama.ScrollBars = ScrollBars.Vertical;

// TextBox'ı temizleme
textBox1.Clear();

3. Button (Düğme)

Button, kullanıcının bir işlemi tetiklemesi için kullanılır. Tıklandığında Click olayı çalışır.

Özellik Açıklama
Text Buton üzerindeki yazı
Enabled Aktif/Pasif durumu
BackColor Arka plan rengi
FlatStyle Düz stil görünümü
Image Buton üzerinde resim
Button Click Olayı
private void button1_Click(object sender, EventArgs e)
{
    // Label'a yazı yaz
    label1.Text = "Butona Tıklandı!";
    
    // Butonu pasif yap
    button1.Enabled = false;
}

// Buton özelliklerini değiştirme
button1.Text = "Kaydet";
button1.BackColor = Color.Green;
button1.ForeColor = Color.White;

Pratik Örnek: Selamlama Uygulaması

Kullanıcının adını alıp selamlayan basit bir uygulama yapalım:

Merhaba, Ali!

Selamlama Uygulaması Kodu
private void btnSelamla_Click(object sender, EventArgs e)
{
    string ad = txtAd.Text;
    
    if (string.IsNullOrWhiteSpace(ad))
    {
        MessageBox.Show("Lütfen adınızı girin!");
        return;
    }
    
    lblSonuc.Text = "Merhaba, " + ad + "!";
    
    // veya string interpolation ile:
    lblSonuc.Text = $"Merhaba, {ad}!";
}

2 Şifre giriş ekranı yapın. Şifre "1234" ise "Giriş Başarılı", değilse "Hatalı Şifre" yazsın. 
private void btnGiris_Click(object sender, EventArgs e)
{
    if (txtSifre.Text == "1234")
    {
        lblSonuc.Text = "Giriş Başarılı!";
        lblSonuc.ForeColor = Color.Green;
    }
    else
    {
        lblSonuc.Text = "Hatalı Şifre!";
        lblSonuc.ForeColor = Color.Red;
    }
}
