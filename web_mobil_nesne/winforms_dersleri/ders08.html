1. MenuStrip (Ana Menü)

MenuStrip, uygulamanın üst kısmındaki ana menüyü oluşturur. Notepad, Word gibi uygulamalardaki "Dosya", "Düzen", "Görünüm" menüleri buna örnektir.

Dosya Düzen Görünüm Yardım

MenuStrip Oluşturma (Tasarım Görünümü)

  1. Toolbox'tan MenuStrip sürükleyin
  2. "Type Here" yazan yere menü başlığı yazın (örn: "Dosya")
  3. Alt menüler için aşağı tıklayarak devam edin
  4. Çizgi için "-" (tire) yazın
💡 Kısayol Tuşu: Menü adının önüne & koyarak kısayol belirleyin: "&Dosya" → Alt+D ile açılır

MenuStrip Olayları

Menü Click Olayları
// Dosya > Yeni
private void yeniToolStripMenuItem_Click(object sender, EventArgs e)
{
    txtIcerik.Clear();
    this.Text = "Yeni Belge";
}

// Dosya > Aç
private void acToolStripMenuItem_Click(object sender, EventArgs e)
{
    OpenFileDialog dialog = new OpenFileDialog();
    dialog.Filter = "Metin Dosyaları|*.txt";
    
    if (dialog.ShowDialog() == DialogResult.OK)
    {
        txtIcerik.Text = File.ReadAllText(dialog.FileName);
        this.Text = Path.GetFileName(dialog.FileName);
    }
}

// Dosya > Kaydet
private void kaydetToolStripMenuItem_Click(object sender, EventArgs e)
{
    SaveFileDialog dialog = new SaveFileDialog();
    dialog.Filter = "Metin Dosyaları|*.txt";
    
    if (dialog.ShowDialog() == DialogResult.OK)
    {
        File.WriteAllText(dialog.FileName, txtIcerik.Text);
        MessageBox.Show("Dosya kaydedildi!");
    }
}

// Dosya > Çıkış
private void cikisToolStripMenuItem_Click(object sender, EventArgs e)
{
    Application.Exit();
}

Kısayol Tuşları (ShortcutKeys)

Properties panelinden ShortcutKeys özelliğini ayarlayarak menü öğelerine kısayol atayabilirsiniz.

Menü Öğesi Kısayol
Yeni Ctrl+N
Aç Ctrl+O
Kaydet Ctrl+S
Kes Ctrl+X
Kopyala Ctrl+C
Yapıştır Ctrl+V
Geri Al Ctrl+Z
Kod ile Kısayol Atama
// Form_Load'da
kaydetToolStripMenuItem.ShortcutKeys = Keys.Control | Keys.S;
acToolStripMenuItem.ShortcutKeys = Keys.Control | Keys.O;
yeniToolStripMenuItem.ShortcutKeys = Keys.Control | Keys.N;

2. ToolStrip (Araç Çubuğu)

ToolStrip, sık kullanılan komutlara hızlı erişim sağlayan ikon butonları içerir.

ToolStrip Öğeleri

Öğe Açıklama
ToolStripButton Tıklanabilir buton
ToolStripSeparator Ayırıcı çizgi
ToolStripLabel Metin etiketi
ToolStripTextBox Metin kutusu
ToolStripComboBox Açılır liste
ToolStripDropDownButton Açılır menü butonu
ToolStrip Olayları
// ToolStrip butonları için ortak handler
private void toolStrip_Click(object sender, EventArgs e)
{
    ToolStripButton btn = (ToolStripButton)sender;
    
    switch (btn.Name)
    {
        case "tsbYeni":
            YeniDosya();
            break;
        case "tsbAc":
            DosyaAc();
            break;
        case "tsbKaydet":
            DosyaKaydet();
            break;
    }
}

// Arama kutusu
private void tstArama_TextChanged(object sender, EventArgs e)
{
    string arama = tstArama.Text;
    // Arama işlemi...
}

// Yazı boyutu ComboBox
private void tscYaziBoyutu_SelectedIndexChanged(object sender, EventArgs e)
{
    int boyut = int.Parse(tscYaziBoyutu.SelectedItem.ToString());
    txtIcerik.Font = new Font(txtIcerik.Font.FontFamily, boyut);
}

3. ContextMenuStrip (Sağ Tık Menüsü)

ContextMenuStrip, bir kontrole sağ tıklandığında açılan menüdür.

ContextMenuStrip Kullanımı
// Tasarım görünümünde:
// 1. ContextMenuStrip ekleyin
// 2. Kontrolün ContextMenuStrip özelliğine atayın

// Veya kod ile bağlama
listBox1.ContextMenuStrip = contextMenuStrip1;

// Menü açılmadan önce kontrol
private void contextMenuStrip1_Opening(object sender, CancelEventArgs e)
{
    // Eleman seçili değilse menüyü açma
    if (listBox1.SelectedIndex == -1)
    {
        e.Cancel = true;
    }
}

// Sağ tık > Sil
private void silToolStripMenuItem_Click(object sender, EventArgs e)
{
    if (listBox1.SelectedIndex != -1)
    {
        listBox1.Items.RemoveAt(listBox1.SelectedIndex);
    }
}

// Sağ tık > Düzenle
private void duzenleToolStripMenuItem_Click(object sender, EventArgs e)
{
    if (listBox1.SelectedItem != null)
    {
        txtDuzenle.Text = listBox1.SelectedItem.ToString();
    }
}

private string mevcutDosya = "";

private void kaydetToolStripMenuItem_Click(object sender, EventArgs e)
{

🎯 Sıra Sende!

1 Dosya menüsü içeren (Yeni, Aç, Kaydet, Çıkış) bir MenuStrip oluşturun.
  1. Toolbox'tan MenuStrip sürükleyin
  2. "Dosya" yazın, Enter'a basın
  3. Alt öğeler: Yeni, Aç, Kaydet, -, Çıkış
  4. Her öğeye çift tıklayarak event handler oluşturun
2 Yeni, Aç, Kaydet butonları içeren bir ToolStrip ekleyin.
  1. ToolStrip ekleyin
  2. Açılır menüden Button ekleyin
  3. Image özelliğinden ikon seçin veya Text yazın
  4. ToolTipText ile ipucu ekleyin
3 ListBox'a sağ tık menüsü ekleyin (Sil, Düzenle). 
// ContextMenuStrip oluştur ve bağla
listBox1.ContextMenuStrip = contextMenuStrip1;

private void silToolStripMenuItem_Click(object sender, EventArgs e)
{
    if (listBox1.SelectedIndex != -1)
        listBox1.Items.RemoveAt(listBox1.SelectedIndex);
}
