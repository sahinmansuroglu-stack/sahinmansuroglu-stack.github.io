1. CheckBox (İşaret Kutusu)

CheckBox, evet/hayır şeklinde birden fazla seçim yapılmasına olanak tanır. Birbirinden bağımsız seçenekler için kullanılır.



Özellik Açıklama
Checked İşaretli mi? (true/false)
Text Yanındaki yazı
CheckState Checked, Unchecked, Indeterminate
ThreeState Üç durumlu mı?
CheckBox Kullanımı
// CheckBox durumunu kontrol etme
if (chkKahvalti.Checked)
{
    MessageBox.Show("Kahvaltı seçildi.");
}

// Birden fazla CheckBox kontrolü
string secimler = "";

if (chkPeynir.Checked) secimler += "Peynir, ";
if (chkZeytin.Checked) secimler += "Zeytin, ";
if (chkYumurta.Checked) secimler += "Yumurta, ";

if (!string.IsNullOrEmpty(secimler))
{
    lblSonuc.Text = "Seçimler: " + secimler.TrimEnd(',', ' ');
}
CheckedChanged Olayı
private void chkOtoKayit_CheckedChanged(object sender, EventArgs e)
{
    if (chkOtoKayit.Checked)
    {
        lblDurum.Text = "Otomatik kayıt AKTİF";
    }
    else
    {
        lblDurum.Text = "Otomatik kayıt KAPALI";
    }
}

2. RadioButton (Seçenek Düğmesi)

RadioButton, bir grup seçenek arasından sadece birinin seçilmesine izin verir.



Özellik Açıklama
Checked Seçili mi? (true/false)
Text Yanındaki yazı
RadioButton Kullanımı
// Hangi RadioButton seçili?
if (rbErkek.Checked)
{
    lblCinsiyet.Text = "Cinsiyet: Erkek";
}
else if (rbKadin.Checked)
{
    lblCinsiyet.Text = "Cinsiyet: Kadın";
}

// Pratik yöntem - metot ile
private string SecilenCinsiyet()
{
    if (rbErkek.Checked) return "Erkek";
    if (rbKadin.Checked) return "Kadın";
    return "Belirtilmedi";
}

3. GroupBox (Grup Kutusu)

GroupBox, ilişkili kontrolleri bir arada gruplamak için kullanılır. RadioButton'lar için özellikle önemlidir - aynı GroupBox içindeki RadioButton'lar bir grup oluşturur.

Cinsiyet
Medeni Durum
📌 Önemli: Aynı form üzerinde birden fazla RadioButton grubu oluşturmak için her grubu ayrı bir GroupBox içine yerleştirin. Böylece her grup bağımsız çalışır.
Kod ile GroupBox Oluşturma
// Kod ile GroupBox oluşturma
GroupBox gbCinsiyet = new GroupBox();
gbCinsiyet.Text = "Cinsiyet";
gbCinsiyet.Location = new Point(20, 20);
gbCinsiyet.Size = new Size(150, 100);

RadioButton rbErkek = new RadioButton();
rbErkek.Text = "Erkek";
rbErkek.Location = new Point(10, 25);
rbErkek.Checked = true;

RadioButton rbKadin = new RadioButton();
rbKadin.Text = "Kadın";
rbKadin.Location = new Point(10, 50);

// RadioButton'ları GroupBox'a ekle
gbCinsiyet.Controls.Add(rbErkek);
gbCinsiyet.Controls.Add(rbKadin);

// GroupBox'ı forma ekle
this.Controls.Add(gbCinsiyet);

Pratik Örnek: Sipariş Formu

Pizza Sipariş Formu
private void btnSiparis_Click(object sender, EventArgs e)
{
    // Pizza boyutu (RadioButton)
    string boyut = "";
    decimal fiyat = 0;
    
    if (rbKucuk.Checked) { boyut = "Küçük"; fiyat = 50; }
    else if (rbOrta.Checked) { boyut = "Orta"; fiyat = 75; }
    else if (rbBuyuk.Checked) { boyut = "Büyük"; fiyat = 100; }
    
    // Ekstra malzemeler (CheckBox)
    string ekstralar = "";
    
    if (chkPeynir.Checked) { ekstralar += "Ekstra Peynir, "; fiyat += 10; }
    if (chkSucuk.Checked) { ekstralar += "Sucuk, "; fiyat += 15; }
    if (chkMantar.Checked) { ekstralar += "Mantar, "; fiyat += 8; }
    if (chkZeytin.Checked) { ekstralar += "Zeytin, "; fiyat += 5; }
    
    // Sonucu göster
    string siparis = $@"
=== SİPARİŞ ÖZETİ ===
Boyut: {boyut}
Ekstralar: {(ekstralar.Length > 0 ? ekstralar.TrimEnd(',', ' ') : "Yok")}
TOPLAM: {fiyat:C}";
    
    MessageBox.Show(siparis, "Sipariş");
}

Tasarım görünümünde:

  2. İçine rbErkek ve rbKadin RadioButton ekleyin
  3. İkinci GroupBox sürükleyin, Text = "Medeni Durum"
  4. İçine rbBekar ve rbEvli RadioButton ekleyin
