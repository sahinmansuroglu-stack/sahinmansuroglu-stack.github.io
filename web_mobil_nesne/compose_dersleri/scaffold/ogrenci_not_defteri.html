1
Veri Modeli — Öğrenci sınıfını oluştur

Önce bir öğrencinin hangi bilgileri tutacağını tanımlayalım. Buna data class diyoruz. 

// Bir öğrencinin bilgilerini tutan veri sınıfı
data class Ogrenci(
    val id: Int,          // Benzersiz numara
    val ad: String,        // Öğrenci adı
    val soyad: String,     // Öğrenci soyadı
    val not1: Int,         // 1. ders notu
    val not2: Int          // 2. ders notu
)

💡data class nedir? Sadece veri taşımak için kullanılan özel bir sınıftır. Kotlin otomatik olarak toString(), equals(), copy() gibi fonksiyonları üretir.

2
State Yönetimi — Verileri hafızada tut

Öğrenci listesini, dialog/sheet durumlarını state olarak tanımlıyoruz. State değişince ekran güncellenir.

// Öğrenci listesi (eklenip silinebilir)
var ogrenciler by remember {
    mutableStateOf(
        listOf(
            Ogrenci(1, "Ahmet", "Yılmaz", 85, 92),
            Ogrenci(2, "Elif", "Kaya", 78, 65)
        )
    )
}

// BottomSheet açık mı?
var sheetAcik by remember { mutableStateOf(false) }

// Dialog açık mı? Hangi öğrenci silinecek?
var silinecekOgrenci by remember {
    mutableStateOf<Ogrenci?>(null)
}

// Form alanları (yeni öğrenci eklerken)
var yeniAd by remember { mutableStateOf("") }
var yeniSoyad by remember { mutableStateOf("") }
var yeniNot1 by remember { mutableStateOf("") }
var yeniNot2 by remember { mutableStateOf("") }

🔑Neden silinecekOgrenci nullable (Ogrenci?)?

Eğer null ise dialog kapalıdır. Bir öğrenci atandığında dialog açılır ve o öğrencinin bilgisi gösterilir.

7
Öğrenci Kartı — Card bileşeni

Her öğrencinin bilgilerini gösteren kart:

@Composable
fun OgrenciKarti(
    ogrenci: Ogrenci,
    onSil: () -> Unit   // Silme butonuna tıklanınca
) {
    Card(
        modifier = Modifier.fillMaxWidth(),
        elevation = CardDefaults.cardElevation(4.dp)
    ) {
        Row(
            modifier = Modifier.padding(16.dp).fillMaxWidth(),
            verticalAlignment = Alignment.CenterVertically
        ) {
            Column(modifier = Modifier.weight(1f)) {
                Text(
                    "${ogrenci.ad} ${ogrenci.soyad}",
                    style = MaterialTheme.typography.titleMedium
                )
                Spacer(modifier = Modifier.height(4.dp))
                Row(horizontalArrangement = Arrangement.spacedBy(8.dp)) {
                    Text("Not 1: ${ogrenci.not1}",
                        style = MaterialTheme.typography.bodySmall)
                    Text("Not 2: ${ogrenci.not2}",
                        style = MaterialTheme.typography.bodySmall)
                }
            }
            // Silme butonu
            IconButton(onClick = onSil) {
                Icon(Icons.Default.Delete,
                    "Sil", tint = Color.Red)
            }
        }
    }
}

LazyColumn — Kartları listele
// Scaffold'un content lambda'sı içinde:
LazyColumn(
    modifier = Modifier.padding(paddingValues),
    contentPadding = PaddingValues(16.dp),
    verticalArrangement = Arrangement.spacedBy(12.dp)
) {
    items(ogrenciler) { ogrenci ->
        OgrenciKarti(
            ogrenci = ogrenci,
            onSil = { silinecekOgrenci = ogrenci }
            // ↑ Dialog'u aç, silinecek öğrenciyi kaydet
        )
    }
}

⚠️items(ogrenciler) — Burada items(10) yerine doğrudan listeyi veriyoruz. Liste her değiştiğinde ekran otomatik güncellenir!

                    ogrenciler = ogrenciler + Ogrenci(
                        yeniId, yeniAd, yeniSoyad,
                        yeniNot1.toIntOrNull() ?: 0,
                        yeniNot2.toIntOrNull() ?: 0

                    yeniNot1 = ""; yeniNot2 = ""
                    sheetAcik = false
                },
                modifier = Modifier.fillMaxWidth()
            ) {
                Text("Kaydet")
            }
            Spacer(modifier = Modifier.height(32.dp))
        }
    }
}

Tam Kod — Hepsini birleştir
📄 OgrenciNotDefteri() — Tüm kodu göster 
@OptIn(ExperimentalMaterial3Api::class)
@Composable
fun OgrenciNotDefteri() {
    var ogrenciler by remember { mutableStateOf(listOf(
        Ogrenci(1,"Ahmet","Yılmaz",85,92),
        Ogrenci(2,"Elif","Kaya",78,65)
    )) }
    var sheetAcik by remember { mutableStateOf(false) }
    var silinecekOgrenci by remember { mutableStateOf<Ogrenci?>(null) }
    var yeniAd by remember { mutableStateOf("") }
    var yeniSoyad by remember { mutableStateOf("") }
    var yeniNot1 by remember { mutableStateOf("") }
    var yeniNot2 by remember { mutableStateOf("") }

    Scaffold(
        topBar = {
            TopAppBar(
                title = { Text("📓 Öğrenci Not Defteri") },
                navigationIcon = {
                    IconButton(onClick = { }) {
                        Icon(Icons.Default.Menu, "Menü") }
                },
                actions = { IconButton(onClick = { }) {
                    Icon(Icons.Default.Search, "Ara") } },
                colors = TopAppBarDefaults.topAppBarColors(
                    containerColor = MaterialTheme.colorScheme.primaryContainer)
            )
        },
        floatingActionButton = {
            FloatingActionButton(onClick = { sheetAcik = true }) {
                Icon(Icons.Default.Add, "Ekle")
            }
        },
        bottomBar = {
            NavigationBar {
                NavigationBarItem(true, { },
                    { Icon(Icons.Default.List, null) },
                    label = { Text("Liste") })
                NavigationBarItem(false, { },
                    { Icon(Icons.Default.Star, null) },
                    label = { Text("İstatistik") })
                NavigationBarItem(false, { },
                    { Icon(Icons.Default.Settings, null) },
                    label = { Text("Ayarlar") })
            }
        }
    ) { paddingValues ->
        LazyColumn(
            modifier = Modifier.padding(paddingValues),
            contentPadding = PaddingValues(16.dp),
            verticalArrangement = Arrangement.spacedBy(12.dp)
        ) {
            items(ogrenciler) { ogr ->
                OgrenciKarti(ogr) { silinecekOgrenci = ogr }
            }
        }
    }

    // BottomSheet
    if (sheetAcik) {
        ModalBottomSheet(onDismissRequest = { sheetAcik = false },
            sheetState = rememberModalBottomSheetState()
        ) {
            Column(Modifier.padding(16.dp)) {
                Text("Yeni Öğrenci", style = MaterialTheme.typography.headlineSmall)
                Spacer(Modifier.height(12.dp))
                OutlinedTextField(yeniAd, { yeniAd = it }, label = { Text("Ad") }, modifier = Modifier.fillMaxWidth())
                OutlinedTextField(yeniSoyad, { yeniSoyad = it }, label = { Text("Soyad") }, modifier = Modifier.fillMaxWidth())
                OutlinedTextField(yeniNot1, { yeniNot1 = it }, label = { Text("Not 1") }, modifier = Modifier.fillMaxWidth())
                OutlinedTextField(yeniNot2, { yeniNot2 = it }, label = { Text("Not 2") }, modifier = Modifier.fillMaxWidth())
                Spacer(Modifier.height(12.dp))
                Button(onClick = {
                    val id = (ogrenciler.maxOfOrNull { it.id } ?: 0) + 1
                    ogrenciler = ogrenciler + Ogrenci(id, yeniAd, yeniSoyad,
                        yeniNot1.toIntOrNull() ?: 0, yeniNot2.toIntOrNull() ?: 0)
                    yeniAd=""; yeniSoyad=""; yeniNot1=""; yeniNot2=""
                    sheetAcik = false
                }, Modifier.fillMaxWidth()) { Text("Kaydet") }
                Spacer(Modifier.height(32.dp))
            }
        }
    }

    // Dialog
    silinecekOgrenci?.let { ogr ->
        AlertDialog(
            onDismissRequest = { silinecekOgrenci = null },
            title = { Text("Öğrenciyi Sil") },
            text = { Text("${ogr.ad} ${ogr.soyad} silinsin mi?") },
            confirmButton = { TextButton(onClick = {
                ogrenciler = ogrenciler.filter { it.id != ogr.id }
                silinecekOgrenci = null
            }) { Text("Sil", color = Color.Red) } },
            dismissButton = { TextButton(onClick = {
                silinecekOgrenci = null
            }) { Text("İptal") } }
        )
    }
}

🧑‍💻 Sıra Sende!

Aşağıdaki ipuçlarını sırayla oku. Her biri seni çözüme bir adım yaklaştırır:

💡 İpuçları
İpucu 1: Ortalamayı Nerede Hesaplamalı?
🔹 data class Ogrenci içine bir hesaplanmış özellik (computed property) ekleyebilirsin. Kotlin'de val ortalama: Double get() = ... şeklinde bir özellik tanımlanabilir.
📐 Yapı şöyle olmalı: data class'ın süslü parantezleri içine yaz. Constructor parametresi olarak değil, body içinde tanımla. Yani data class Ogrenci(...) { val ortalama: Double get() = ... } şeklinde.
⚠️ Dikkat: get() ile tanımlanan özellikler her çağrıldığında hesaplanır, bellekte yer tutmaz. Bu projede performans farkı olmaz ama bunu bilmek önemli.
İpucu 2: Ortalama Formülü
🔹 İki notun ortalaması: (not1 + not2) / 2.0
⚠️ Neden 2.0? Kotlin'de Int / Int sonucu yine Int olur. Yani 85 / 2 = 42 (noktadan sonrası kaybolur!). 2.0 yazınca sonuç Double olur: 85 / 2.0 = 42.5. Alternatif olarak .toDouble() de kullanabilirsin.
İpucu 3: Kartda Nasıl Gösterilir?
🔹 OgrenciKarti composable'ında, Not 1 ve Not 2'yi gösteren Row'un hemen altına yeni bir Text ekle.
📐 String template kullan: "Ortalama: ${ogrenci.ortalama}" — Ayrıca bu Text'e style = MaterialTheme.typography.bodyMedium ve fontWeight = FontWeight.Bold ekleyerek öne çıkarabilirsin.
💡 Konum: Column içindeki, Row(horizontalArrangement = ...) bloğunun altına, Spacer'dan sonra eklemen en uygun yer.
İpucu 4: Renk Nasıl Değiştirilir?
🔹 Önce bir renk değişkeni oluştur, sonra onu Text'in color parametresine ver.
📐 Yapı şöyle:
val renk = when {
  ogrenci.ortalama < 50 -> Color.Red
  ogrenci.ortalama < 70 -> Color(0xFFFF9800) // Turuncu
  else -> Color(0xFF4CAF50) // Yeşil
}
💡 Kullanım: Text("Ort: ...", color = renk) — Bu değişkeni OgrenciKarti fonksiyonunun başında (return'den önce) tanımla.
İpucu 5: Ondalık Basamak Kontrolü
🔹 Ortalamanın çok uzun görünmemesi için "%.1f".format(ortalama) kullanabilirsin. Bu, 1 ondalık basamak gösterir (örn: 88.5).
📐 Tam kullanım: Text("Ort: ${"%.1f".format(ogrenci.ortalama)}") — String template içinde format kullanırken ${...} süslü parantez gerekir çünkü nokta var.
💡 Bonus: Geçen öğrenci için ✓, kalan için ✗ sembolü de ekleyebilirsin: if(ortalama >= 50) "✓" else "✗"

✅Başarı Kriterleri:

  • Her kartda not ortalaması görünüyor
  • Ortalama 50 altı → kırmızı renk
  • Ortalama 50-70 arası → turuncu renk
  • Ortalama 70 üstü → yeşil renk

🧑‍💻 Sıra Sende!

Filtreleme ve sekme yönetimi biraz daha zor. İpuçlarını dikkatlice takip et:

💡 İpuçları
İpucu 1: Aktif Sekmeyi Nasıl Takip Edersin?
🔹 Yeni bir state tanımla: var aktifSekme by remember { mutableStateOf(0) }
📐 Numaralandırma: 0 = Tümü, 1 = Geçenler, 2 = Kalanlar, 3 = En İyi. Bunu diğer state'lerin yanına (sheetAcik, silinecekOgrenci vb.) ekle.
💡 İleri seviye: Sayı yerine enum class Sekme { TUMU, GECENLER, KALANLAR, EN_IYI } de tanımlayabilirsin — daha okunur kod olur.
İpucu 2: NavigationBarItem'ı Nasıl Güncellersin?
🔹 Her NavigationBarItem'ın selected parametresini aktifSekme == 0, aktifSekme == 1 gibi kontrol et.
📐 Yapı şöyle:
NavigationBarItem(
  selected = aktifSekme == 1,
  onClick = { aktifSekme = 1 },
  icon = { Icon(Icons.Default.CheckCircle, null) },
  label = { Text("Geçenler") }
)
⚠️ Dikkat: Mevcut 3 sekmeyi (Liste, İstatistik, Ayarlar) silip 4 yenisiyle değiştir. Toplam 4 item: Tümü (0), Geçenler (1), Kalanlar (2), En İyi (3).
İpucu 3: Listeyi Nasıl Filtrelersin?
🔹 Kotlin'in filter fonksiyonunu düşün: ogrenciler.filter { it.ortalama >= 50 } geçenleri verir.
📐 LazyColumn'da kullanımı: items(ogrenciler) yerine items(filtrelenmisListe) yaz. Filtrelenmiş listeyi LazyColumn'dan hemen önce hesapla.
💡 Önemli: filter orijinal listeyi değiştirmez, yeni bir liste döndürür. Bu yüzden ogrenciler state'i güvende kalır.
İpucu 4: when ile Sekmeye Göre Filtreleme
🔹 Scaffold'un content lambda'sında, LazyColumn'dan önce filtreleme yap:
📐 Yapı şöyle:
val gosterilecekler = when(aktifSekme) {
  0 -> ogrenciler // Tümü
  1 -> ogrenciler.filter { it.ortalama >= 50 }
  2 -> ogrenciler.filter { it.ortalama < 50 }
  3 -> ogrenciler.sortedByDescending { it.ortalama }.take(3)
  else -> ogrenciler
}
⚠️ Dikkat: when kullanırken else dalını unutma! Kotlin when ifadesi tüm durumları karşılamalı.
İpucu 5: Geçme Notu Kaç?
🔹 Geçme barajı olarak 50 kullanabilirsin. it.ortalama >= 50 → geçen, it.ortalama < 50 → kalan.
📐 Sabit tanımla: Kodun en üstüne private const val GECME_NOTU = 50.0 ekle. Böylece geçme notu değişirse tek yerden güncelersin: it.ortalama >= GECME_NOTU
İpucu 6: Icon Seçimi
🔹 Her sekme için uygun icon:
📋 Tümü → Icons.Default.List
✅ Geçenler → Icons.Default.CheckCircle
⚠️ Kalanlar → Icons.Default.Warning
🏆 En İyi → Icons.Default.Star veya Icons.Default.EmojiEvents
İpucu 7: Liste Boşsa Ne Gösterilir?
🔹 Filtrelenmiş liste boş olabilir. LazyColumn'dan önce kontrol et:
📐 Yapı şöyle:
if (gosterilecekler.isEmpty()) {
  // Ekranın ortasına mesaj göster
  Box(Modifier.fillMaxSize(), contentAlignment = Center) {
    Text("Bu kategoride öğrenci yok")
  }
} else { LazyColumn(...) { ... } }

✅Başarı Kriterleri:

  • Sekmelere tıklayınca ilgili öğrenciler filtreleniyor
  • Aktif sekme görsel olarak seçili görünüyor
  • Liste boşken "Öğrenci bulunamadı" gibi bir mesaj gösteriliyor

🧑‍💻 Sıra Sende!

Sıralama ve özel tasarım gerektiren bu özellik için ipuçları:

💡 İpuçları
İpucu 1: Listeyi Nasıl Sıralarsın?
🔹 Kotlin'de sortedByDescending fonksiyonu var: ogrenciler.sortedByDescending { it.ortalama } — Bu en yüksek ortalamalıyı başa koyar.
📐 Farkı anla: sortedBy → küçükten büyüğe (ascending), sortedByDescending → büyükten küçüğe (descending). En iyiler için descending kullan.
💡 Önemli: sortedByDescending orijinal listeyi değiştirmez, sıralanmış yeni bir liste döndürür.
İpucu 2: Sadece İlk 3'ü Nasıl Alırsın?
🔹 .take(3) fonksiyonunu zincirle: ogrenciler.sortedByDescending { it.ortalama }.take(3)
⚠️ Dikkat: Listede 3'ten az öğrenci varsa take(3) hata vermez, mevcut kadarını döndürür. Yani 2 öğrenci varsa 2 döner. Kotlin bunu senin için halleder!
İpucu 3: Sıra Numarasını Nasıl Bilirsin?
🔹 Normalde items(liste) kullanıyorduk. Sıra numarası için itemsIndexed kullan.
📐 Yapı şöyle:
itemsIndexed(enIyiler) { index, ogr ->
  // index: 0, 1, 2
  // ogr: Ogrenci nesnesi
}
💡 Hatırla: index 0'dan başlar! 0 = birinci (🥇), 1 = ikinci (🥈), 2 = üçüncü (🥉).
İpucu 4: Madalya Emojisi ve Renk Atama
🔹 when(index) ile hem madalya hem renk belirle:
📐 Yapı şöyle:
val (madalya, renk) = when(index) {
  0 -> "🥇" to Color(0xFFFBBF24) // Altın
  1 -> "🥈" to Color(0xFF94A3B8) // Gümüş
  2 -> "🥉" to Color(0xFFB45309) // Bronz
  else -> "" to Color.Gray
}
💡 Destructuring: val (a, b) = Pair yapısı Kotlin'de destructuring declaration olarak bilinir. to operatörü Pair oluşturur.
İpucu 5: Özel Kart Tasarımı
🔹 Card'ın colors parametresini kullan: CardDefaults.cardColors(containerColor = ...)
📐 Arka plan renkleri:
Altın kartı → Color(0xFFFFFBEB) (açık sarı)
Gümüş kartı → Color(0xFFF8FAFC) (açık gri)
Bronz kartı → Color(0xFFFFF7ED) (açık turuncu)
💡 Ekstra süsleme: Kartın soluna renkli çizgi eklemek için Modifier.border(width = 4.dp, color = renk, shape = ...) veya Card içine bir Box koyup sol kenara renkli bir şerit ekleyebilirsin.
İpucu 6: TopAppBar Başlığını Değiştirme
🔹 Aktif sekme "En İyi" olduğunda TopAppBar başlığını da değiştirebilirsin:
📐 Yapı: title = { Text(if(aktifSekme == 3) "🏆 En İyi Öğrenciler" else "📓 Not Defteri") }
💡 Bonus: TopAppBar'ın rengini de değiştirebilirsin: containerColor = if(aktifSekme == 3) Color(0xFFF59E0B) else MaterialTheme.colorScheme.primaryContainer

✅Başarı Kriterleri:

  • En İyi sekmesinde en yüksek 3 öğrenci sıralı görünüyor
  • Her sıra farklı renk/madalya ile süslenmiş
  • 3'ten az öğrenci varsa mevcut kadar gösteriliyor
  • Yeni öğrenci eklenince sıralama otomatik güncelleniyor

Arama özelliği birden fazla state ve koşullu UI gerektiriyor. İpuçlarını sırayla takip et:

💡 İpuçları
İpucu 1: Arama State'leri
🔹 İki yeni state lazım:
📐 Yapı:
var aramaAcik by remember { mutableStateOf(false) }
var aramaMetni by remember { mutableStateOf("") }
İlki arama çubuğunun açık olup olmadığını, ikincisi aranan metni tutar.
İpucu 2: Arama İkonuna onClick Ekle
🔹 Mevcut arama ikonunun onClick'i boş: onClick = { }. Bunu onClick = { aramaAcik = true } yap.
📐 actions bloğunda:
actions = {
  IconButton(onClick = { aramaAcik = !aramaAcik }) {
    Icon(Icons.Default.Search, "Ara")
  }
}
İpucu 3: Arama Çubuğunu Nereye Koymalı?
🔹 aramaAcik true olduğunda, LazyColumn'ın üstüne (Scaffold content içinde) bir OutlinedTextField göster.
📐 Yapı şöyle:
Column(Modifier.padding(paddingValues)) {
  if (aramaAcik) {
    OutlinedTextField(
      value = aramaMetni,
      onValueChange = { aramaMetni = it },
      label = { Text("Öğrenci ara...") },
      modifier = Modifier.fillMaxWidth().padding(16.dp)
    )
  }
  LazyColumn(...) { ... }
}
İpucu 4: İsme Göre Filtreleme
🔹 Arama metni boş değilse listeyi filtrele:
📐 Yapı:
val aramaSonucu = if (aramaMetni.isBlank()) {
  gosterilecekler // sekme filtresinden gelen liste
} else {
  gosterilecekler.filter {
    "${it.ad} ${it.soyad}".contains(aramaMetni, ignoreCase = true)
  }
}
💡 ignoreCase = true büyük/küçük harf farkını görmezden gelir. "ahmet" yazınca "Ahmet" de bulunur.
İpucu 5: Arama Kapatma
🔹 Arama kapatıldığında metni temizle:
📐 🔍 ikona tekrar tıklayınca:
onClick = {
  aramaAcik = !aramaAcik
  if (!aramaAcik) aramaMetni = "" // kapatırken temizle
}
💡 Ekstra: TextField'a trailingIcon olarak bir ✕ (Close) ikonu ekleyebilirsin: trailingIcon = { IconButton(onClick = { aramaMetni = "" }) { Icon(Icons.Default.Close, "Temizle") } }
İpucu 6: leadingIcon ile Search İkonu
🔹 OutlinedTextField'a sol tarafta bir 🔍 ikonu koy:
📐 leadingIcon = { Icon(Icons.Default.Search, null) } — Bu parametre TextField'ın sol tarafına otomatik ikon koyar.

✅Başarı Kriterleri:

  • 🔍 ikonuna tıklayınca arama çubuğu açılıyor
  • Yazı yazınca liste anlık filtreleniyor
  • Büyük/küçük harf farkı gözetilmiyor
  • Arama kapatılınca metin temizleniyor ve tam liste geri geliyor
  • Arama + sekme filtreleri birlikte çalışıyor
