Material Bileşenler 🎨

Scaffold, TopAppBar, Card, Dialog ve BottomSheet — Profesyonel uygulama arayüzleri oluşturmayı öğren!

💡Bu derste ne öğreneceksin?

  • Bir uygulamanın iskeletini (Scaffold) kurmayı
  • Üst çubuk (TopAppBar), alt menü (BottomBar) ve eylem butonu (FAB) eklemeyi
  • Bilgi kartları (Card) oluşturmayı
  • Kullanıcıya soru soran Dialog penceresi göstermeyi
  • Alttan açılan menü (BottomSheet) yapmayı

🧱 Önce Temeli Anlayalım

Telefonundaki herhangi bir uygulamayı aç. Ne görüyorsun?

Her uygulamada ortak olan 4 şey:

1
Üst çubuk (TopAppBar) — Uygulamanın adı, menü ve arama ikonu burada.
2
İçerik alanı — Kartlar, listeler, yazılar... asıl içerik burada.
3
Alt menü (BottomBar) — "Ana Sayfa", "Favoriler", "Profil" gibi sekmeler.
4
Eylem butonu (FAB) — Sağ alttaki yuvarlak "+" butonu.

İşte Jetpack Compose'da Scaffold bileşeni, bu 4 parçayı bir arada tutan iskelet yapıdır. Bir evin çatısı gibi düşün — önce çatıyı kurarsın, sonra odaları döşersin!

🏠

Gerçek hayatta:
Önce evin temeli + çatısı yapılır, sonra odalar döşenir.

📱

Compose'da:
Önce Scaffold kurulur, sonra içerik eklenir.

🏗️ Scaffold — Uygulamanın İskeleti

Scaffold, İngilizce'de "iskele, çatı" demektir. Bir uygulamanın temel yapı taşıdır.

Scaffold'un parçaları:

Scaffold(
    topBar = { /* Üst çubuk buraya */ },
    bottomBar = { /* Alt menü buraya */ },
    floatingActionButton = { /* + butonu buraya */ }
) { paddingValues ->
    // Ana içerik buraya
}

⚠️paddingValues ne işe yarar?

Scaffold, üst çubuğun ve alt menünün kapladığı alanı hesaplar ve sana paddingValues olarak verir. Bunu içeriğe uygulamazsan, içerik üst çubuğun altında kalır ve görünmez!

📌 TopAppBar — Üst Çubuk

Uygulamanın en üstündeki çubuktur. Başlık, menü ikonu ve eylem butonlarını barındırır.

TopAppBar'ın 3 parçası:

1
navigationIcon — Sol taraftaki ikon (genelde ☰ menü ikonu)
2
title — Ortadaki başlık yazısı
3
actions — Sağ taraftaki ikon butonları (arama, ayarlar vb.)

TopAppBar kodu:

@OptIn(ExperimentalMaterial3Api::class)

TopAppBar(
    title = { Text("Uygulama Başlığı") },

    // Sol taraftaki ikon
    navigationIcon = {
        IconButton(onClick = { }) {
            Icon(Icons.Default.Menu, "Menü")
        }
    },

    // Sağ taraftaki ikonlar
    actions = {
        IconButton(onClick = { }) {
            Icon(Icons.Default.Search, "Ara")
        }
        IconButton(onClick = { }) {
            Icon(Icons.Default.MoreVert, "Daha fazla")
        }
    },

    // Arka plan rengi
    colors = TopAppBarDefaults.topAppBarColors(
        containerColor = MaterialTheme.colorScheme.primaryContainer
    )
)
🤔 @OptIn(ExperimentalMaterial3Api::class) ne demek?

Bazı Material 3 bileşenleri hâlâ deneysel (experimental) aşamadadır. Google bu bileşenleri gelecekte değiştirebilir. Bu yüzden @OptIn yazarak "Bunun deneysel olduğunu biliyorum, yine de kullanmak istiyorum" demiş oluruz.

🔽 BottomBar — Alt Navigasyon Menüsü

Instagram, WhatsApp gibi uygulamaların altındaki sekme çubuğudur.

NavigationBar {
    NavigationBarItem(
        selected = true,          // Bu sekme aktif mi?
        onClick = { },              // Tıklanınca ne olsun?
        icon = { Icon(Icons.Default.Home, null) },
        label = { Text("Ana Sayfa") }
    )
    NavigationBarItem(
        selected = false,
        onClick = { },
        icon = { Icon(Icons.Default.Favorite, null) },
        label = { Text("Favoriler") }
    )
    NavigationBarItem(
        selected = false,
        onClick = { },
        icon = { Icon(Icons.Default.Person, null) },
        label = { Text("Profil") }
    )
}

📌selected = true olan öğe vurgulanır (rengi değişir). Hangi sayfada olduğunu gösterir.

⊕ FloatingActionButton (FAB)

Ekranın sağ alt köşesinde yüzen, uygulamanın en önemli eylemini temsil eden butondur.

FloatingActionButton(onClick = { }) {
    Icon(Icons.Default.Add, "Ekle")
}

Örneğin: Not uygulamasında "Yeni not ekle", mesajlaşmada "Yeni sohbet başlat".

🃏 Card — Bilgi Kartı

Card, bilgileri güzel bir kutu içinde gösteren bileşendir. Gölge efekti ile içerikler ayrışır.

Card kodu:

@Composable
fun OrnekKart(index: Int) {
    Card(
        modifier = Modifier.fillMaxWidth(),
        elevation = CardDefaults.cardElevation(
            defaultElevation = 4.dp
        )
    ) {
        Column(modifier = Modifier.padding(16.dp)) {
            Text(
                "Kart Başlığı $index",
                style = MaterialTheme.typography.titleMedium
            )
            Spacer(modifier = Modifier.height(8.dp))
            Text(
                "Bu bir örnek kart içeriğidir.",
                style = MaterialTheme.typography.bodyMedium
            )
        }
    }
}

💡elevation (yükseklik) = Kartın gölge miktarı. 4.dp hafif gölge verir, 8.dp daha belirgin.

📜 LazyColumn ile Kartları Listeleme

10 tane kartı tek tek yazmak yerine, LazyColumn ile otomatik listeliyoruz:

LazyColumn(
    modifier = Modifier.padding(paddingValues),
    contentPadding = PaddingValues(16.dp),
    verticalArrangement = Arrangement.spacedBy(12.dp)
) {
    items(10) { index ->
        OrnekKart(index)
    }
}

🔁items(10) → 0'dan 9'a kadar 10 kart oluşturur. Her birinde index değeri farklıdır (0, 1, 2...).

📏spacedBy(12.dp) → Kartlar arasında 12dp boşluk bırakır.

📐contentPadding → Listenin kenarlarından 16dp iç boşluk.

💻 Hepsini Birleştiren Tam Kod

Şimdi tüm parçaları Scaffold içinde birleştirelim:

📄 MaterialBilesenler() — Tam kodu göster 
@OptIn(ExperimentalMaterial3Api::class)
@Composable
fun MaterialBilesenler() {
    Scaffold(
        topBar = {
            TopAppBar(
                title = { Text("Uygulama Başlığı") },
                navigationIcon = {
                    IconButton(onClick = { }) {
                        Icon(Icons.Default.Menu, "Menü")
                    }
                },
                actions = {
                    IconButton(onClick = { }) {
                        Icon(Icons.Default.Search, "Ara")
                    }
                    IconButton(onClick = { }) {
                        Icon(Icons.Default.MoreVert, "Daha fazla")
                    }
                },
                colors = TopAppBarDefaults.topAppBarColors(
                    containerColor = MaterialTheme.colorScheme.primaryContainer
                )
            )
        },
        floatingActionButton = {
            FloatingActionButton(onClick = { }) {
                Icon(Icons.Default.Add, "Ekle")
            }
        },
        bottomBar = {
            NavigationBar {
                NavigationBarItem(
                    selected = true, onClick = { },
                    icon = { Icon(Icons.Default.Home, null) },
                    label = { Text("Ana Sayfa") }
                )
                NavigationBarItem(
                    selected = false, onClick = { },
                    icon = { Icon(Icons.Default.Favorite, null) },
                    label = { Text("Favoriler") }
                )
                NavigationBarItem(
                    selected = false, onClick = { },
                    icon = { Icon(Icons.Default.Person, null) },
                    label = { Text("Profil") }
                )
            }
        }
    ) { paddingValues ->
        LazyColumn(
            modifier = Modifier.padding(paddingValues),
            contentPadding = PaddingValues(16.dp),
            verticalArrangement = Arrangement.spacedBy(12.dp)
        ) {
            items(10) { index ->
                OrnekKart(index)
            }
        }
    }
}

💬 Dialog — Onay Penceresi

Kullanıcıya bir soru sormak veya onay almak istediğinde Dialog kullanırsın. Örneğin: "Silmek istediğinize emin misiniz?"

Dialog nasıl çalışır?

1
Bir state değişkeni oluşturursun: var dialogAcik by remember { mutableStateOf(false) }
2
Butona tıklanınca state'i true yaparsın → Dialog açılır
3
Dialog kapatılınca state'i false yaparsın → Dialog kapanır
☰Uygulamam
Dialog Aç
Silme Onayı

Bu öğeyi silmek istediğinizden emin misiniz?

İptal Sil

Dialog kodu:

@Composable
fun DialogOrnek() {
    // 1) State: Dialog açık mı kapalı mı?
    var dialogAcik by remember { mutableStateOf(false) }

    // 2) Butona tıkla → dialogAcik = true
    Button(onClick = { dialogAcik = true }) {
        Text("Dialog Aç")
    }

    // 3) Eğer dialogAcik true ise, Dialog'u göster
    if (dialogAcik) {
        AlertDialog(
            onDismissRequest = { dialogAcik = false },
            title = { Text("Silme Onayı") },
            text = { Text("Bu öğeyi silmek istediğinizden emin misiniz?") },
            confirmButton = {
                TextButton(onClick = { dialogAcik = false }) {
                    Text("Sil", color = Color.Red)
                }
            },
            dismissButton = {
                TextButton(onClick = { dialogAcik = false }) {
                    Text("İptal")
                }
            }
        )
    }
}

✅Önemli kavramlar:

  • onDismissRequest → Kullanıcı dışarı tıklayınca veya geri tuşuna basınca çalışır
  • confirmButton → "Evet/Sil" gibi onay butonu
  • dismissButton → "İptal/Hayır" gibi vazgeçme butonu

📋 BottomSheet — Alttan Açılan Menü

BottomSheet, ekranın altından yukarı kayan bir paneldir. Paylaşım seçenekleri, filtre menüsü gibi durumlarda kullanılır.

☰Uygulamam
Bottom Sheet Aç
Paylaş
📧 E-posta
💬 Mesaj
📋 Kopyala

BottomSheet kodu:

@OptIn(ExperimentalMaterial3Api::class)
@Composable
fun BottomSheetOrnek() {
    var sheetAcik by remember { mutableStateOf(false) }
    val sheetState = rememberModalBottomSheetState()

    Button(onClick = { sheetAcik = true }) {
        Text("Bottom Sheet Aç")
    }

    if (sheetAcik) {
        ModalBottomSheet(
            onDismissRequest = { sheetAcik = false },
            sheetState = sheetState
        ) {
            Column(modifier = Modifier.padding(16.dp)) {
                Text("Paylaş",
                    style = MaterialTheme.typography.headlineSmall)
                Spacer(modifier = Modifier.height(16.dp))

                listOf("📧 E-posta", "💬 Mesaj", "📋 Kopyala").forEach {
                    Text(
                        text = it,
                        modifier = Modifier
                            .fillMaxWidth()
                            .clickable { sheetAcik = false }
                            .padding(12.dp)
                    )
                }

                Spacer(modifier = Modifier.height(32.dp))
            }
        }
    }
}

🔑Dialog vs BottomSheet farkı:

💬

Dialog
Ekranın ortasında açılır.
Evet/Hayır soruları için.

📋

BottomSheet
Alttan yukarı kayar.
Seçenek listesi için.
