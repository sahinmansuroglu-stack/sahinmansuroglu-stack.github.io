State Nedir?

State, zamanla değişebilen ve UI'ı etkileyen herhangi bir değerdir. Compose'da state değiştiğinde UI otomatik olarak güncellenir (recomposition).

State Olmadan (Çalışmaz!)
@Composable
fun SayacYanlis() {
    // ❌ Bu çalışmaz! Her recomposition'da sıfırlanır
    var sayac = 0
    
    Button(onClick = { sayac++ }) {
        Text("Sayaç: $sayac")
    }
}

remember ve mutableStateOf

Doğru State Kullanımı
@Composable
fun SayacDogru() {
    // ✅ remember: Recomposition'lar arasında değeri saklar
    // mutableStateOf: Değer değişince recomposition tetikler
    var sayac by remember { mutableStateOf(0) }
    
    Column(horizontalAlignment = Alignment.CenterHorizontally) {
        Text(
            text = "Sayaç: $sayac",
            fontSize = 32.sp,
            fontWeight = FontWeight.Bold
        )
        
        Spacer(modifier = Modifier.height(16.dp))
        
        Row(horizontalArrangement = Arrangement.spacedBy(8.dp)) {
            Button(onClick = { sayac-- }) {
                Text("-")
            }
            Button(onClick = { sayac++ }) {
                Text("+")
            }
            Button(onClick = { sayac = 0 }) {
                Text("Sıfırla")
            }
        }
    }
}

// Alternatif syntax
@Composable
fun SayacAlternatif() {
    // .value ile erişim
    val sayac = remember { mutableStateOf(0) }
    
    Button(onClick = { sayac.value++ }) {
        Text("Sayaç: ${sayac.value}")
    }
}

Farklı State Türleri

State Türleri
@Composable
fun FarkliStateTurleri() {
    // String state
    var isim by remember { mutableStateOf("") }
    
    // Boolean state
    var acikMi by remember { mutableStateOf(false) }
    
    // Int state (optimize edilmiş)
    var sayac by remember { mutableIntStateOf(0) }
    
    // Float state (optimize edilmiş)
    var deger by remember { mutableFloatStateOf(0.5f) }
    
    // Liste state
    var liste by remember { mutableStateOf(listOf("A", "B", "C")) }
    
    // Değiştirilebilir liste
    val mutableListe = remember { mutableStateListOf("A", "B") }
    
    Column {
        OutlinedTextField(
            value = isim,
            onValueChange = { isim = it },
            label = { Text("İsim") }
        )
        
        Switch(
            checked = acikMi,
            onCheckedChange = { acikMi = it }
        )
        
        Slider(
            value = deger,
            onValueChange = { deger = it }
        )
        
        // Liste işlemleri
        Button(onClick = { 
            mutableListe.add("Yeni")
        }) {
            Text("Ekle (${mutableListe.size})")
        }
    }
}

State Hoisting

State hoisting, state'i üst bileşene taşıyarak bileşenleri stateless yapma tekniğidir.

State Hoisting Örneği
// ❌ Stateful composable (state içeride)
@Composable
fun StatefulSayac() {
    var count by remember { mutableStateOf(0) }
    
    Button(onClick = { count++ }) {
        Text("Count: $count")
    }
}

// ✅ Stateless composable (state dışarıda)
@Composable
fun StatelessSayac(
    count: Int,
    onIncrement: () -> Unit
) {
    Button(onClick = onIncrement) {
        Text("Count: $count")
    }
}

// Kullanım
@Composable
fun SayacKullanimi() {
    var count by remember { mutableStateOf(0) }
    
    Column {
        // Aynı state'i paylaşan iki bileşen
        StatelessSayac(count = count, onIncrement = { count++ })
        Text("Mevcut değer: $count")
        Button(onClick = { count = 0 }) {
            Text("Sıfırla")
        }
    }
}
💡 State Hoisting Avantajları:
• Bileşenler test edilebilir
• Bileşenler yeniden kullanılabilir
• State tek bir yerden yönetilir
• Preview'da farklı state'ler gösterilebilir

rememberSaveable

Konfigürasyon Değişikliklerinde State
@Composable
fun SaveableStateOrnek() {
    // remember: Sadece recomposition'da saklar
    // Ekran döndürme gibi konfigürasyon değişikliklerinde kaybolur
    var normalState by remember { mutableStateOf("") }
    
    // rememberSaveable: Konfigürasyon değişikliklerinde de saklar
    // Bundle'a kaydedilebilen tipler için otomatik çalışır
    var saveableState by rememberSaveable { mutableStateOf("") }
    
    Column {
        OutlinedTextField(
            value = normalState,
            onValueChange = { normalState = it },
            label = { Text("Normal (kaybolur)") }
        )
        
        OutlinedTextField(
            value = saveableState,
            onValueChange = { saveableState = it },
            label = { Text("Saveable (kalır)") }
        )
        
        Text("Ekranı döndürüp test edin!")
    }
}

Pratik Örnek: Todo Uygulaması

Basit Todo List
@Composable
fun TodoApp() {
    var yeniGorev by remember { mutableStateOf("") }
    val gorevler = remember { mutableStateListOf<String>() }
    
    Column(modifier = Modifier.padding(16.dp)) {
        // Giriş alanı
        Row(
            modifier = Modifier.fillMaxWidth(),
            verticalAlignment = Alignment.CenterVertically
        ) {
            OutlinedTextField(
                value = yeniGorev,
                onValueChange = { yeniGorev = it },
                label = { Text("Yeni görev") },
                modifier = Modifier.weight(1f)
            )
            
            Spacer(modifier = Modifier.width(8.dp))
            
            Button(
                onClick = {
                    if (yeniGorev.isNotBlank()) {
                        gorevler.add(yeniGorev)
                        yeniGorev = ""
                    }
                }
            ) {
                Icon(Icons.Default.Add, null)
            }
        }
        
        Spacer(modifier = Modifier.height(16.dp))
        
        // Görev listesi
        gorevler.forEachIndexed { index, gorev ->
            Card(
                modifier = Modifier
                    .fillMaxWidth()
                    .padding(vertical = 4.dp)
            ) {
                Row(
                    modifier = Modifier
                        .fillMaxWidth()
                        .padding(12.dp),
                    verticalAlignment = Alignment.CenterVertically
                ) {
                    Text(gorev, modifier = Modifier.weight(1f))
                    
                    IconButton(onClick = { gorevler.removeAt(index) }) {
                        Icon(
                            Icons.Default.Delete,
                            contentDescription = "Sil",
                            tint = Color.Red
                        )
                    }
                }
            }
        }
        
        if (gorevler.isEmpty()) {
            Text(
                "Henüz görev yok",
                color = Color.Gray,
                modifier = Modifier.padding(16.dp)
            )
        }
    }
}
