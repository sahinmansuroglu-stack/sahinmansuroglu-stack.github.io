Başlangıç Senaryosu (Statik Arayüz)

Bu kodda butonlara bassanız da ekranda hiçbir şey değişmez. Çünkü Compose'a "ekranı güncelle" (Recomposition) demedik.

@Composable
fun SayacEkrani() {
    var sayi = 0 // Bu normal bir değişkendir, Compose bunu takip etmez!

    Column(
        modifier = Modifier.fillMaxSize(),
        verticalArrangement = Arrangement.Center,
        horizontalAlignment = Alignment.CenterHorizontally
    ) {
        // Bu yazı 'sayi' artsa bile hep "0" kalacaktır
        Text(text = "Sayaç: $sayi", fontSize = 30.sp)
        
        Button(onClick = { 
            sayi++ // Değişken artar ama ekran yenilenmez!
            println("Sayı şu an: $sayi") 
        }) {
            Text("Artır")
        }
    }
}

Eğitim Adımları: State Tanımlama

1. remember ve mutableStateOf Kullanımı

Compose'un bir değişkeni izlemesi için onu mutableStateOf içine almalıyız. remember ise ekran her yenilendiğinde değerin sıfırlanmamasını sağlar.

// KODUNU ŞUNA DÖNÜŞTÜR:
var sayi by remember { mutableStateOf(0) } // 'by' kullanmak için importlara dikkat!

2. TextField (Yazı Girişi) ile State

Kullanıcının girdiği metni anlık olarak saklamak için de State kullanılır.

var isim by remember { mutableStateOf("") }

TextField(
    value = isim,
    onValueChange = { yeniYazi -> isim = yeniYazi },
    label = { Text("Adınızı Yazın") }
)
Text("Hoş geldin, $isim!")

3. Liste (List) State Yönetimi

Eğer bir listeye eleman ekleyip çıkaracaksanız mutableStateListOf kullanmalısınız.

val gorevListesi = remember { mutableStateListOf() }

Button(onClick = { gorevListesi.add("Yeni Görev") }) {
    Text("Ekle")
}

LazyColumn {
    items(gorevListesi) { gorev ->
        Text(gorev)
    }
}

Sıra Sende: Dinamik Uygulama

Aşağıdaki kodları kullanarak projenize yeni özellikler ekleyin.

GÖREV 1: Dinamik Başlık Değiştirme

Bir butona basıldığında üstteki başlık metnini ve rengini değiştiren bir yapı kurun.

// Değişkenleri tanımla:
var baslik by remember { mutableStateOf("Merhaba") }
var baslikRengi by remember { mutableStateOf(Color.Black) }

// UI içine ekle:
Text(text = baslik, color = baslikRengi)
Button(onClick = { 
    baslik = "Kayıt Başarılı!"
    baslikRengi = Color.Blue
}) {
    Text("Onayla")
}

GÖREV 2: Listeden Eleman Silme

Daha önce öğrendiğin LazyColumn içindeki her elemanın yanına bir "Sil" butonu ekle.

// Liste döngüsü içinde:
Row(verticalAlignment = Alignment.CenterVertically) {
    Text(text = item, modifier = Modifier.weight(1f))
    IconButton(onClick = { gorevListesi.remove(item) }) {
        Icon(Icons.Default.Delete, contentDescription = "Sil")
    }
}

GÖREV 3: Sayı Tahmin Kontrolü

Kullanıcının girdiği sayıya göre "Yukarı" veya "Aşağı" diyen bir sistem yapın.

var tahmin by remember { mutableStateOf("") }
var sonucMesaji by remember { mutableStateOf("Tahmin Et!") }
val hedefSayi = 42

TextField(value = tahmin, onValueChange = { tahmin = it })
Button(onClick = {
    val t = tahmin.toIntOrNull() ?: 0
    sonucMesaji = if (t < hedefSayi) "YUKARI!" else if (t > hedefSayi) "AŞAĞI!" else "BİLDİN!"
}) {
    Text("Kontrol Et")
}
Text(sonucMesaji)
