📱 Bottom Navigation

Alt Menü ile Geçiş Mekanizması

💬 Açıklama

Bottom Navigation (Alt Menü), Instagram, Twitter veya WhatsApp gibi milyonlarca uygulamanın ekranının en altında yer alan ana sekme çubuğudur.

Bu sistemi uygulamamıza dahil edebilmek için ilk adımda bu düğmelerin ne olacağını tasarlamamız gerekir. Tüm düğmelerin ortak özellikleri şunlardır:

  • Gidilecek Sayfa: Menülere tıklanınca nerenin açılacağı (Route)
  • Sekme Yazısı: Altında ne yazdığı ("Ana Sayfa", "Arama")
  • İkon: Görsel sembolü (Ev işareti, Büyüteç işareti vb.)

🛠️ Alt Menü Seçeneklerini Tanımlama

Bir sealed class kullanarak alt menüde olacak her düğmenin bütün sabit ve önemli özelliklerini bir arada, kalıp halinde belirleriz:

// Alt menüdeki her sekmenin taslağını yazıyoruz
sealed class AltMenu(
    val route: String,     // Yönlendirilecek adres
    val baslik: String,    // Buton altında görünen İsim
    val ikon: ImageVector  // Simge
) {
    // İlk Sekme: Anasayfa
    object Ev : AltMenu("anasayfa", "Ana Sayfa", Icons.Default.Home)
    
    // İkinci Sekme: Arama
    object Ara : AltMenu("arama", "Keşfet", Icons.Default.Search)
    
    // Üçüncü Sekme: Profil
    object Profil : AltMenu("profil", "Ben", Icons.Default.Person)
}Kotlin

Bu taslağı oluşturduktan sonra diğer adıma yani bu taslağı sayfaya gömmeye geçebiliriz (Bunun için Scaffold adımına geçeceğiz).

🎯 Senin Sıran: Koda Özellik Ekle!

Sealed class yapısını kullanarak menümüzü genişletelim:

  • Özellik 1: Ayarlar Sekmesi Ekle
    Uygulamamızın alt menüsüne dördüncü bir buton olarak "Ayarlar"ı ekle.
    💡 İpucu: AltMenu sealed class'ı içine object Ayarlar : AltMenu("ayarlar", "Ayarlar", Icons.Default.Settings) bloğunu eklemelisin.

  • Özellik 2: Bildirimler Sekmesi Ekle
    Menümüze çan simgesi ile gösterilen "Bildirimler" adresini oluştur.
    💡 İpucu: Yeni object ismini Bildirimler yapıp ikonu Icons.Default.Notifications olarak tanımla.
