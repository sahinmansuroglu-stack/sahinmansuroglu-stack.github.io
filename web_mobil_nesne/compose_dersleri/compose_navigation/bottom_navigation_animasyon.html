Bottom Navigation & Animation

Sayfa Geçişlerinde Göz Alıcı Animasyonlar Eklemek

ADIM 0

Başlangıç Senaryosu (Sert Geçişli Menü)

Bu kodda Bottom Navigation çalışıyor ancak sayfalar arası geçiş çok "sert" ve anidir. Animasyon yoktur.

// Alt Menü Öğeleri Seçenekleri
sealed class Sayfalar(val rota: String, val isim: String) {
    object Anasayfa : Sayfalar("home", "Ana Sayfa")
    object Profil : Sayfalar("profile", "Profil")
}

@Composable
fun MainScreen() {
    val navController = rememberNavController()
    Scaffold(
        bottomBar = {
            BottomNavigation(backgroundColor = Color.White) {
                val backStackEntry by navController.currentBackStackEntryAsState()
                val currentRoute = backStackEntry?.destination?.route

                BottomNavigationItem(
                    selected = currentRoute == Sayfalar.Anasayfa.rota,
                    onClick = { navController.navigate(Sayfalar.Anasayfa.rota) },
                    label = { Text(Sayfalar.Anasayfa.isim) },
                    icon = { Icon(Icons.Default.Home, "") }
                )
                BottomNavigationItem(
                    selected = currentRoute == Sayfalar.Profil.rota,
                    onClick = { navController.navigate(Sayfalar.Profil.rota) },
                    label = { Text(Sayfalar.Profil.isim) },
                    icon = { Icon(Icons.Default.Person, "") }
                )
            }
        }
    ) { padding ->
        NavHost(navController, startDestination = "home", Modifier.padding(padding)) {
            composable("home") { Text("Ana Sayfa İçeriği", fontSize = 30.sp) }
            composable("profile") { Text("Profil Sayfası İçeriği", fontSize = 30.sp) }
        }
    }
}

Eğitim Adımları: Animasyon Ekleme

1. Composable İçinde Giriş/Çıkış Belirleme

Yeni Compose versiyonlarında composable fonksiyonu içine enterTransition ve exitTransition ekleyebiliriz.

composable(
    route = "home",
    enterTransition = { fadeIn(animationSpec = tween(700)) }, // Yavaşça belirsin
    exitTransition = { fadeOut(animationSpec = tween(700)) }  // Yavaşça kaybolsun
) { 
    Anasayfa() 
}

2. Kayma (Slide) Efekti Vermek

Sayfanın yan taraftan kayarak gelmesini istiyorsak slideInHorizontally kullanırız.

enterTransition = {
    slideInHorizontally(initialOffsetX = { it }) + fadeIn()
},
exitTransition = {
    slideOutHorizontally(targetOffsetX = { -it }) + fadeOut()
}
✨

Sıra Sende: Profesyonel Geçişler

Aşağıdaki kodları NavHost içindeki composable bloklarına uygulayarak farklı animasyon türlerini dene.

GÖREV 1: Aşağıdan Yukarı Kayma (Pop-up Etkisi)

Detay sayfasının veya profilin bir pencere gibi aşağıdan yukarı açılmasını sağla.

// Bunu "profile" composable'ına ekle:
enterTransition = {
    slideInVertically(initialOffsetY = { it }) + fadeIn()
},
exitTransition = {
    slideOutVertically(targetOffsetY = { it }) + fadeOut()
}

GÖREV 2: Ölçeklenme (Zoom) Animasyonu

Sayfanın küçükten büyüğe gelerek açılmasını (Büyüme efekti) sağlar.

enterTransition = {
    scaleIn(initialScale = 0.8f) + fadeIn()
},
exitTransition = {
    scaleOut(targetScale = 0.8f) + fadeOut()
}

GÖREV 3: Yaylanma (Spring) Efekti

Animasyonun daha doğal ve "canlı" görünmesi için sert hareketler yerine yaylanma (spring) ekle.

// Tween yerine spring kullanarak daha yumuşak geçiş sağla
enterTransition = {
    slideInHorizontally(
        animationSpec = spring(dampingRatio = Spring.DampingRatioMediumBouncy)
    )
}
