En Basit Haliyle: Parametre Gönderme

Sadece 3 Adımda Veri Transferini Öğren

0. Başlangıç (Sadece Butonlar)

Kodun en saf hali budur. Bir butona basınca diğer sayfaya geçer ama hiçbir bilgi götürmez.

Önce bu iskeleti oluşturun. 
@Composable
fun BasitUygulama() {
    val navController = rememberNavController()
    NavHost(navController = navController, startDestination = "giris") {
        composable("giris") { 
            GirisEkrani(tiklandi = { navController.navigate("selam") }) 
        }
        composable("selam") { 
            SelamEkrani() 
        }
    }
}

@Composable
fun GirisEkrani(tiklandi: () -> Unit) {
    Button(onClick = tiklandi) { Text("Selam Gönder") }
}

@Composable
fun SelamEkrani() {
    Text("Merhaba!")
}

Görevin: "Merhaba" Yerine "Merhaba [İsim]" Yazdır

YOLU TANIMLA

1. Rota Adresini Güncelle

Nereye: NavHost içindeki "selam" yazan yere.

// "selam" olan yeri şu yap:
"selam/{isim}"
VERİYİ YOLLA

2. Butona Değer Ekle

Nereye: GirisEkrani'ndaki navigate kısmına.

// Veriyi adrese ekleyerek gönder:
navController.navigate("selam/Ahmet")
VERİYİ YAKALA

3. Veriyi Oku ve Ekrana Bas

Nereye: NavHost içindeki composable("selam/...") bloğuna.

composable("selam/{isim}") { backStackEntry ->
    // 1. Kutuyu aç ve içinden "isim" olanı al
    val gelenIsim = backStackEntry.arguments?.getString("isim") ?: "Yabancı"
    
    // 2. Bu ismi ekrana gönder
    SelamEkrani(isim = gelenIsim)
}

// SelamEkrani'nı da güncellemeyi unutma:
@Composable
fun SelamEkrani(isim: String) {
    Text("Merhaba $isim!")
}

Görev 2: Arka Plan Rengi Değiştir

Selam sayfasına bir Surface ekle ve rengini gelen isme göre değiştir.

// SelamEkrani içine kopyala:
val renk = if(isim == "Ahmet") Color.Blue else Color.Red
Surface(color = renk, modifier = Modifier.fillMaxSize()) {
    Text("Merhaba $isim", color = Color.White)
}
