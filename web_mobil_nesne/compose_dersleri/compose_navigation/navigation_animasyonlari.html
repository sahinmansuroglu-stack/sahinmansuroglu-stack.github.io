Kayarak ve Yumuşak Geçişler Ekleme

💬 Açıklama

Zaten bildiğiniz bir şey: Telefonunuzda bir fotoğrafa tıkladığınızda şıp diye açılmaz, kenardan kayarak ekrana girer. Geri tuşuna bastığınızda da geldiği yöne doğru kayarak çıkar.

Uygulamamızı daha pürüzsüz ve profesyonel yapmak için Sayfa Geçiş Animasyonlarını composable() fonksiyonumuzun özellikleri ile birlikte yazarız.

🛠️ Animasyon Türleri

  • enterTransition: Sayfaya ileri doğru ilk giriş anı animasyonu.
  • exitTransition: Üzerine yeni sayfa açıldığında altta kalan eski sayfanın çıkış animasyonu.
  • popEnterTransition: Geri tuşuyla eski sayfaya dönerken, eski sayfanın tekrar ekrana giriş animasyonu.
  • popExitTransition: Geri tuşuna bastığınızda, kapanan sayfanın ekrandan çıkış animasyonu.

📝 Kod Örneği

Örneğin bir detay sayfasının sağ taraftan yavaşça kayarak gelmesini ve kapanırken aynı yolla çıkmasını sağlayalım:

composable(
    route = "detay_sayfam",
    
    // 1. Tıklandı, detay sayfasına GİRİYORUZ: Yeni sayfa sağdan (1000 noktasından) kayarak gelsin
    enterTransition = {
        slideInHorizontally(
            initialOffsetX = { 1000 }, 
            animationSpec = tween(300) // 300 ms sürsün
        )
    },
    
    // 2. Geri tuşuna basıldı, detay sayfasından ÇIKIYORUZ: Sağa doğru kayıp ekrandan çıksın
    popExitTransition = {
        slideOutHorizontally(
            targetOffsetX = { 1000 },
            animationSpec = tween(300)
        )
    }
) { 
    DetaySayfam()
}Kotlin

🎯 Senin Sıran: Koda Özellik Ekle!

Aşağıdaki görevlerle sayfalarının sahneye giriş çıkışını daha havalı hale getir:

  • Özellik 1: Yukarıdan Aşağıya Kayma Efekti
    Yandan değil de, sayfamızın yukarıdan ekran çerçevesine düşmesini (slayt) sağlayalım.
    💡 İpucu: slideInHorizontally (Yatay) yerine slideInVertically (Dikey) metodunu çağırıp initialOffsetY değerini ayarla.

  • Özellik 2: Kararma / Belirme (Fade) Efekti
    Yeni sayfa sağdan sola kayarken, eski sayfanın arkada yavaşça kararıp kaybolmasını (Fade Out) sağlayabilir misin?
    💡 İpucu: Animasyonları birleştirebiliriz! Eski sayfa kapanırken exitTransition = { fadeOut(animationSpec = tween(300)) } ayarı kullanırsan o arkadan yavaşça kaybolur!
