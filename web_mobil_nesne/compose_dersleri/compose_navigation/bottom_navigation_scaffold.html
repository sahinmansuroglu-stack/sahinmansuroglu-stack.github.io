📐 Bottom Navigation + Scaffold

Menüyü Ekranın Altına Sabitleyelim

💬 Açıklama

Bir önceki konuda özelliklerini tanımladığımız "Alt Menü"yü ekranda doğru yere yani en alta sabitlemek için Scaffold kullanırız.

Scaffold, uygulamamız için ana şablondur. Scaffold'un bize sunduğu bottomBar alanına alt menümüzü, gövdesine (content) sayfalarımızın haritası olan NavHost'u yerleştiririz.

🛠️ Nasıl Kurulur?

@Composable
fun AnaUygulamaEkrani() {
    val sofor = rememberNavController()
    // Oluşturduğumuz sekmeleri bir listeye koyduk.
    val sekmeler = listOf(AltMenu.Ev, AltMenu.Ara, AltMenu.Profil)
    
    Scaffold(
        bottomBar = {
            // Alt çubuğu (NavigationBar) burada oluştururuz
            NavigationBar {
                // Şu an hangi sayfada olduğumuzu anla
                val aktifSayfaGecmisi by sofor.currentBackStackEntryAsState()
                val guncelAdres = aktifSayfaGecmisi?.destination?.route
                
                // Her sekme için döngü oluşturarak çubuğa ekliyoruz
                sekmeler.forEach { sekme ->
                    NavigationBarItem(
                        icon = { Icon(sekme.ikon, sekme.baslik) },
                        label = { Text(sekme.baslik) },
                        selected = guncelAdres == sekme.route, // Seçiliyse parlasın
                        onClick = {
                            sofor.navigate(sekme.route) {
                                // Bu ayarlar, geri tuşuna basınca uygulamanın mantıklı döngüye girmesi içindir
                                popUpTo(sofor.graph.findStartDestination().id) { saveState = true }
                                launchSingleTop = true // Üst üste iki sayfa açılmasını önler
                                restoreState = true    // Sekmeler arası geçişte durumu korur
                            }
                        }
                    )
                }
            }
        }
    ) { icBosluk ->
        // Gövdemize tüm rotaların haritasını yerleştiririz
        NavHost(
            navController = sofor,
            startDestination = AltMenu.Ev.route,
            modifier = Modifier.padding(icBosluk) // Scaffold'un sağladığı boşluğa saygı göster
        ) {
            composable(AltMenu.Ev.route) { AnaSayfam() }
            composable(AltMenu.Ara.route) { AramaSayfam() }
            composable(AltMenu.Profil.route) { ProfilSayfam() }
        }
    }
}Kotlin

🎯 Senin Sıran: Koda Özellik Ekle!

Scaffold şablonumuza yeni sayfalar ekleyelim:

  • Özellik 1: Yeni Eklenen Ayarlar Sekmesini Haritaya (NavHost) Dahil Et
    Eğer bir önceki konuda Ayarlar sekmesi eklediyseniz, buna tıklandığında boş bir "Ayarlar Sayfası" açılsın.
    💡 İpucu: sekmeler listene `AltMenu.Ayarlar` dahil edip, NavHost içerisine composable(AltMenu.Ayarlar.route) { AyarlarSayfam() } kodunu eklemelisin.

  • Özellik 2: Profil İçerisinden Ana Sayfaya Kısayol
    Bazen alt menü haricinde, sayfanın ortasındaki bir butona tıklayarak da sekmeler arasında gezebiliriz. Profil sayfasına bir buton koy ve oradan Ana Sayfa'ya atlat.
    💡 İpucu: Profil fonksiyonuna parametre olarak lambda geçebilir veya sofor.navigate(AltMenu.Ev.route) çağırabilirsin. Scaffold alt barı bunu otomatik algılar!
