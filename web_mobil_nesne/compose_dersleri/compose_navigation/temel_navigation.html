🧭 Temel Navigation

Jetpack Compose'da Sayfalar Arası Geçişin Temelleri

💬 Açıklama

Uygulamamızdaki ekranlar (sayfalar) arasında geçiş yapmak için kullandığımız yapıya Navigation denir. Bu yapıyı evimizdeki odalar arasında dolaşmaya benzetebiliriz.

Temel bir gezinme (navigation) işlemi şu 4 parçadan oluşur:

  • Route (Adres): Her sayfanın kimliği yani adresidir (Örn: "anasayfa").
  • NavController (Şoför): Sayfayı değiştirmeyi veya geri dönmeyi sağlayan komutları verir.
  • NavHost (Harita): Hangi adresin hangi sayfayı açacağını bilen iç haritadır.
  • Sayfalar: Ekranda gördüğümüz @Composable fonksiyonlarıdır.

🛠️ Örnek Kod

Aşağıda Ana Sayfadan Detay Sayfasına giden basit bir geçişi görebilirsiniz.

// 1. Temsili adreslerimizi (Route) belirliyoruz
object Adresler {
    const val ANA_SAYFA = "anaEkran"
    const val DETAY = "detayEkran"
}

@Composable
fun UygulamaGecisleri() {
    // 2. Şoförümüzü yaratıyoruz
    val sofor = rememberNavController()
    
    // 3. Haritamızı oluşturuyoruz (İlk hangi sayfa açılacak?)
    NavHost(
        navController = sofor,
        startDestination = Adresler.ANA_SAYFA
    ) {
        // 4. Sayfalarımızı haritaya ekliyoruz
        composable(Adresler.ANA_SAYFA) {
            AnaEkran(
                detayaGit = { sofor.navigate(Adresler.DETAY) } // Detaya git emri
            )
        }
        
        composable(Adresler.DETAY) {
            DetayEkran(
                geriDon = { sofor.popBackStack() } // Geri dön emri
            )
        }
    }
}Kotlin

🖥️ Sayfaların Tasarımı

Sayfalarımız yukarıdaki haritadan aldıkları görevleri uygulayan butonlara sahiptir.

@Composable
fun AnaEkran(detayaGit: () -> Unit) {
    Column {
        Text("Burası Ana Sayfa")
        Button(onClick = detayaGit) {
            Text("Detaya İlerle")
        }
    }
}

@Composable
fun DetayEkran(geriDon: () -> Unit) {
    Column {
        Text("Burası Detay Sayfası")
        Button(onClick = geriDon) {
            Text("Geri Dön")
        }
    }
}Kotlin

🎯 Senin Sıran: Koda Özellik Ekle!

Yukarıdaki örneği anladıysan sıra sende. Aşağıdaki görevleri sırasıyla projene eklemeye çalış:

  • Özellik 1: Hakkımızda Sayfası Ekle
    Mevcut uygulamaya bir de "Hakkımızda" sayfası ekle. Ana ekrana bir buton daha koyup bu sayfaya yönlendir.
    💡 İpucu: Önce Adresler objesine const val HAKKIMIZDA = "hakkimizda" ekle. Sonra yeni bir composable blok aç.

  • Özellik 2: Detay Sayfasından Ana Sayfaya Dönüş Butonu
    Kullanıcı detay sayfasına geldiğinde, "Geri Dön" butonuna basarsa geldiği yere dönsün.
    💡 İpucu: DetayEkran içine bir Button ekle ve tıklandığında süslü parantez içine sofor.popBackStack() komutunu çağır (Bu komut onu otomatik olarak önceki sayfaya atar).

  • Özellik 3: Tüm Sayfaları Birbirine Bağla
    Hakkımızda sayfasından doğrudan Detay sayfasına geçiş sağlayan bir kısayol oluştur.
    💡 İpucu: Hakkımızda ekranı için yazdığın fonksiyon dışarıdan detayaGit: () -> Unit parametresi almalı ve NavHost içindeyken bu parametreye sofor.navigate(Adresler.DETAY) atanmalı.
