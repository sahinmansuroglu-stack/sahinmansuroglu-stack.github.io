📦 Parametre ile Navigation

Sayfalar Arasında Veri Aktarımı Kılavuzu

💬 Açıklama

Bazen bir sayfadan diğerine geçerken beraberinde bilgi taşımamız gerekir. Örneğin, bir uygulamada listeden "Ahmet" kullanıcısına tıkladığımızda detay sayfasının sadece "Ahmet" ile ilgili bilgileri gösterebilmesi için ID aktarımı yapmamız zorunludur.

Bu işleme Parametre Gönderme diyoruz. İnternet sitesi linklerinde gördüğümüz site.com/urun/15 mantığının aynısıdır.

🛠️ Zorunlu Parametre Örneği

Parametreleri her zaman süslü parantez içine koyarak adlandırırız: "{id}"

// NavHost haritasına ekliyoruz
composable(
    route = "urun/{id}",                  // Adreste {id} adıyla yer açtık
    arguments = listOf(
        navArgument("id") {
            type = NavType.IntType      // Gönderilen bilgi bir Tam Sayı (Int) olmalı
        }
    )
) { gelenVeri ->
    // Sayfa açılırken bilgiyi okuyoruz ve asıl sayfaya gönderiyoruz
    val urunI numarasi = gelenVeri.arguments?.getInt("id") ?: 0
    UrunDetaySayfasi(id = numarasi)
}

// Tıklandığında sayfaya giderken 5 sayısını taşıma:
sofor.navigate("urun/5")Kotlin

⚠️ İsteğe Bağlı (Opsiyonel) Parametreler

Gidilecek sayfaya veri gelmese de sayfanın çökmemesini istediğimiz durumlarda soru işareti "?" kullanırız.

composable(
    route = "arama?kelime={kelime}",     // ?kelime= şeklinde tanımlandı
    arguments = listOf(
        navArgument("kelime") {
            type = NavType.StringType // Metin türünde bekliyoruz
            defaultValue = ""         // Eğer değer gelmezse boş kalacak
        }
    )
) { gelenVeri ->
    val aranan = gelenVeri.arguments?.getString("kelime") ?: ""
    AramaSayfasi(aramaMetni = aranan)
}Kotlin

🎯 Senin Sıran: Koda Özellik Ekle!

Mevcut kod üzerinden aşağıdaki yönlendirmeleri kendin yazmaya çalış:

  • Özellik 1: Profil Sayfasına İsim Gönder
    Profil sayfası oluştur ve bu sayfaya "isim" parametresi gönder. Ana sayfada örneğin "Murat'ın Profiline Git" butonu olsun.
    💡 İpucu: route = "profil/{isim}" tanımla ve argüman tipini NavType.StringType olarak belirle.

  • Özellik 2: Gelen İsmi Ekranda Başlık Olarak Göster
    Bir önceki görevde gönderilen ismi yakalayarak Profil sayfasında ekrana yazdır (Örn: "Murat Kullanıcısının Profili").
    💡 İpucu: Sayfa composable'ı açılırken gelenVeri.arguments?.getString("isim") ile ismi okumalı ve bunu Text() bileşeni içinde kullanmalısın.

  • Özellik 3: İsteğe Bağlı "Yaş" Parametresi Ekle
    Oluşturduğun profil adresine isteğe bağlı opsiyonel (varsa yazılacak, yoksa boş kalacak) bir de yaş verisi ekle.
    💡 İpucu: route = "profil/{isim}?yas={yas}" yapısını kur ve defaultValue = 0 veya "" tanımlamasını unutma.
