TEMPLATE Öğrencilere Verilecek Başlangıç Kodu

Öğrenciler ders başında bu kodu kopyalayacak. Bu kodda navigasyon var ancak veri transferi yok. Hedefimiz bu kodu çalışır hale getirmek! 
object Adresler {
    const val ANA_SAYFA = "anaEkran"
    const val DETAY = "detayEkran"
}

@Preview
@Composable
fun UygulamaGecisleri() {
    val sofor = rememberNavController()

    NavHost(
        navController = sofor,
        startDestination = Adresler.ANA_SAYFA
    ) {
        composable(Adresler.ANA_SAYFA) {
            AnaEkran(
                detayaGit = { sofor.navigate(Adresler.DETAY) }
            )
        }

        composable(Adresler.DETAY) {
            DetayEkran(
                geriDon = { sofor.popBackStack() }
            )
        }
    }
}

@Composable
fun AnaEkran(detayaGit: () -> Unit) {
    Column(modifier = Modifier.padding(16.dp)) {
        Text("Öğrenci Listesi")
        
        LazyColumn(modifier = Modifier.weight(1f)) {
            items(ogrenciListesi) { ogrenci ->
                Column(modifier = Modifier.padding(vertical = 8.dp)) {
                    Text("Ad Soyad: ${ogrenci.adSoyad}")
                    Text("Okul: ${ogrenci.okul}")
                    Text("Puanlar: ${ogrenci.puan1}, ${ogrenci.puan2}")
                    HorizontalDivider()
                }
            }
        }

        Button(onClick = detayaGit) {
            Text("Detaya İlerle")
        }
    }
}

@Composable
fun DetayEkran(geriDon: () -> Unit) {
    Column(modifier = Modifier.padding(16.dp)) {
        Text("Burası Detay Sayfası")
        Button(onClick = geriDon) {
            Text("Geri Dön")
        }
    }
}

data class Ogrenci(val adSoyad: String, val okul: String, val puan1: Int, val puan2: Int)

val ogrenciListesi = listOf(
    Ogrenci("Ahmet Yılmaz", "Atatürk Lisesi", 85, 90),
    Ogrenci("Ayşe Kaya", "Fen Lisesi", 95, 100),
    Ogrenci("Mehmet Demir", "Anadolu Lisesi", 70, 80),
    Ogrenci("Fatma Şahin", "Cumhuriyet Lisesi", 88, 92)
)

Görevi Tamamlama Adımları

1. Rotayı Parametreli Hale Getirin

Nereye: En tepedeki Adresler nesnesine.

Neden: Navigasyonun bir veri beklediğini yolu (path) üzerinden tanımlamamız gerekir.

// "detayEkran" olan kısmı şununla değiştir:
const val DETAY = "detay/{index}"

2. Ana Sayfada Tıklananın Sırasını Gönderin

Nereye: AnaEkran fonksiyonundaki LazyColumn içine.

Neden: Herhangi bir satıra tıklandığında hangi öğrenciye tıklandığını (index) dışarıya bildirmeliyiz.

// 1. Fonksiyon parametresini değiştir:
fun AnaEkran(detayaGit: (index: Int) -> Unit)

// 2. Loop içindeki Text'e tıklanma ekle (items yerine itemsIndexed kullan):
itemsIndexed(ogrenciListesi) { index, ogrenci ->
    Text(
        text = ogrenci.adSoyad, 
        modifier = Modifier
            .clickable { detayaGit(index) } 
            .padding(8.dp)
            .fillMaxWidth()
    )
}

3. NavHost'u Bu Değişikliğe Göre Ayarlayın

Nereye: UygulamaGecisleri içindeki NavHost bloğuna.

Neden: Gelen index'i kutudan çıkarıp (bundle) diğer sayfaya elden teslim edeceğiz.

// 1. AnaEkran çağrısını güncelle:
composable(Adresler.ANA_SAYFA) {
    AnaEkran(detayaGit = { index -> sofor.navigate("detay/$index") })
}

// 2. Detay sayfasını parametreyi okuyacak hale getir:
composable(
    route = Adresler.DETAY,
    arguments = listOf(navArgument("index") { type = NavType.IntType })
) { yigin ->
    val secilenIndex = yigin.arguments?.getInt("index") ?: 0
    DetayEkran(gelenIndex = secilenIndex, geriDon = { sofor.popBackStack() })
}

4. Detay Sayfasında Veriyi Gösterin

Nereye: DetayEkran fonksiyonuna.

Neden: Gelen sayıya göre listedeki gerçek öğrenci bilgilerine ulaşıp ekrana yazdırmak için.

// Fonksiyonu ve içeriğini güncelle:
@Composable
fun DetayEkran(gelenIndex: Int, geriDon: () -> Unit) {
    val ogrenci = ogrenciListesi.getOrElse(gelenIndex) { null }

    Column(modifier = Modifier.padding(16.dp)) {
        if (ogrenci != null) {
            Text("Seçilen: ${ogrenci.adSoyad}", style = MaterialTheme.typography.headlineMedium)
            Text("Okul: ${ogrenci.okul}")
            Text("Not Ortalaması: ${(ogrenci.puan1 + ogrenci.puan2) / 2}")
        }
        Button(onClick = geriDon) { Text("Geri Dön") }
    }
}

İnternet kullanmadan, öğrendiğin bu mantığı pekiştirmek için aşağıdaki mini özellikleri ekle:

2. Koşullu Renk

if(ort < 50) Color.Red else Color.Green

3. Yeni Liste Elemanı
