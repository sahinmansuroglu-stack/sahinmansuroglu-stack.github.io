Navigation Nedir?

🏠 Gerçek Hayat Benzetmesi:
Bir evi düşünün: Salon, mutfak, yatak odası, banyo... Her oda ayrı bir sayfadır. Odalar arasında yürüyerek geçersiniz — bu da navigation'dır. Evin planı (haritası) NavHost'tur. Sizi yönlendiren kişi ise NavController'dır.

Parça 1: Route (Adres) Tanımları

🏷️ Route Nedir?

Her sayfanın bir ismi (adresi) olmalı. Tıpkı bir web sitesindeki URL gibi düşünün:

  • instagram.com/home → Ana Sayfa
  • instagram.com/profile → Profil
  • instagram.com/settings → Ayarlar

Biz de uygulamamızdaki her sayfaya bir string isim vereceğiz:

// Her sayfanın benzersiz bir adresi var
// Bu adresleri bir "object" içinde topluyoruz ki
// yazdığımız zaman hata yapmayalım

object Routes {
    const val HOME = "home"        // Ana Sayfa adresi
    const val DETAIL = "detail"    // Detay Sayfası adresi
    const val PROFILE = "profile"  // Profil adresi
    const val SETTINGS = "settings"// Ayarlar adresi
}Kotlin
💡 Neden object kullanıyoruz? Route isimlerini doğrudan "home" şeklinde yazsak, bir yerde "hme" gibi yazım hatası yapabiliriz. Object içinde tanımlarsak, Routes.HOME yazarken IDE bizi yönlendirir ve hata yapmamızı engeller.

Parça 2: NavController — Şoför

🚗 NavController Nedir?

NavController, uygulamanızdaki navigasyon şoförüdür. Sayfalar arası geçişi o yönetir.

// NavController'ı oluşturuyoruz
// "remember" sayesinde ekran döndüğünde bile kaybolmaz
val navController = rememberNavController()Kotlin

🎮 NavController Ne Yapabilir?

navController.navigate("detail") → "detail" sayfasına git
navController.popBackStack() → Geri dön (önceki sayfaya)
🎯 Benzetme: NavController bir taksi şoförü gibidir. Ona "şu adrese git" dersiniz (navigate), ya da "geri dön" dersiniz (popBackStack).

Parça 3: NavHost — Harita

🗺️ NavHost Nedir?

NavHost, tüm sayfalarınızın kayıtlı olduğu haritadır. Hangi adrese gidildiğinde hangi sayfanın gösterileceğini belirler.

NavHost(
    navController = navController,         // Şoförümüz
    startDestination = Routes.HOME     // Uygulama açıldığında ilk sayfa
) {
    // Buraya sayfaları ekliyoruz ↓
    
    composable(Routes.HOME) {
        // Route "home" olunca bu sayfa gösterilir
        AnaSayfa(...)
    }
    
    composable(Routes.DETAIL) {
        // Route "detail" olunca bu sayfa gösterilir
        DetaySayfasi(...)
    }
}Kotlin
NavHost (Harita)
"home" → AnaSayfa

"detail" → DetaySayfasi

"profile" → ProfilSayfasi

"settings" → AyarlarSayfasi
⚠️ Dikkat: startDestination parametresi zorunludur. Uygulama ilk açıldığında hangi sayfanın gösterileceğini belirler.

Parça 4: Sayfa Composable'ları

📄 Sayfaları Tasarlıyoruz

Şimdi haritamıza ekleyeceğimiz sayfaları oluşturalım. Her sayfa normal bir @Composable fonksiyondur.

Ana Sayfa

İki buton içerir: biri Detay sayfasına, diğeri Profil sayfasına gider.

@Composable
fun AnaSayfa(
    onDetayaGit: () -> Unit,   // Detay butonuna basınca çalışacak
    onProfil: () -> Unit       // Profil butonuna basınca çalışacak
) {
    Column(
        modifier = Modifier
            .fillMaxSize()
            .padding(16.dp),
        horizontalAlignment = Alignment.CenterHorizontally,
        verticalArrangement = Arrangement.Center
    ) {
        Text("Ana Sayfa", style = MaterialTheme.typography.headlineLarge)
        
        Spacer(modifier = Modifier.height(16.dp))
        
        Button(onClick = onDetayaGit) {
            Text("Detay Sayfasına Git")
        }
        
        Button(onClick = onProfil) {
            Text("Profil Sayfasına Git")
        }
    }
}Kotlin
💡 Lambda Parametreleri: onDetayaGit: () -> Unit bir fonksiyon parametresidir. Sayfa, hangi sayfaya gidileceğini bilmez — sadece "bana verilen fonksiyonu çalıştır" der. Navigation mantığı dışarıda (NavHost içinde) kalır. Bu sayede sayfalar birbirinden bağımsız olur!

Detay Sayfası

Geri dönme butonu ile önceki sayfaya dönmeyi sağlar.

@Composable
fun DetaySayfasi(onGeriDon: () -> Unit) {
    Column(
        modifier = Modifier.fillMaxSize().padding(16.dp)
    ) {
        IconButton(onClick = onGeriDon) {
            Icon(Icons.AutoMirrored.Filled.ArrowBack, "Geri")
        }
        Text("Detay Sayfası", style = MaterialTheme.typography.headlineLarge)
    }
}Kotlin

// 1️⃣ Route Tanımları
object Routes {
    const val HOME = "home"
    const val DETAIL = "detail"
    const val PROFILE = "profile"
    const val SETTINGS = "settings"
}

// 2️⃣ Ana Navigation Fonksiyonu
@Composable
fun AppNavigation() {
    // 🚗 NavController oluştur
    val navController = rememberNavController()
    
    // 🗺️ NavHost — Haritamız
    NavHost(
        navController = navController,
        startDestination = Routes.HOME  // İlk açılan sayfa
    ) {
        // 📄 Sayfa 1: Ana Sayfa
        composable(Routes.HOME) {
            AnaSayfa(
                onDetayaGit = { navController.navigate(Routes.DETAIL) },
                onProfil = { navController.navigate(Routes.PROFILE) }
            )
        }
        
        // 📄 Sayfa 2: Detay
        composable(Routes.DETAIL) {
            DetaySayfasi(
                onGeriDon = { navController.popBackStack() }
            )
        }
        
        // 📄 Sayfa 3: Profil
        composable(Routes.PROFILE) {
            ProfilSayfasi(navController)
        }
        
        // 📄 Sayfa 4: Ayarlar
        composable(Routes.SETTINGS) {
            AyarlarSayfasi(navController)
        }
    }
}Kotlin — Tam Kod

Parametre ile Veri Aktarımı

📦 Sayfaya Veri Göndermek

Bir ürüne tıklayınca detay sayfasına ürün numarasını göndermemiz gerekir. Tıpkı bir web sitesinde urun/5 yazması gibi.

🎯 Benzetme: Bir kargocuya "şu paketi 5 numaralı adrese götür" dersiniz. Burada "5" parametredir. Route'a parametre eklemek de aynı mantıktır: "urun/5"

Zorunlu Parametre Gerekli

Süslü parantez {urunId} ile tanımlanır:

// 1️⃣ Route'u parametre ile tanımla
composable(
    route = "urun/{urunId}",               // {urunId} → parametre yeri
    arguments = listOf(
        navArgument("urunId") { 
            type = NavType.IntType          // Tam sayı bekliyoruz
        }
    )
) { backStackEntry ->
    // 2️⃣ Parametreyi oku
    val urunId = backStackEntry.arguments?.getInt("urunId") ?: 0
    UrunDetay(urunId = urunId, onGeriDon = { navController.popBackStack() })
}

// 3️⃣ Sayfaya giderken parametreyi gönder
navController.navigate("urun/$urunId")  // örn: "urun/5"Kotlin

Opsiyonel (İsteğe Bağlı) Parametre Seçimlik

Soru işareti ? ile query parametresi olarak tanımlanır:

composable(
    route = "arama?query={query}",          // ? ile opsiyonel
    arguments = listOf(
        navArgument("query") {
            type = NavType.StringType
            defaultValue = ""               // Varsayılan değer
            nullable = true                 // null olabilir
        }
    )
) { backStackEntry ->
    val query = backStackEntry.arguments?.getString("query")
    AramaSayfasi(query = query ?: "")
}Kotlin

🛍️ Tam Örnek: Ürün Listesi → Detay

// Ürün Listesi Sayfası
@Composable
fun UrunListesi(onUrunTikla: (Int) -> Unit) {
    val urunler = (1..20).map { "Ürün $it" }
    
    LazyColumn(contentPadding = PaddingValues(16.dp)) {
        itemsIndexed(urunler) { index, urun ->
            Card(
                modifier = Modifier
                    .fillMaxWidth()
                    .padding(vertical = 4.dp)
                    .clickable { onUrunTikla(index + 1) }
            ) {
                Text(text = urun, modifier = Modifier.padding(16.dp))
            }
        }
    }
}

// Ürün Detay Sayfası
@Composable
fun UrunDetay(urunId: Int, onGeriDon: () -> Unit) {
    Column(modifier = Modifier.padding(16.dp)) {
        IconButton(onClick = onGeriDon) {
            Icon(Icons.AutoMirrored.Filled.ArrowBack, "Geri")
        }
        Text("Ürün Detayı", style = MaterialTheme.typography.headlineLarge)
        Text("Ürün ID: $urunId", style = MaterialTheme.typography.titleLarge)
    }
}Kotlin

Bottom Navigation — Alt Menü

📱 Bottom Navigation Nedir?

Instagram, WhatsApp gibi uygulamalardaki alt menü çubuğudur. Kullanıcı bir sekmeye tıklayınca o ekran gösterilir.

🎯 Benzetme: Bir televizyon kumandası düşünün. Her düğme farklı bir kanala geçer. Alt menüdeki her ikon da farklı bir sayfaya geçirir.

Adım 1: Sekme Bilgilerini Tanımla

Her sekmenin route'u, başlığı ve ikonu olmalı:

// sealed class = sadece burada tanımlanan alt sınıfları olabilir
sealed class BottomNavItem(
    val route: String,        // Sayfa adresi
    val title: String,        // Menüde yazacak isim
    val icon: ImageVector      // Menüdeki ikon
) {
    object Home : BottomNavItem("home", "Ana Sayfa", Icons.Default.Home)
    object Search : BottomNavItem("search", "Ara", Icons.Default.Search)
    object Favorites : BottomNavItem("favorites", "Favoriler", Icons.Default.Favorite)
    object Profile : BottomNavItem("profile", "Profil", Icons.Default.Person)
}Kotlin

Adım 2: Scaffold + NavigationBar

Scaffold bize bottomBar alanı verir. NavigationBar ise alt menü çubuğudur:

@Composable
fun MainScreen() {
    val navController = rememberNavController()
    val items = listOf(
        BottomNavItem.Home, BottomNavItem.Search,
        BottomNavItem.Favorites, BottomNavItem.Profile
    )
    
    Scaffold(
        bottomBar = {
            NavigationBar {
                // Hangi sayfadayız? Onu takip et
                val navBackStackEntry by navController
                    .currentBackStackEntryAsState()
                val currentRoute = navBackStackEntry
                    ?.destination?.route
                
                items.forEach { item ->
                    NavigationBarItem(
                        icon = { Icon(item.icon, item.title) },
                        label = { Text(item.title) },
                        // Aktif mi? Şu anki route ile eşleş
                        selected = currentRoute == item.route,
                        onClick = {
                            navController.navigate(item.route) {
                                // Geri tuşunda ana sayfaya dön
                                popUpTo(navController.graph
                                    .findStartDestination().id) {
                                    saveState = true
                                }
                                launchSingleTop = true  // Çift tıklamayı engelle
                                restoreState = true    // Eski state'i geri yükle
                            }
                        }
                    )
                }
            }
        }
    ) { paddingValues ->
        NavHost(
            navController = navController,
            startDestination = BottomNavItem.Home.route,
            modifier = Modifier.padding(paddingValues)
        ) {
            composable(BottomNavItem.Home.route) { HomePage() }
            composable(BottomNavItem.Search.route) { SearchPage() }
            composable(BottomNavItem.Favorites.route) { FavoritesPage() }
            composable(BottomNavItem.Profile.route) { ProfilePage() }
        }
    }
}Kotlin
💡 3 Önemli Özellik:
• popUpTo → Geri tuşuna basınca direkt ana sayfaya döner
• launchSingleTop → Aynı sekmeye iki kez tıklayınca yeni kopya oluşturmaz
• restoreState → Sekmeler arası geçişte önceki durumu hatırlar

Navigation Animasyonları

🎬 Geçiş Animasyonları

Sayfalar arası geçişte kayma efekti ekleyerek uygulamayı daha profesyonel yapabiliriz:

composable(
    route = "detail/{id}",
    // → Sayfaya GİRERKEN: Sağdan kayarak gir
    enterTransition = {
        slideInHorizontally(
            initialOffsetX = { 1000 },       // Sağdan başla
            animationSpec = tween(300)     // 300ms sürsün
        )
    },
    // ← Yeni sayfaya geçerken ESKİ sayfa: Sola kayarak çık
    exitTransition = {
        slideOutHorizontally(
            targetOffsetX = { -1000 },      // Sola git
            animationSpec = tween(300)
        )
    },
    // → Geri tuşuyla DÖNÜLÜRKEN: Soldan kayarak gir
    popEnterTransition = {
        slideInHorizontally(
            initialOffsetX = { -1000 },
            animationSpec = tween(300)
        )
    },
    // ← Geri tuşuyla bu sayfa KAPATILIRKEN: Sağa kayarak çık
    popExitTransition = {
        slideOutHorizontally(
            targetOffsetX = { 1000 },
            animationSpec = tween(300)
        )
    }
) { backStackEntry ->
    DetaySayfasi(...)
}Kotlin
Animasyon Yönleri
← Sayfa A
 ⟵ exitTransition  |  enterTransition ⟶ 
Sayfa B →


İleri: A sola kayar, B sağdan gelir  |  Geri: B sağa kayar, A soldan gelir

// ====== 1. ROUTE TANIMLARI ======
sealed class OkulNavItem(
    val route: String,
    val title: String,
    val icon: ImageVector
) {
    object Ogrenciler : OkulNavItem("ogrenciler", "Öğrenciler", Icons.Default.Person)
    object Dersler : OkulNavItem("dersler", "Dersler", Icons.Default.Book)
    object Notlar : OkulNavItem("notlar", "Notlar", Icons.Default.Star)
    object Ayarlar : OkulNavItem("ayarlar", "Ayarlar", Icons.Default.Settings)
}

// ====== 2. VERİ MODELİ ======
data class Ogrenci(
    val id: Int,
    val ad: String,
    val sinif: String,
    val numara: Int
)

// ====== 3. ANA EKRAN ======
@Composable
fun OkulApp() {
    val navController = rememberNavController()
    val items = listOf(
        OkulNavItem.Ogrenciler, OkulNavItem.Dersler,
        OkulNavItem.Notlar, OkulNavItem.Ayarlar
    )
    
    Scaffold(
        bottomBar = {
            NavigationBar {
                val navBackStackEntry by navController
                    .currentBackStackEntryAsState()
                val currentRoute = navBackStackEntry
                    ?.destination?.route
                
                items.forEach { item ->
                    NavigationBarItem(
                        icon = { Icon(item.icon, item.title) },
                        label = { Text(item.title) },
                        selected = currentRoute == item.route,
                        onClick = {
                            navController.navigate(item.route) {
                                popUpTo(navController.graph
                                    .findStartDestination().id) {
                                    saveState = true
                                }
                                launchSingleTop = true
                                restoreState = true
                            }
                        }
                    )
                }
            }
        }
    ) { paddingValues ->
        NavHost(
            navController = navController,
            startDestination = OkulNavItem.Ogrenciler.route,
            modifier = Modifier.padding(paddingValues)
        ) {
            composable(OkulNavItem.Ogrenciler.route) {
                OgrenciListesi(
                    onOgrenciTikla = { ogrenciId ->
                        navController.navigate("ogrenci/$ogrenciId")
                    }
                )
            }
            composable(OkulNavItem.Dersler.route) { DerslerSayfasi() }
            composable(OkulNavItem.Notlar.route) { NotlarSayfasi() }
            composable(OkulNavItem.Ayarlar.route) { AyarlarSayfasi() }
            
            // TODO: Öğrenci detay sayfasını buraya ekle!
            // İpucu: Bölüm 8'deki parametre kullanımına bak
        }
    }
}

// ====== 4. ÖĞRENCİ LİSTESİ ======
@Composable
fun OgrenciListesi(onOgrenciTikla: (Int) -> Unit) {
    val ogrenciler = listOf(
        Ogrenci(1, "Ali Yılmaz", "10-A", 101),
        Ogrenci(2, "Ayşe Demir", "10-B", 205),
        Ogrenci(3, "Mehmet Kaya", "11-A", 302),
        Ogrenci(4, "Fatma Şahin", "11-B", 415),
        Ogrenci(5, "Emre Çelik", "12-A", 510)
    )
    
    LazyColumn(contentPadding = PaddingValues(16.dp)) {
        items(ogrenciler) { ogrenci ->
            Card(
                modifier = Modifier
                    .fillMaxWidth()
                    .padding(vertical = 4.dp)
                    .clickable { onOgrenciTikla(ogrenci.id) }
            ) {
                Column(modifier = Modifier.padding(16.dp)) {
                    Text(ogrenci.ad, fontWeight = FontWeight.Bold)
                    Text("Sınıf: ${ogrenci.sinif} | No: ${ogrenci.numara}")
                }
            }
        }
    }
}Kotlin — İskelet Proje
