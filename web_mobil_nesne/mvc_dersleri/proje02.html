📝 Kişisel Blog Sayfası

Ders 1-2 Konularını Kapsayan Proje

Proje Hakkında

MVC yapısını kullanarak basit bir kişisel blog sayfası oluşturuyoruz.

📚 Kapsanan Konular:
• MVC mimarisi
• Controller ve Action metotları
• View oluşturma
• ViewData ile veri aktarımı
• Razor syntax temelleri

1. HomeController

Controllers/HomeController.cs
using Microsoft.AspNetCore.Mvc;

public class HomeController : Controller
{
    // Ana sayfa
    public IActionResult Index()
    {
        ViewData["Title"] = "Ana Sayfa";
        ViewData["BlogName"] = "Teknoloji Günlüğü";
        ViewData["Description"] = "Yazılım ve teknoloji hakkında yazılarım";
        
        return View();
    }
    
    // Hakkımda sayfası
    public IActionResult Hakkimda()
    {
        ViewData["Title"] = "Hakkımda";
        ViewData["Author"] = "Ahmet Yılmaz";
        ViewData["Bio"] = "11. sınıf öğrencisi, yazılım meraklısı";
        
        return View();
    }
    
    // İletişim sayfası
    public IActionResult Iletisim()
    {
        ViewData["Title"] = "İletişim";
        ViewData["Email"] = "ahmet@example.com";
        
        return View();
    }
}
