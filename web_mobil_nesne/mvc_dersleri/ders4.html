View ve Razor Syntax

HTML ve C# Birleşimi

Razor Nedir?

Razor, HTML içinde C# kodu yazmamızı sağlayan bir view engine'dir. @ sembolü ile C# kodlarını HTML'e gömeriz.

💡 Razor Dosyaları: .cshtml uzantılıdır ve Views klasöründe bulunur.

📦 Modeller (Models)

View'larda kullandığımız Model.Name, Model.Stock gibi ifadeler birer C# sınıfından gelir. Bu sınıflara Model denir ve genellikle Models/ klasöründe tanımlanır.

📁 Dosya Konumu: Models/Product.cs  |  Models/Student.cs

🎮 Controller'lar

Controller'lar, kullanıcıdan gelen isteği karşılar, gerekli veriyi hazırlar ve View'a gönderir. View(model) çağrısı, veriyi Razor View'a iletir.

📁 Dosya Konumu: Controllers/ProductController.cs  |  Controllers/StudentController.cs

ProductController

Controllers/ProductController.cs
using Microsoft.AspNetCore.Mvc;
using MyApp.Models;

namespace MyApp.Controllers
{
    public class ProductController : Controller
    {
        // Örnek statik veri listesi — gerçek projede veritabanından gelir
        private static List<Product> _products = new()
        {
            new Product { Id = 1, Name = "Laptop",  Price = 25000, Stock = 15, Description = "Güçlü iş laptopu" },
            new Product { Id = 2, Name = "Mouse",   Price = 350,   Stock = 5,  Description = "Kablosuz mouse" },
            new Product { Id = 3, Name = "Monitor", Price = 8500,  Stock = 0,  Description = "27 inç 4K ekran" },
        };

        // GET /Product/Index → tüm ürün listesini View'a gönderir
        public IActionResult Index()
        {
            return View(_products);
        }

        // GET /Product/Details/1 → sadece o id'ye ait ürünü gönderir
        public IActionResult Details(int id)
        {
            var product = _products.FirstOrDefault(p => p.Id == id);
            if (product == null) return NotFound();
            return View(product);
        }
    }
}

🔗 Controller → View bağlantısı: ProductController.Index() metodu View(_products) döndürünce, ASP.NET Views/Product/Index.cshtml dosyasını otomatik bulur ve modeli oraya gönderir.

1 Aşağıdaki Book modelini oluşturun: Id, Title (kitap adı), Author (yazar), PageCount (sayfa sayısı) alanları. 
namespace MyApp.Models
{
    public class Book
    {
        public int Id { get; set; }
        public string Title { get; set; }
        public string Author { get; set; }
        public int PageCount { get; set; }
    }
}
2 BookController oluşturun. Index() action'ı 3 kitap içeren bir List<Book> hazırlayıp View'a göndersin. 
public class BookController : Controller
{
    public IActionResult Index()
    {
        var books = new List<Book>
        {
            new Book { Id = 1, Title = "Küçük Prens",    Author = "Antoine de Saint-Exupéry", PageCount = 96  },
            new Book { Id = 2, Title = "1984",            Author = "George Orwell",              PageCount = 352 },
            new Book { Id = 3, Title = "Suç ve Ceza",    Author = "Fyodor Dostoyevski",         PageCount = 671 },
        };
        return View(books);
    }
}
3 Views/Book/Index.cshtml sayfasında kitapları tablo olarak listeleyin. Sayfa sayısı 300'den fazlaysa "Uzun Kitap", değilse "Kısa Kitap" badge'ini görüntüleyin.

4 BookController'a Details(int id) action'ını ekleyin. Bu action verilen id'ye sahip kitabı bulup View'a göndersin; bulunamazsa NotFound() döndürsün. 
// BookController içine ekle
private static List<Book> _books = new()
{
    new Book { Id = 1, Title = "Küçük Prens", Author = "Antoine de Saint-Exupéry", PageCount = 96  },
    new Book { Id = 2, Title = "1984",          Author = "George Orwell",              PageCount = 352 },
    new Book { Id = 3, Title = "Suç ve Ceza",  Author = "Fyodor Dostoyevski",         PageCount = 671 },
};

public IActionResult Details(int id)
{
    var book = _books.FirstOrDefault(b => b.Id == id);
    if (book == null)
        return NotFound();

    return View(book);
}
