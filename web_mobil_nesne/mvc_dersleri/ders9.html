Layout, Partial View ve ViewBag/ViewData

Kod Tekrarını Azaltma ve Şablonlar

Layout Nedir?

Layout, tüm sayfalarda tekrar eden HTML yapısını (header, footer, menü) tek bir yerde tanımlamamızı sağlar. Ana şablon dosyasıdır.

💡 Avantajları:
• Kod tekrarını önler
• Tutarlı tasarım sağlar
• Bakımı kolaylaştırır

_Layout.cshtml Yapısı

Views/Shared/_Layout.cshtml
<!DOCTYPE html>
<html lang="tr">
<head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>@ViewData["Title"] - Okul Yönetim</title>
    
    <!-- Bootstrap CSS -->
    <link href="~/lib/bootstrap/dist/css/bootstrap.min.css" rel="stylesheet" />
    <link href="~/css/site.css" rel="stylesheet" />
</head>
<body>
    <!-- Navbar -->
    <nav class="navbar navbar-expand-lg navbar-dark bg-dark">
        <div class="container">
            <a class="navbar-brand" asp-controller="Home" asp-action="Index">
                🏫 Okul Yönetim
            </a>
            <button class="navbar-toggler" type="button" 
                    data-bs-toggle="collapse" data-bs-target="#navbarNav">
                <span class="navbar-toggler-icon"></span>
            </button>
            
            <div class="collapse navbar-collapse" id="navbarNav">
                <ul class="navbar-nav me-auto">
                    <li class="nav-item">
                        <a class="nav-link" asp-controller="Home" 
                           asp-action="Index">Ana Sayfa</a>
                    </li>
                    <li class="nav-item">
                        <a class="nav-link" asp-controller="Student" 
                           asp-action="Index">Öğrenciler</a>
                    </li>
                    <li class="nav-item">
                        <a class="nav-link" asp-controller="Product" 
                           asp-action="Index">Ürünler</a>
                    </li>
                </ul>
            </div>
        </div>
    </nav>
    
    <!-- Ana İçerik -->
    <main class="container mt-4">
        @RenderBody()  <!-- Sayfa içeriği buraya gelir -->
    </main>

    </footer>

    <script src="~/lib/jquery/dist/jquery.min.js"></script>
    <script src="~/lib/bootstrap/dist/js/bootstrap.bundle.min.js"></script>
    
    <!-- Sayfa özel scriptleri -->
    @await RenderSectionAsync("Scripts", required: false)

</html>

Sections (Bölümler)

Layout'ta belirli yerlere sayfa özel içerik eklemek için Section kullanılır.

Layout'ta Section tanımı
<!-- _Layout.cshtml -->
<head>
    ...
    @await RenderSectionAsync("Styles", required: false)
</head>
<body>
    ...
    @await RenderSectionAsync("Scripts", required: false)
</body>
View'de Section kullanımı
@section Styles {
    <link href="~/css/custom-page.css" rel="stylesheet" />
}

<h1>Sayfa İçeriği</h1>

@section Scripts {
    <script src="~/js/custom-page.js"></script>
    <script>
        console.log("Sayfa yüklendi!");
    </script>
}

Partial View Nedir?

Partial View, tekrar kullanılabilir küçük view parçalarıdır. Örneğin: ürün kartı, kullanıcı bilgi kutusu.

Views/Shared/_ProductCard.cshtml
@model Product

<div class="card h-100">
    <div class="card-body">
        <h5 class="card-title">@Model.Name</h5>
        <p class="card-text text-muted">@Model.Description</p>
        <p class="card-text">
            <strong>@Model.Price.ToString("C2")</strong>
        </p>
        @if (Model.Stock > 0)
        {
            <span class="badge bg-success">Stokta</span>
        }
        else
        {
            <span class="badge bg-danger">Tükendi</span>
        }
    </div>
    <div class="card-footer">
        <a asp-action="Details" asp-route-id="@Model.Id" 
           class="btn btn-primary btn-sm">Detay</a>
    </div>
</div>
Partial View Kullanımı
@model List<Product>

<div class="row">
    @foreach (var product in Model)
    {
        <div class="col-md-4 mb-3">
            <!-- Partial view çağırma -->
            @await Html.PartialAsync("_ProductCard", product)
            
            <!-- veya Tag Helper ile -->
            <partial name="_ProductCard" model="product" />
        </div>
    }
</div>

Örnek: Alert Partial

Views/Shared/_Alert.cshtml
<!-- TempData mesajlarını göster -->
@if (TempData["Success"] != null)
{
    <div class="alert alert-success alert-dismissible fade show">
        <i class="bi bi-check-circle"></i> @TempData["Success"]
        <button type="button" class="btn-close" data-bs-dismiss="alert"></button>
    </div>
}

@if (TempData["Error"] != null)
{
    <div class="alert alert-danger alert-dismissible fade show">
        <i class="bi bi-x-circle"></i> @TempData["Error"]
        <button type="button" class="btn-close" data-bs-dismiss="alert"></button>
    </div>
}

@if (TempData["Warning"] != null)
{
    <div class="alert alert-warning alert-dismissible fade show">
        <i class="bi bi-exclamation-triangle"></i> @TempData["Warning"]
        <button type="button" class="btn-close" data-bs-dismiss="alert"></button>
    </div>
}
Layout'ta kullanımı
<main class="container mt-4">
    <partial name="_Alert" />
    @RenderBody()
</main>

ViewBag vs ViewData vs TempData

Özellik ViewBag ViewData TempData
Tip dynamic Dictionary Dictionary
Ömür Tek istek Tek istek Sonraki isteğe kadar
Kullanım Controller → View Controller → View Redirect sonrası mesaj
Kullanım Örnekleri
// Controller'da

// ViewBag - dinamik
ViewBag.Title = "Ana Sayfa";
ViewBag.StudentCount = 35;

// ViewData - dictionary
ViewData["Message"] = "Hoş geldiniz";

// TempData - redirect sonrası
TempData["Success"] = "Kayıt başarıyla eklendi!";
return RedirectToAction("Index");
View'de
<h1>@ViewBag.Title</h1>
<p>Öğrenci Sayısı: @ViewBag.StudentCount</p>

<p>@ViewData["Message"]</p>

@if (TempData["Success"] != null)
{
    <div class="alert alert-success">@TempData["Success"]</div>
}

🎯 Sıra Sende!

1 Layout'un amacı nedir? @RenderBody() ne işe yarar?

Layout: Tüm sayfalarda ortak olan HTML yapısını (header, footer, menü) tek yerde tanımlar.

@RenderBody(): Her sayfanın kendine özel içeriğinin yerleştirileceği noktayı belirtir.

2 _StudentCard adında öğrenci bilgilerini gösteren bir Partial View yazın. 
<!-- Views/Shared/_StudentCard.cshtml -->
@model Student

<div class="card">
    <div class="card-body">
        <h5 class="card-title">@Model.FirstName @Model.LastName</h5>
        <p class="card-text">Sınıf: @Model.Grade</p>
        <p class="card-text">Email: @Model.Email</p>
        <a asp-action="Details" asp-route-id="@Model.Id" 
           class="btn btn-primary btn-sm">Detay</a>
    </div>
</div>
3 ViewBag ve TempData arasındaki fark nedir?

ViewBag:

  • Sadece tek istek için geçerli
  • Controller'dan View'e veri taşır
  • Redirect sonrası kaybolur

TempData:

  • Bir sonraki isteğe kadar saklanır
  • Redirect sonrası mesaj göstermek için ideal
  • Okunduktan sonra silinir
4 Sayfa özel JavaScript eklemek için Section nasıl kullanılır?

@section Scripts {

            console.log("Sayfa hazır!");
        });
    </script>
}
