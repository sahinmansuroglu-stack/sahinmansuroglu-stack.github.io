Entity Framework ve Veritabanı

ORM ile Veritabanı İşlemleri

Entity Framework Nedir?

Entity Framework (EF), Microsoft'un geliştirdiği bir ORM (Object-Relational Mapping) aracıdır. C# nesneleri ile veritabanı tabloları arasında köprü kurar.

💡 ORM Avantajları:
• SQL yazmadan veritabanı işlemleri
• C# nesneleri ile çalışma
• Veritabanı bağımsızlığı
• Otomatik migration desteği

DbContext Oluşturma

Data/ApplicationDbContext.cs
using Microsoft.EntityFrameworkCore;

public class ApplicationDbContext : DbContext
{
    public ApplicationDbContext(DbContextOptions<ApplicationDbContext> options)
        : base(options)
    {
    }
    
    // DbSet = Veritabanı Tablosu
    public DbSet<Student> Students { get; set; }
    public DbSet<Classroom> Classrooms { get; set; }
    
    protected override void OnModelCreating(ModelBuilder modelBuilder)
    {
        // Seed Data - Başlangıç verileri
        modelBuilder.Entity<Classroom>().HasData(
            new Classroom { Id = 1, Name = "11-A", Capacity = 30 },
            new Classroom { Id = 2, Name = "11-B", Capacity = 28 }
        );
    }
}

LINQ ile Veritabanı Sorguları

Temel LINQ Sorguları
using Microsoft.AspNetCore.Mvc;
using Microsoft.EntityFrameworkCore;
using System.Linq;

public class StudentController : Controller
{
    private readonly ApplicationDbContext _context;
    
    public StudentController(ApplicationDbContext context)
    {
        _context = context;
    }
    
    // Tümünü getir
    public IActionResult Index()
    {
        var students = _context.Students.ToList();
        return View(students);
    }
    
    // ID ile bul
    public IActionResult Details(int id)
    {
        var student = _context.Students.Find(id);
        // veya
        var student2 = _context.Students
            .FirstOrDefault(s => s.Id == id);
        return View(student);
    }
    
    // Filtreleme
    public IActionResult ByGrade(int grade)
    {
        var students = _context.Students
            .Where(s => s.Grade == grade)
            .OrderBy(s => s.LastName)
            .ToList();
        return View(students);
    }
    
    // Include ile ilişkili veri
    public IActionResult WithClassroom()
    {
        var students = _context.Students
            .Include(s => s.Classroom)
            .ToList();
        return View(students);
    }
}

Yaygın LINQ Metodları

Metod Açıklama Örnek
Where() Filtreleme .Where(x => x.Price > 100)
OrderBy() Sıralama (A-Z) .OrderBy(x => x.Name)
OrderByDescending() Ters sıralama .OrderByDescending(x => x.Date)
Select() Projeksiyon .Select(x => x.Name)
FirstOrDefault() İlk kayıt veya null .FirstOrDefault(x => x.Id == 1)
Any() Kayıt var mı? .Any(x => x.Stock == 0)
Count() Kayıt sayısı .Count(x => x.IsActive)
Sum() Toplam .Sum(x => x.Price)
Take() Belirli sayıda al .Take(10)
Skip() Atla .Skip(20).Take(10)

Örnek: Arama ve Filtreleme

Gelişmiş Arama
public IActionResult Search(string query, int? grade, string sort)
{
    var students = _context.Students.AsQueryable();
    
    // Arama
    if (!string.IsNullOrEmpty(query))
    {
        students = students.Where(s => 
            s.FirstName.Contains(query) || 
            s.LastName.Contains(query));
    }
    
    // Sınıf filtresi
    if (grade.HasValue)
    {
        students = students.Where(s => s.Grade == grade);
    }
    
    // Sıralama
    students = sort switch
    {
        "name" => students.OrderBy(s => s.FirstName),
        "name_desc" => students.OrderByDescending(s => s.FirstName),
        "grade" => students.OrderBy(s => s.Grade),
        _ => students.OrderBy(s => s.Id)
    };
    
    return View(students.ToList());
}

1 Entity Framework nedir ve ne işe yarar?

Entity Framework, ORM (Object-Relational Mapping) aracıdır.

  • C# nesneleri ile veritabanı arasında köprü kurar
  • SQL yazmadan veritabanı işlemleri yapılır
  • Model sınıfları otomatik tablolara dönüşür
  • LINQ ile sorgulama yapılır
2 Product için DbContext sınıfı yazın (Id, Name, Price, Stock alanları). 
public class Product
{
    public int Id { get; set; }
    public string Name { get; set; }
    public decimal Price { get; set; }
    public int Stock { get; set; }
}

public class AppDbContext : DbContext
{
    public AppDbContext(DbContextOptions<AppDbContext> options) 
        : base(options) { }
    
    public DbSet<Product> Products { get; set; }
}
