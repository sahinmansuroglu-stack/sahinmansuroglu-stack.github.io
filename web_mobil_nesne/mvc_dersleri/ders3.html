Controller ve Routing

URL Yönlendirme ve Action Metodları

Routing Nedir?

Routing, gelen URL'leri uygun Controller ve Action'a yönlendirme işlemidir. ASP.NET MVC'de routing kuralları Program.cs dosyasında tanımlanır.

Program.cs - Varsayılan Route
app.MapControllerRoute(
    name: "default",
    pattern: "{controller=Home}/{action=Index}/{id?}"
);
URL Controller Action Id
/ Home Index null
/Product Product Index null
/Product/Details Product Details null
/Product/Details/5 Product Details 5

Parametreli Action Metodları

1. Route Parametresi (URL'den)

URL: /Product/Details/5
public IActionResult Details(int id)
{
    // id = 5 olarak gelir
    ViewBag.ProductId = id;
    return View();
}

2. Query String Parametresi

URL: /Product/Search?name=laptop&minPrice=1000
public IActionResult Search(string name, decimal? minPrice)
{
    // name = "laptop", minPrice = 1000
    ViewBag.SearchName = name;
    ViewBag.MinPrice = minPrice ?? 0;
    return View();
}

3. Varsayılan Parametre Değeri

URL: /Product/List veya /Product/List/2
public IActionResult List(int page = 1)
{
    // page belirtilmezse 1 olur
    ViewBag.CurrentPage = page;
    return View();
}

Attribute Routing

Route'ları action üzerinde özel olarak tanımlayabilirsiniz.

Özel Route Tanımlama
public class ProductController : Controller
{
    // GET: /urunler
    [Route("urunler")]
    public IActionResult Index()
    {
        return View();
    }

    // GET: /urun/5
    [Route("urun/{id}")]
    public IActionResult Details(int id)
    {
        return View();
    }

    // GET: /kategori/elektronik/urunler
    [Route("kategori/{categoryName}/urunler")]
    public IActionResult ByCategory(string categoryName)
    {
        ViewBag.Category = categoryName;
        return View();
    }
}

Örnek: Ürün Controller (Tam)

Controllers/ProductController.cs
using Microsoft.AspNetCore.Mvc;

public class ProductController : Controller
{
    // Örnek ürün listesi (normalde veritabanından gelir)
    private static List<Product> _products = new List<Product>
    {
        new Product { Id = 1, Name = "Laptop", Price = 15000 },
        new Product { Id = 2, Name = "Mouse", Price = 250 },
        new Product { Id = 3, Name = "Keyboard", Price = 500 }
    };

    // GET: /Product
    public IActionResult Index()
    {
        return View(_products);
    }

    // GET: /Product/Details/1
    public IActionResult Details(int id)
    {
        var product = _products.FirstOrDefault(p => p.Id == id);
        
        if (product == null)
            return NotFound();
            
        return View(product);
    }

    // GET: /Product/Search?query=lap
    public IActionResult Search(string query)
    {
        var results = _products
            .Where(p => p.Name.Contains(query, StringComparison.OrdinalIgnoreCase))
            .ToList();
        
        ViewBag.Query = query;
        return View(results);
    }
}

Örnek: Sayfalama (Pagination)

Sayfalı Liste
// GET: /Product/List?page=2&pageSize=10
public IActionResult List(int page = 1, int pageSize = 10)
{
    var allProducts = GetAllProducts(); // 100 ürün olsun
    
    var pagedProducts = allProducts
        .Skip((page - 1) * pageSize)
        .Take(pageSize)
        .ToList();
    
    ViewBag.CurrentPage = page;
    ViewBag.TotalPages = (int)Math.Ceiling(allProducts.Count / (double)pageSize);
    
    return View(pagedProducts);
}
