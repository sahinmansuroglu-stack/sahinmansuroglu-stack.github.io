ASP.NET MVC'ye Giriş ve Kurulum

Web Geliştirme Dünyasına İlk Adım

Giriş

Bu derste, modern web uygulamaları geliştirmek için kullanılan ASP.NET MVC framework'ünü tanıyacağız. HTML, CSS, JavaScript ve jQuery bilgilerinizi artık sunucu tarafında C# ile birleştireceğiz.

💡 Neden Backend?
Frontend kullanıcının gördüğü kısmı oluşturur. Ancak verilerin saklanması, kullanıcı girişi, veritabanı işlemleri için Backend gereklidir.

ASP.NET MVC Nedir?

ASP.NET MVC, Microsoft tarafından geliştirilen web framework'üdür.

MVC Açılımı

  • M - Model: Verilerinizi temsil eder
  • V - View: Kullanıcıya gösterilen arayüz
  • C - Controller: Model ve View arasındaki köprü
📦 Model
↔
🎮 Controller
↔
🖥️ View

İlk MVC Projesi

  1. Visual Studio → Create new project
  2. "ASP.NET Core Web App (Model-View-Controller)" seçin
  3. Proje adı: IlkMvcProjesi
  4. .NET 8.0 seçin → Create

Proje Yapısı

IlkMvcProjesi/
├── Controllers/     // Controller sınıfları
│   └── HomeController.cs
├── Models/          // Model sınıfları
├── Views/           // Razor View dosyaları
│   ├── Home/
│   │   └── Index.cshtml
│   └── Shared/
│       └── _Layout.cshtml
├── wwwroot/         // CSS, JS, resimler
└── Program.cs       // Başlangıç noktası

Örnek 1: Temel Controller

HomeController.cs
using Microsoft.AspNetCore.Mvc;

public class HomeController : Controller
{
    // GET: /Home/Index
    public IActionResult Index()
    {
        return View();
    }

    // GET: /Home/Hakkimizda
    public IActionResult Hakkimizda()
    {
        return View();
    }
}

Örnek 2: Temel View

Views/Home/Index.cshtml
@{
    ViewData["Title"] = "Ana Sayfa";
}

<h1>Hoş Geldiniz!</h1>
<p>Bugünün tarihi: @DateTime.Now.ToShortDateString()</p>

URL Yapısı (Routing)

https://localhost:5001/{controller}/{action}/{id?}

/Home/Index     → HomeController.Index()
/Home/Hakkimizda → HomeController.Hakkimizda()
/Urun/Detay/5   → UrunController.Detay(5)
